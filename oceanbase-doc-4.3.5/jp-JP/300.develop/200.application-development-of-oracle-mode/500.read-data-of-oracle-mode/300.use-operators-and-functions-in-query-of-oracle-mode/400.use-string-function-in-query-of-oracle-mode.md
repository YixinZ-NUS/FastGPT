# クエリにおける文字列関数の使用

一般的な文字列関数には、文字列の長さを取得する（Length）、文字列を切り取る（Substr）、文字列の連結、大文字小文字変換（Upper/Lower）、文字列の前後の空白を削除する（Ltrim/Rtrim/Trim）などがあります。

注意が必要なのは、Oracleテナント内で、文字列長関数Lengthの長さの単位は文字であり、Lengthbの文字列長の単位はバイトであり、文字セットに関連しているということです。
