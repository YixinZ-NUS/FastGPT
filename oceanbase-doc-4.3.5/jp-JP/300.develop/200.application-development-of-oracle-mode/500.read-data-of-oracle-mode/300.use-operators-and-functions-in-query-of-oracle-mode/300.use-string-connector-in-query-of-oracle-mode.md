# クエリでの文字列連結演算子の使用

文字列連結演算子。Oracleテナントでは通常 \`\|\|\` が文字列連結演算子として使用されます。

## 例

1. `if_ny` テーブルを作成します。

   ```unknow
   obclient> CREATE TABLE tif_ny (
                   id        NUMBER,
                   f_name     VARCHAR2(20),
                   l_name      VARCHAR2(20),
                   c_date      date,
                   PRIMARY KEY (id)
                   );
   Query OK, 0 rows affected
   ```

2. `tif_ny` テーブルにデータを挿入します。

   ```unknow
   obclient> INSERT INTO tif_ny VALUES(401,'Zhang','San',date'1970-8-11'),
             (402,'Li','Si',date'1971-5-11'),(403,'Wang','Wu',date'1972-7-11');
   Query OK, 3 rows affected
   Records: 3  Duplicates: 0  Warnings: 0
   ```

3. `tif_ny` テーブルのデータをクエリします。

   ```unknow
   obclient> SELECT * FROM tif_ny;
   +-----+--------+--------+-----------+
   | ID  | F_NAME | L_NAME | C_DATE    |
   +-----+--------+--------+-----------+
   | 401 | Zhang  | San    | 11-AUG-70 |
   | 402 | Li     | Si     | 11-MAY-71 |
   | 403 | Wang   | Wu     | 11-JUL-72 |
   +-----+--------+--------+-----------+
   3 rows in set
   ```

4. "\|\|" は文字列連結演算子として、`tif_ny` テーブルの `f_name` 列と `l_name` 列を連結して `name` 列を作成します。

   ```unknow
   obclient> SELECT f_name || ' ' || l_name name FROM tif_ny;
   +-----------+
   | NAME      |
   +-----------+
   | Zhang San |
   | Li Si     |
   | Wang Wu   |
   +-----------+
   3 rows in set
   ```
