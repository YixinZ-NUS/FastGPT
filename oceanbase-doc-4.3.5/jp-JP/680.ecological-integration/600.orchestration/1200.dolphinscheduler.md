# DolphinSchedulerにOceanBaseデータソースを設定する

DolphinSchedulerはOceanBaseデータソースをサポートしています。OceanBaseのMySQLモードを使用している場合は、MySQLの設定に従ってOceanBaseデータソースを設定できます。

DolphinSchedulerのコンソールにログインし、上部のナビゲーションバーで、「ソースセンター > データソースの作成 > ソースタイプの選択」をクリックします。データソースのタイプをMySQLに選択します。最新バージョンのDolphinSchedulerを使用している場合は、OCEANBASEデータソースタイプを選択できます。

DolphinSchedulerにOceanBaseデータソースを設定するには、以下のパラメータを提供する必要があります：

| パラメータ | 説明 |
| ---- | ---- |
| データソース | OceanBaseを選択 |
| データソースの名前 | データソースの名前を入力 |
| IPアドレス | ホスト名：OceanBaseへの接続IPアドレスを入力 |
| ポート | OceanBaseに接続するポートを入力：
| ユーザー名 | OceanBaseに接続するユーザー名を設定 |
| パスワード | OceanBaseに接続するパスワードを設定 |
| データベース名 | OceanBaseに接続するデータベースの名前を入力 |

詳細については、DolphinScheduler [公式ドキュメント](https://dolphinscheduler.apache.org/en-us/docs/3.2.1/guide/datasource/mysql)を参照してください。
