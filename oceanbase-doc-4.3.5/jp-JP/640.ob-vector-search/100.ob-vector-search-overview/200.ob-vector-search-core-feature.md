# ベクトル検索のコア機能

OceanBaseは、エンベディング・ベクトルデータ(vector embeddings)の格納、インデックス作成、検索機能を提供します。具体的な内容は以下の通りです：

|**コア機能**|**説明**|
|---|---|
| ベクトルデータ型 | <ul><li>最大16,000次元のfloat型密ベクトルデータの保存をサポートします。</li><li>疎ベクトルデータの保存をサポートします。</li></ul> |
| ベクトルインデックス | <ul><li>厳密検索と近似近傍検索をサポートします。</li><li>L2距離、内積、コサイン類似度の計算をサポートします。</li><li>HNSW、HNSW_SQ、HNSW_BQインデックスをサポートし、インデックス列の最大次元は4,096です。HNSW_BQインデックスはV4.3.5 BP2バージョンからサポートされます。</li><li>IVF、IVF_SQ、IVF_PQインデックスをサポートし（IVF_SQは実験的機能）、インデックス列の最大次元は4096です。</li></ul> |
| ベクトル検索SQL演算子 | ベクトルの加算、減算、乗算、比較、集計などの基本的な演算子をサポートします。|

## 使用上の制限と注意

* OceanBaseはデフォルトでNULL値を先頭にする（NULL first）比較モードを採用しているため、NULL値を含む列をソートするとNULL値が先頭に配置されます。クエリ実行時には`NOT NULL`条件を追加することをお勧めします。
* OceanBaseの[Communitiy Edition](../../400.deploy/500.deploy-oceanbase-database-community-edition/100.deployment-overview.md)/[デスクトップ版](../../400.deploy/500.deploy-oceanbase-database-community-edition/800.deploy-oceanbase-desktop.md)は、SYSテナントおよびユーザーテナントでのベクトル検索機能の使用をサポートしています。Enterprise Editionは、ユーザーテナントでのみサポートしています。
