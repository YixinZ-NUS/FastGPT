# information_schema.SESSION_VARIABLES

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V1.4 版本开始引入。</p>
</main>

## 功能描述

记录所有 Session 变量（含以 "_" 开头的隐藏变量）信息。

## 字段说明

|    **字段名称**    |    **类型**     | **是否可以为 NULL** |   **描述**    |
|----------------|---------------|----------------|-------------|
| VARIABLE_NAME  | varchar(128)  | NO             | Session 变量名 |
| VARIABLE_VALUE | varchar(1024) | YES            | Session 变量值 |

## 查询示例

查看 Session 变量 `ob_trx_timeout` 的值。

```shell
obclient [oceanbase]> SELECT * FROM information_schema.SESSION_VARIABLES WHERE VARIABLE_NAME='ob_trx_timeout ';
```

查询结果如下：

```shell
+----------------+----------------+
| VARIABLE_NAME  | VARIABLE_VALUE |
+----------------+----------------+
| ob_trx_timeout | 86400000000    |
+----------------+----------------+
1 row in set
```
