# oceanbase.DBA_TAB_MODIFICATIONS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

展示本租户中表的累计修改信息。

## 字段说明

| 字段名称 | 类型 | 是否可以为 NULL | 描述 |
| --- | --- | --- | --- |
| TABLE_OWNER | varchar(128) | NO | 修改的表所属的数据库 |
| TABLE_NAME | varchar(128) | NO | 修改的表的表名 |
| PARTITION_NAME | varchar(128) | NO | 修改的分区名 |
| SUBPARTITION_NAME | varchar(128) | NO | 修改的子分区名 |
| INSERTS | bigint(20) | NO | 大致插入次数 |
| UPDATES | bigint(20) | NO | 大致更新次数 |
| DELETES | bigint(20) | NO | 大致删除次数 |
| TIMESTAMP | date | NO | 表示上次修改表的时间 |
| TRUNCATED | varchar(3) | NO | 目前无实际含义，默认为 NULL |
| DROP_SEGMENTS | bigint(0) | NO | 目前无实际含义，默认为 NULL |

## 查询示例

查看表 `t_subpart` 的累计修改信息。

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.DBA_TAB_MODIFICATIONS WHERE TABLE_NAME='t_subpart';
```
