# oceanbase.DBA_OB_ACCESS_POINT

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.2.0 版本开始引入。</p>
</main>

## 功能描述

展示本租户的租户入口的 Server 列表。在基于网络的物理备库场景中，主要用于获取主租户所在副本的 OBServer 节点的 IP 及 SQL 端口号。

## 字段说明

| 字段 | 类型 | 是否为 NULL | 描述 |
| --- | --- | --- | --- |
| TENANT_ID | bigint(20)   | NO | 租户ID<ul><li>1：系统租户  </li><li>其他值：用户租户 </li></ul>|
| TENANT_NAME | varchar(128) | NO | 租户名 |
| SVR_IP | varchar(46) | NO | 系统日志流副本所属 Server IP |
| SQL_PORT | bigint(20) | NO | 系统日志流副本所属 SQL  Port |

## 查询示例

系统租户查询本租户的租户入口 Server 列表。

```shell
obclient [oceanbase]> SELECT SVR_IP, SQL_PORT FROM oceanbase.DBA_OB_ACCESS_POINT;
```

查询结果如下：

```shell
+----------------+----------+
| SVR_IP         | SQL_PORT |
+----------------+----------+
| 172.xx.xxx.xxx |     2881 |
+----------------+----------+
1 row in set
```
