# oceanbase.DBA_OB_STORAGE_IO_USAGE

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.3.4 版本开始引入。</p>
</main>

## 功能描述

展示当前租户对象存储和云盘的各类 IO 操作情况。

## 字段说明

| 字段名称   | 类型          | 是否可以为 NULL | 描述                          |
|-----------|---------------|----------------|-------------------------------|
| TENANT_ID | bigint(20)    | NO   | 租户 ID     |
| PATH      | varchar(1024) | NO   | 路径格式类似 `oss://`、`file://`、`s3://` 等    |
| ENDPOINT  | varchar(256)  | NO   | <ul><li>无共享（Shared-Nothing，SN）模式下，展示备份恢复、数据导入、数据导出的对象存储的 Endpoint</li> <li>共享存储（Shared-Storage，SS）模式下，展示日志（clog）或数据（Data）存储以及备份恢复、数据导入、数据导出的对象存储设备的 Endpoint</li></ul> |
| TYPE      | varchar(128)  | NO   | 操作类型：<ul><li>`READ`：读取类型</li> <li>`WRITE`：写入类型</li> <li>`READ_BYTES`：对象存储网络入口流量</li> <li>`WRITE_BYTES`：对象存储网络出口流量</li> <li>`TAG`：对象存储中的标签类型。本地盘和云盘不展示该类型</li></ul>     |
| TOTAL     | bigint(20)    | NO   | 每种类型(`TYPE`)对应的累计值：<ul><li>`READ`：读取类型相关操作的次数累计</li> <li>`WRITE`：写入类型相关操作的次数类型</li> <li>`READ_BYTES`：对象存储网络入口流量的总和，单位为字节</li> <li>`WRITE_BYTES`：对象存储网络出口流量的总和，单位为字节</li> <li>`TAG`：对象存储中的标签类型的累计值。本地盘和云盘不展示该统计项</li></ul>      |
