# oceanbase.DBA_OB_LICENSE

<main id="notice" type='explain'>
  <h4>说明</h4>
  <ul><li>该视图仅适用于单机版，在分布式下，该视图查询为空。</li>
  <li>该视图从 V4.3.5 BP3 版本开始引入。</li></ul>
</main>

## 功能描述

展示用户当前加载的 License 信息。

## 字段说明

| **字段** | **类型** | **是否为 NULL** | **描述** |
| --- | --- | --- | --- |
| END_USER | varchar(512) | NO | 授权用户名。 |
| LICENSE_ID | varchar(128) | NO | License 的 ID。 |
| LICENSE_CODE | varchar(128) | NO | 合同号。|
| LICENSE_TYPE | varchar(128) | NO | License 类型。<ul><li>Commercial：商业版。</li><li>Trial：试用版。</li></ul>|
| PRODUCT_TYPE | varchar(128) | NO | 产品类型。<ul><li>SE：Standalone Enterprise Edition 的缩写，代表单机企业版。</li><li>DE：Distributed Enterprise Edition 的缩写，代表分布式企业版。</li></ul>|
| ISSUANCE_DATE | timestamp(6) | NO | License 的颁发时间。 |
| ACTIVATION_TIME | timestamp(6) | NO | License 的激活时间。 |
| EXPIRED_TIME | timestamp(6) | NO | License 的过期时间。 |
| OPTIONS | varchar(1024) | NO | License 中所包含的功能选件。<ul><li>EE：Enterprise Edition 的缩写，代表 OceanBase 单机企业版基础功能。</li><li>PS：Primary-Standby 的缩写，代表 OceanBase 单机集群间复制选件。</li><li>MT：Multitenant 的缩写，代表 OceanBase 单机多租户选件。</li><li>AP：OLAP 缩写，代表 OceanBase 单机 OLAP 选件。</li></ul>|
| CORE_NUM | bigint(20) unsigned | NO | 授权使用的 CPU 核数。|
| NODE_NUM | bigint(20) unsigned | NO | 授权节点数，单机版固定为 1。|
| CLUSTER_ULID | varchar(30) | NO | 集群启动时自动产生。 |

## 查询示例

系统租户查看已加载的 License 信息。

```shell
obclient [oceanbase]> select * from oceanbase.DBA_OB_LICENSE;
```
