# oceanbase.DBA_OB_ARCHIVE_DEST

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

展示当前租户的归档路径配置。由于系统租户不包含用户数据，是集群管理用的租户，不支持备份恢复，故 `sys` 租户未配置归档路径。`sys` 租户下，该视图的查询结果始终为空。

## 字段说明

| 字段名称 | 类型 | 是否可以为 NULL | 描述 |
| --- | --- | --- | --- |
| DEST_NO | bigint(20) | NO | 标识 log_archive_dest_n |
| NAME | varchar(1024) | NO | 字段名，包含如下字段：<ul><li>`binding`：归档和业务的优先模式</li> <li>`dest_id`：归档目的端的 ID</li> <li>`path`：归档路径信息</li> <li>`piece_switch_interval`：Piece 的切换周期</li> <li>`state`：归档目的端的状态</li></ul> |
| VALUE | varchar(4096) | NO | `name` 字段对应字段值 |
