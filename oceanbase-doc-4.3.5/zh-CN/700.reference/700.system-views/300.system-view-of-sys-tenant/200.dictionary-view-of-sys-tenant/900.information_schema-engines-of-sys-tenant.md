# information_schema.ENGINES

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V1.4 版本开始引入。</p>
</main>

## 功能描述

记录存储引擎相关信息。

## 字段说明

|   **字段名称**   |    **类型**     | **是否可以为 NULL** |     **描述**      |
|--------------|---------------|----------------|-----------------|
| ENGINE       | varchar(64)   | NO             | 引擎类型            |
| SUPPORT      | varchar(8)    | NO             | 是否支持当前存储引擎      |
| COMMENT      | varchar(80)   | NO             | 注释              |
| TRANSACTIONS | varchar(3)    | NO             | 事务相关内容          |
| XA           | varchar(3)    | NO             | 分布式事务 XA 相关内容   |
| SAVEPOINTS   | varchar(3)    | NO             | Savepoints 相关内容 |

## 查询示例

查看存储引擎相关信息。

```shell
obclient [oceanbase]> SELECT * FROM information_schema.ENGINES;
```

查询结果如下：

```shell
+--------+---------+-----------------------+--------------+------+------------+
| ENGINE | SUPPORT | COMMENT               | TRANSACTIONS | XA   | SAVEPOINTS |
+--------+---------+-----------------------+--------------+------+------------+
| InnoDB | YES     | Supports transactions | YES          | NO   | YES        |
+--------+---------+-----------------------+--------------+------+------------+
1 row in set
```
