# oceanbase.DBA_OB_CCL_RULES

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.3.5 BP3 版本开始引入。</p>
</main>

## 功能描述

用于记录所有租户的限流规则。

## 字段说明

| 字段名称 | 类型 | 是否可以为 NULL | 描述 |
| --- | --- | --- | --- |
|TENANT_ID| bigint(20)| NO| 租户 ID|
|CCL_RULE_ID|bigint(20)|NO|限流规则 ID|
|CCL_RULE_NAME| varchar(128)| NO| 限流规则名称|
|AFFECT_USER_NAME| varchar(64)| NO |限流生效的用户|
|AFFECT_HOST| varchar(128)| NO| 限流生效的 IP 地址|
|AFFECT_FOR_ALL_DATABASES| tinyint(4)| NO| 是否对全部数据库生效|
|AFFECT_FOR_ALL_TABLES| tinyint(4)| NO| 是否对全部表生效|
|AFFECT_DML| bigint(20)| NO| 限流生效的 DML 类型语句|
|AFFECT_DATABASE| varchar(128)| YES |限流生效的数据库|
|AFFECT_TABLE| varchar(256)| YES| 限流生效的表|
|AFFECT_SCOPE| bigint(20)| NO |限流生效粒度:<li>规则级别: 0 </li><li>Format SQLID 级别: 1</li>|
|CCL_KEYWORDS| varchar(1048576)| NO |限流关键字|
|MAX_CONCURRENCY| bigint(20)| NO |限流的并发度上限|

## 查询示例

查询所有租户的限流规则。

```shell
obclient[oceanbase]> SELECT * FROM DBA_OB_CCL_RULES;
```
