# oceanbase.CDB_OB_TABLET_TO_LS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

系统租户下展示所有租户的 Tablet 列表以及 Tablet 所属的日志流信息。

## 字段说明

|   字段名称    |     类型     | 是否可以为 NULL |        描述        |
|-----------|------------|------------|------------------|
| TENANT_ID | bigint(20) | NO         | 租户 ID            |
| TABLET_ID | bigint(20) | NO         | TABLET ID        |
| LS_ID     | bigint(1)  | NO         | TABLET 所属的日志流 ID |

## 查询示例

系统租户查询租户 ID 为 1002 的租户的 1001 号 日志流的 Tablet 列表信息，展示前 5 条信息。

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.CDB_OB_TABLET_TO_LS WHERE TENANT_ID=1002 AND LS_ID=1001 LIMIT 5;
```

查询结果如下所示：

```shell
+-----------+-----------+-------+
| TENANT_ID | TABLET_ID | LS_ID |
+-----------+-----------+-------+
|      1002 |    200004 |  1001 |
|      1002 |    200005 |  1001 |
|      1002 |    200007 |  1001 |
|      1002 |    200008 |  1001 |
|      1002 |    200009 |  1001 |
+-----------+-----------+-------+
5 rows in set
```
