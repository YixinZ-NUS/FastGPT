# oceanbase.CDB_OB_SENSITIVE_RULE_PLAINACCESS_USERS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.3.5 BP3 版本开始引入。</p>
</main>

## 功能描述

展示所有租户下所有拥有某条敏感规则关联列的明文数据访问权限的用户或角色。

## 字段说明

|        字段名称         |      类型       | 是否可以为 NULL |                      描述                                |
|---------------------|---------------|------------|-----------------------------------------------------------------|
| TENANT_ID         | int(20)    | NO         | 租户 ID                                           |
| RULE_NAME         | varchar(256)    | NO         | 敏感规则名称                                           |
| USER_NAME      | varchar(128)    | NO         | 用户名称                                        |
| USER_TYPE        | varchar(4)    | NO         | 用户类型，取值为 `USER` 或 `ROLE`。`USER` 表示用户，`ROLE` 表示角色                                      |

## 查询示例

查看所有租户下所有拥有某条敏感规则关联列的明文数据访问权限的用户或角色。

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.CDB_OB_SENSITIVE_RULE_PLAINACCESS_USERS;
```

查询结果如下：

```shell
+-----------+-----------+-----------+-----------+
| TENANT_ID | RULE_NAME | USER_NAME | USER_TYPE |
+-----------+-----------+-----------+-----------+
| 1         | r1        | root      | USER      |
+-----------+-----------+-----------+-----------+
1 row in set
```
