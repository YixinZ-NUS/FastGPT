# oceanbase.CDB_OB_BACKUP_DELETE_POLICY

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

展示各租户下为过期清理/自动清理设置的清理策略。

## 字段说明

| 字段名称 | 类型 | 是否可以为 NULL | 描述 |
| --- | --- | --- | --- |
| TENANT_ID | bigint(20) | NO | 租户 ID |
| POLICY_NAME | varchar(256) | NO | 清理策略名 |
| RECOVERY_WINDOW | varchar(64) | NO | 可恢复时间窗口，默认值：0 |

## 查询示例

系统租户查看所有租户的清理策略。

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.CDB_OB_BACKUP_DELETE_POLICY;
```

查询结果如下：

```shell
+-----------+-------------+-----------------+
| TENANT_ID | POLICY_NAME | RECOVERY_WINDOW |
+-----------+-------------+-----------------+
|      1002 | default     | 7d              |
+-----------+-------------+-----------------+
1 row in set
```
