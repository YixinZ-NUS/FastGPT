# oceanbase.DBA_RECYCLEBIN

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

展示当前租户下所有回收站对象的信息。

## 字段说明

|      字段名称   |  类型     | 是否可以为 NULL |    描述             |
|----------------|---------------|------------|--------------------|
| OWNER          | varchar(128) | NO   | 对象原始所有者的名称     |
| OBJECT_NAME    | varchar(128) | NO   | 对象的新名称     |
| ORIGINAL_NAME  | varchar(128) | NO   | 对象的原始名称     |
| OPERATION      | varchar(9)   | NO   | 暂不支持该字段，默认为 NULL      |
| TYPE           | varchar(25)  | NO   | 对象类型     |
| TS_NAME        | varchar(30)  | NO   | 对象所属的表空间的名称     |
| CREATETIME     | date         | YES  | 创建对象的时间     |
| DROPTIME       | date         | YES  | 对象进入回收站的时间     |
| DROPSCN        | bigint(0)    | NO   | 暂不支持该字段，默认为 NULL     |
| PARTITION_NAME | varchar(128) | NO   | 暂不支持该字段，默认为 NULL     |
| CAN_UNDROP     | varchar(3)   | NO   | 是否可以将对象从回收站中恢复：<ul><li>YES </li><li>NO </li></ul>      |
| CAN_PURGE      | varchar(3)   | NO   | 是否可以将对象清除：<ul><li>YES </li><li>NO </li></ul>      |
| RELATED        | bigint(0)    | NO   | 暂不支持该字段，默认为 NULL     |
| BASE_OBJECT    | bigint(0)    | NO   | 暂不支持该字段，默认为 NULL    |
| PURGE_OBJECT   | bigint(0)    | NO   | 暂不支持该字段，默认为 NULL     |
| SPACE          | bigint(0)    | NO   | 暂不支持该字段，默认为 NULL     |
