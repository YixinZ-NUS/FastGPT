# information_schema.GLOBAL_STATUS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V1.4 版本开始引入。</p>
</main>

## 功能描述

记录 OceanBase 数据库的全局服务器状态变量信息，包括当前活跃连接数、数据库活跃时间（Uptime）等。

## 字段说明

|    **字段名称**    |    **类型**     | **是否可以为 NULL** |   **描述**    |
|----------------|---------------|----------------|-------------|
| VARIABLE_NAME  | varchar(128)  | NO             | 全局状态对应的变量名称 |
| VARIABLE_VALUE | varchar(1024) | YES            | 全局状态对应的变量内容 |

## 查询示例

查看 OceanBase 数据库中的全局服务器状态变量及其对应的值。

```shell
obclient [oceanbase]> SELECT * FROM information_schema.GLOBAL_STATUS;
```

查询结果如下：

```shell
+-------------------+----------------+
| VARIABLE_NAME     | VARIABLE_VALUE |
+-------------------+----------------+
| Threads_connected | 1              |
| Uptime            | 254110         |
+-------------------+----------------+
2 rows in set
```
