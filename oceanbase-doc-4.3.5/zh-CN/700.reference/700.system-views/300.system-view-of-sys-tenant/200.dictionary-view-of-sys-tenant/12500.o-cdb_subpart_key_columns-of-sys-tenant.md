# oceanbase.CDB_SUBPART_KEY_COLUMNS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

展示所有租户的分区对象（分区表、分区索引）的子分区键包含的列。

## 字段说明

|      **字段名称**      |    **类型**     | **是否可以为 NULL** |                                                              **描述**                                                              |
|--------------------|---------------|----------------|----------------------------------------------------------------------------------------------------------------------------------|
| CON_ID             | bigint(20)    | NO             | 租户 ID                                         |
| OWNER              | varchar(128)  | NO             | 分区表或分区索引的拥有者                                  |
| NAME               | varchar(128)  | NO             | 分区表或分区索引的名字                                   |
| OBJECT_TYPE        | varchar(5)    | NO             | 分区对象的类型： <li> TABLE   <li> INDEX    |
| COLUMN_NAME        | text          | NO             | 列的名字                                          |
| COLUMN_POSITION    | bigint(21)    | NO             | 列在分区键中的位置                                     |
| COLLATED_COLUMN_ID | bigint(0)     | NO             | 目前暂未支持该字段，当前该字段默认为 NULL                       |

## 查询示例

系统租户下，查看租户 ID 为 1002 的租户的分区表 `tbl2_f_rl` 中子分区建包含的列。

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.CDB_SUBPART_KEY_COLUMNS WHERE CON_ID=1002 AND NAME='tbl2_f_rl';
```

查询结果如下：

```shell
+--------+----------+-----------+-------------+-------------+-----------------+--------------------+
| CON_ID | OWNER    | NAME      | OBJECT_TYPE | COLUMN_NAME | COLUMN_POSITION | COLLATED_COLUMN_ID |
+--------+----------+-----------+-------------+-------------+-----------------+--------------------+
|   1002 | infotest | tbl2_f_rl | TABLE       | col2        |               1 |               NULL |
+--------+----------+-----------+-------------+-------------+-----------------+--------------------+
1 row in set
```
