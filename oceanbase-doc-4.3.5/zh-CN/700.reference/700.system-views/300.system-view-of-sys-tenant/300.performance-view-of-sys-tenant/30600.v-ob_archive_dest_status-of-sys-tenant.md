# V$OB_ARCHIVE_DEST_STATUS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.1.0 版本开始引入。</p>
</main>

## 功能描述

展示所有用户租户各个归档目的端状态。

## 字段说明

| 字段 | 类型 | 是否可以为 NULL | 描述 |
| --- | --- | --- | --- |
| TENANT_ID | bigint(20)   | NO | 租户 ID |
| DEST_ID | bigint(20)   | NO | 归档目标端 ID |
| PATH | varchar(4096) | NO | 归档路径 |
| STATUS | varchar(64) | NO | 归档状态 |
| CHECKPOINT_SCN | bigint(20) unsigned | NO | 已归档日志连续位点 |
| SYNCHRONIZED | varchar(32) | NO | 归档是否与租户日志同步：<ul><li> `YES` 指与租户日志同步  </li><li> `NO` 指与租户日志不同步 </li></ul>|
| COMMENT | varchar(262144) | NO | 错误提示信息 |

## 查询示例

系统租户查询用户租户各个归档目的端的状态。

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.V$OB_ARCHIVE_DEST_STATUS;
```

查询结果如下：

```shell
+-----------+---------+-------------------------------------+--------+---------------------+--------------+---------+
| TENANT_ID | DEST_ID | PATH                                | STATUS | CHECKPOINT_SCN      | SYNCHRONIZED | COMMENT |
+-----------+---------+-------------------------------------+--------+---------------------+--------------+---------+
|      1002 |    1001 | file:///home/admin/oceanbase/arglog | DOING  | 1736321349494697000 | NO           |         |
+-----------+---------+-------------------------------------+--------+---------------------+--------------+---------+
1 row in set
```
