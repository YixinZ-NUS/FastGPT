# V$OB_TENANT_MEMORY

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

展示当前 OBServer 节点上每个租户的内存使用状况。

## 字段说明

|   字段名称    |     类型      | 是否可以为 NULL |    描述     |
|-----------|-------------|------------|-----------|
| TENANT_ID | bigint(20)  | NO         | 租户 ID     |
| SVR_IP    | varchar(46) | NO         | 服务器 IP 地址 |
| SVR_PORT  | bigint(20)  | NO         | 服务器端口号    |
| HOLD      | bigint(20)  | NO         | 租户已使用内存，单位为字节   |
| FREE      | bigint(21)  | NO         | 租户剩余内存，单位为字节    |

## 查询示例

系统租户查询租户 ID 为 1002 的租户在当前节点上的内存使用情况。

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.V$OB_TENANT_MEMORY WHERE TENANT_ID = 1002;
```

查询结果如下：

```shell
+-----------+----------------+----------+-----------+------------+
| TENANT_ID | SVR_IP         | SVR_PORT | HOLD      | FREE       |
+-----------+----------------+----------+-----------+------------+
|      1002 | 172.xx.xxx.xxx |     2882 | 723124224 | 4645584896 |
+-----------+----------------+----------+-----------+------------+
1 row in set
```
