## 查询示例

查看数据库中表 `TBL2_F_RL` 的信息。

```shell
obclient [SYS]> SELECT OWNER, TABLE_NAME, STATUS, PCT_FREE, TEMPORARY, NESTED, ROW_MOVEMENT, DROPPED, EXTERNAL FROM SYS.DBA_TABLES WHERE TABLE_NAME = 'TBL2_F_RL';
```

```shell
| SYS   | TBL2_F_RL  | VALID  |        0 | N         | NO     | NO           | NO      | NO       |

1 row in set
```
