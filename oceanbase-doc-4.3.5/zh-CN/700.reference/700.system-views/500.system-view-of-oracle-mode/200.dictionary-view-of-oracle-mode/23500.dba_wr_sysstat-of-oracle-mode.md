# DBA_WR_SYSSTAT

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该视图从 V4.2.1 版本开始引入。</p>
</main>

## 功能描述

展示所有租户的系统统计项的详细信息。

## 字段说明

| **字段** | **类型** | **是否为 NULL** | **描述** |
| --- | --- | --- | --- |
| CLUSTER_ID | NUMBER(38) | NO | 集群 ID |
| TENANT_ID | NUMBER(38) | NO | 租户 ID |
| SNAP_ID | NUMBER(38) | NO | 快照 ID |
| SVR_IP | VARCHAR2(46) | NO | 节点 IP |
| SVR_PORT | NUMBER(38) | NO | 节点 PORT |
| STAT_ID | NUMBER(38) | NO | 统计项 ID |
| VALUE | NUMBER(38) | YES | 统计项的值 |

## 查询示例

查看本租户的系统统计项的详细信息，展示第 1 条记录。

```shell
obclient [SYS]> SELECT * FROM SYS.DBA_WR_SYSSTAT  WHERE ROWNUM <= 1\G
```

返回结果如下：

```shell
*************************** 1. row ***************************
CLUSTER_ID: 10001
 TENANT_ID: 1004
   SNAP_ID: 1331
    SVR_IP: xx.xx.xx.xx
  SVR_PORT: 28825
   STAT_ID: 10000
     VALUE: 53915269
1 row in set (0.063 sec)
```
