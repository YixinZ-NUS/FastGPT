# USER_SYS_PRIVS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.77 版本开始引入。</p>
</main>

## 功能描述

列出了授予当前用户的系统权限。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|   **字段名称**   |    **类型**     | **是否可以为 NULL** |      **描述**      |
|--------------|---------------|----------------|------------------|
| GRANTEE      | VARCHAR2(128) | NO             | 接受授权的受让人名称、用户或角色 |
| PRIVILEGE    | VARCHAR2(40)  | NO             | 授予的系统权限             |
| ADMIN_OPTION | VARCHAR2(3)   | NO             | 是否授予管理选项           |

## 查询示例

查看 `SYS` 用户被授予的系统权限，展示前 5 条记录。

```shell
obclient [SYS]> SELECT * FROM USER_SYS_PRIVS WHERE USERNAME='SYS' AND ROWNUM <=5;
```

查询结果如下：

```shell
+----------+------------------+--------------+
| USERNAME | PRIVILEGE        | ADMIN_OPTION |
+----------+------------------+--------------+
| SYS      | CREATE SESSION   | YES          |
| SYS      | CREATE TABLE     | YES          |
| SYS      | CREATE ANY TABLE | YES          |
| SYS      | ALTER ANY TABLE  | YES          |
| SYS      | BACKUP ANY TABLE | YES          |
+----------+------------------+--------------+
5 rows in set
```
