# ALL_PROFILES

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

展示当前用户可访问的所有 Profile 文件及其参数信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|   **字段名称**    |    **类型**    | **是否可以为 NULL** |              **描述**               |
|---------------|--------------|----------------|-----------------------------------|
| PROFILE       | VARCHAR2(30) | NO             | Profile 文件名称                             |
| RESOURCE_NAME | VARCHAR2(32) | NO             | 资源参数名称                             |
| RESOURCE_TYPE | VARCHAR2(8)  | NO             | 资源参数类型：<ul><li>`KERNEL`</li> <li>`PASSWORD`</li></ul>   |
| LIMIT_ON_RESOURCE | VARCHAR2(40) | NO         | 资源参数的限制值                           |

## 查询示例

查看当前用户可访问的 Profile 文件及其参数信息。

```shell
obclient [SYS]> SELECT * FROM SYS.ALL_PROFILES;
```

查询结果如下：

```shell
+----------+---------------+---------------+-------------------+
| PROFILE  | RESOURCE_NAME | RESOURCE_TYPE | LIMIT_ON_RESOURCE |
+----------+---------------+---------------+-------------------+
| DEFAULT  | NULL          | NULL          | NULL              |
| profile1 | NULL          | NULL          | NULL              |
+----------+---------------+---------------+-------------------+
2 rows in set
```
