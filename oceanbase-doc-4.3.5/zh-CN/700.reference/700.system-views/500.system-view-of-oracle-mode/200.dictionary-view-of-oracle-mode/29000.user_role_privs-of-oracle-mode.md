# USER_ROLE_PRIVS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.77 版本开始引入。</p>
</main>

## 功能描述

描述授予当前用户的角色。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|   **字段名称**   |    **类型**    | **是否可以为 NULL** |           **描述**           |
|--------------|--------------|----------------|----------------------------|
| GRANTEE      | VARCHAR2(30) | NO             | 用户的名称，或 PUBLIC             |
| GRANTED_ROLE | VARCHAR2(30) | NO             | 授予用户的角色名称                  |
| ADMIN_OPTION | VARCHAR2(3)  | NO             | 指示授权是否带有 ADMIN 选项          |
| DEFAULT_ROLE | VARCHAR2(3)  | NO             | 指示角色是否被指定为用户的 DEFAULT ROLE |

## 查询示例

查看当前用户所拥有的角色。

```shell
obclient [SYS]> SELECT * FROM USER_ROLE_PRIVS;
```

查询结果如下：

```shell
+---------+--------------+--------------+--------------+
| GRANTEE | GRANTED_ROLE | ADMIN_OPTION | DEFAULT_ROLE |
+---------+--------------+--------------+--------------+
| SYS     | ROLE1        | YES          | YES          |
| SYS     | ROLE2        | YES          | YES          |
+---------+--------------+--------------+--------------+
2 rows in set
```
