# DBA_OB_ARCHIVE_DEST

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

展示当前租户的归档路径配置。

## 字段说明

| 字段名称 | 类型 | 是否可以为 NULL | 描述 |
| --- | --- | --- | --- |
| DEST_NO | NUMBER(38) | NO | 标识 log_archive_dest_n |
| NAME | VARCHAR2(1024) | NO | 字段名，包含如下字段：<ul><li>`binding`：归档和业务的优先模式</li> <li>`dest_id`：归档目的端的 ID</li> <li>`path`：归档路径信息</li> <li>`piece_switch_interval`：Piece 的切换周期</li> <li>`state`：归档目的端的状态</li></ul> |
| VALUE | VARCHAR2(4096) | NO | `name` 字段对应字段值 |

## 查询示例

用户租户查看本租户的归档路径配置。

```shell
obclient[SYS]> SELECT * FROM SYS.DBA_OB_ARCHIVE_DEST;
```

查询结果如下：

```shell
+---------+-----------------------+-------------------------------------------------------------------------------------------------------------------------------+
| DEST_NO | NAME                  | VALUE                                                                                                                         |
+---------+-----------------------+-------------------------------------------------------------------------------------------------------------------------------+
|       0 | binding               | OPTIONAL                                                                                                                      |
|       0 | dest_id               | 1004                                                                                                                          |
|       0 | path                  | oss://example_patharchive?host=cn-******.aliyuncs.com&access_id=******&access_key=*****&checksum_type=md5&enable_worm=true    |
|       0 | piece_switch_interval | 1d                                                                                                                            |
|       0 | state                 | ENABLE                                                                                                                        |
+---------+-----------------------+-------------------------------------------------------------------------------------------------------------------------------+
5 rows in set
```
