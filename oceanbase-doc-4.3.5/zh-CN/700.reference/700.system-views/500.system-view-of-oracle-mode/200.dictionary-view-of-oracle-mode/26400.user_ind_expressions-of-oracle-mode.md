# USER_IND_EXPRESSIONS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

展示当前用户所拥有的 function-based 索引的信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|     **字段名称**      |     **类型**     | **是否可以为 NULL** |        **描述**        |
|-------------------|----------------|----------------|----------------------|
| INDEX_NAME        | VARCHAR2(128)  | NO             | 索引名字                 |
| TABLE_NAME        | VARCHAR2(128)  | NO             | 索引所在表的名字             |
| COLUMN_EXPRESSION | VARCHAR2(1000) | NO             | function-base 索引的表达式 |
| COLUMN_POSITION   | NUMBER(38)     | NO             | 列所在索引的位置             |

## 查询示例

查看当前用户所拥有的函数索引的信息。

```shell
obclient [SYS]> SELECT * FROM SYS.USER_IND_EXPRESSIONS;
```

查询结果如下：

```shell
+----------------+------------+-------------------+-----------------+
| INDEX_NAME     | TABLE_NAME | COLUMN_EXPRESSION | COLUMN_POSITION |
+----------------+------------+-------------------+-----------------+
| TBL2_FUNC_IDX1 | TBL2_FUNC  | ("C1" + 1)        |               1 |
+----------------+------------+-------------------+-----------------+
1 row in set
```
