# USER_SOURCE

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

展示当前用户拥有的存储对象的文本源。其列与 `ALL_SOURCE` 中的列相同，去除了 `OWNER` 字段。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|   **字段名称**    |    **类型**    | **是否可以为 NULL** |         **描述**          |
|---------------|--------------|----------------|-------------------------|
| NAME          | VARCHAR2(30) | NO             | 对象名                     |
| TYPE          | VARCHAR2(12) | NO             | 对象类型                    |
| LINE          | NUMBER       | NO             | TEXT 字段对应的源代码中的行号       |
| TEXT          | CLOB         | NO             | 当前行的源代码                 |
| ORIGIN_CON_ID | NUMBER(38)   | NO             | 对象所属的租户的租户 ID。如果是系统包，则值为 0 |

## 查询示例

查看当前用户所拥有的存储对象的文本源，展示前 2 条记录。

```shell
obclient [SYS]> SELECT * FROM SYS.USER_SOURCE WHERE ROWNUM<=2\G
```

查询结果如下：

```shell
*************************** 1. row ***************************
         NAME: DEPT_CTX_PKG
         TYPE: PACKAGE BODY
         LINE: 1
         TEXT: PACKAGE BODY dept_ctx_pkg IS
  PROCEDURE set_dept(dept_val VARCHAR2) IS
  BEGIN
    DBMS_SESSION.SET_CONTEXT('dept_ctx', 'CURRENT_DEPT', dept_val); -- 命名空间与上下文名称需一致<sup>1</sup>
  END;
END;
ORIGIN_CON_ID: 1004
*************************** 2. row ***************************
         NAME: DEPT_CTX_PKG
         TYPE: PACKAGE
         LINE: 1
         TEXT: PACKAGE dept_ctx_pkg IS
  PROCEDURE set_dept(dept_val VARCHAR2);
END;
ORIGIN_CON_ID: 1004
2 rows in set
```
