# USER_IND_SUBPARTITIONS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

展示数据库中用户所拥有的所有索引子分区的信息，与 `DBA_IND_SUBPARTITIONS`、`ALL_IND_SUBPARTITIONS` 相比没有 `INDEX_OWNER` 字段。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 查询示例

查看当前用户为所有者的数据库中，索引子分区（二级分区）的信息。

```shell
obclient [SYS]> SELECT INDEX_NAME, PARTITION_NAME, SUBPARTITION_NAME, HIGH_VALUE, PARTITION_POSITION, SUBPARTITION_POSITION FROM USER_IND_SUBPARTITIONS;
```
