# DBA_MVIEW_DEPS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.3.5 BP2 版本开始引入。</p>
</main>

## 功能描述

该视图用于展示当前租户物化视图的依赖关系。

## 字段说明

| **字段名称** |   **类型**  | **是否可以为 NULL** | **描述** |
|-------------|---------------|---------------------|---------|
| MVIEW_OWNER | VARCHAR2(128) | YES  | 物化视图所有者名称。|
| MVIEW_NAME  | VARCHAR2(256) | YES  | 物化视图名称。|
| DEP_OWNER   | VARCHAR2(128) | YES  | 依赖对象所有者名称。|
| DEP_NAME    | VARCHAR2(256) | YES  | 依赖对象名称。|
| DEP_TYPE    | VARCHAR2(12)  | NO   | 依赖对象类型：<ul><li>`MV`：物化视图</li><li>`TABLE`：普通表</li><li>`EXTERNAL TABLE`：外表</li></ul>|

## 查询示例

查看当前租户物化视图的依赖关系。

```shell
obclient> SELECT * FROM sys.DBA_MVIEW_DEPS;
```

返回结果如下：

```shell
+-------------+------------------+-----------+---------------+------------------------------------------------------------------+
| MVIEW_OWNER | MVIEW_NAME       | DEP_OWNER | DEP_NAME      | DEP_TYPE                                                         |
+-------------+------------------+-----------+---------------+------------------------------------------------------------------+
| SYS         | MV_EXT_TBL1      | SYS       | EXT_TBL1      | EXTERNAL TABLE                                                   |
| SYS         | MV1_TBL3_TBL4    | SYS       | TBL3          | TABLE                                                            |
| SYS         | MV1_TBL3_TBL4    | SYS       | TBL4          | TABLE                                                            |
| SYS         | MV_MV1_TBL3_TBL4 | SYS       | MV1_TBL3_TBL4 | MV                                                               |
+-------------+------------------+-----------+---------------+------------------------------------------------------------------+
4 rows in set
```
