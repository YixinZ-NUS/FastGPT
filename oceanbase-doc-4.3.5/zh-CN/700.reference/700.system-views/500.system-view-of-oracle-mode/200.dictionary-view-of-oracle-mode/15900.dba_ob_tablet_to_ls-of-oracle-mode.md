# DBA_OB_TABLET_TO_LS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

展示本租户的所有 Tablet 列表以及 Tablet 所属的日志流信息。

## 字段说明

|   字段名称    |   类型   | 是否可以为 NULL |        描述        |
|-----------|--------|------------|------------------|
| TABLET_ID | NUMBER | NO         | TABLET ID        |
| LS_ID     | NUMBER | NO         | TABLET 所属的日志流 ID |

## 查询示例

用户租户查询本租户 1001 号日志流的 Tablet 列表信息，展示前 5 条信息。

```shell
obclient [SYS]> SELECT * FROM SYS.DBA_OB_TABLET_TO_LS WHERE LS_ID=1001 AND ROWNUM <= 5;
```

查询结果如下所示：

```shell
+-----------+-------+
| TABLET_ID | LS_ID |
+-----------+-------+
|    200004 |  1001 |
|    200005 |  1001 |
|    200006 |  1001 |
|    200007 |  1001 |
|    200008 |  1001 |
+-----------+-------+
5 rows in set
```
