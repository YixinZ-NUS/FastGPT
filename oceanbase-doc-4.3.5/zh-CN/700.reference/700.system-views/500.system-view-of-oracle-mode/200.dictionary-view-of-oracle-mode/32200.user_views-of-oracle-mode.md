# USER_VIEWS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

展示当前用户拥有的视图。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 查询示例

查看当前用户拥有的视图 `VIEW1` 的信息。

```shell
obclient [SYS]> SELECT VIEW_NAME, TEXT_LENGTH, TEXT FROM SYS.USER_VIEWS WHERE VIEW_NAME='VIEW1';
```

查询结果如下：

```shell
+-----------+-------------+-----------------------------+
| VIEW_NAME | TEXT_LENGTH | TEXT                        |
+-----------+-------------+-----------------------------+
| VIEW1     |          27 | SELECT col1, col2 FROM tbl1 |
+-----------+-------------+-----------------------------+
1 row in set
```
