# DBA_SYNONYMS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

展示当前租户下的所有同义词。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|   **字段名称**   |    **类型**     | **是否可以为 NULL** |                     **描述**                      |
|--------------|---------------|----------------|-------------------------------------------------|
| OWNER        | VARCHAR2(128) | NO             | 同义词的拥有者                                         |
| SYNONYM_NAME | VARCHAR2(128) | NO             | 同义词名字                                           |
| TABLE_OWNER  | VARCHAR2(128) | NO             | 同义词引用对象的拥有者                                     |
| TABLE_NAME   | VARCHAR2(128) | NO             | 同义词引用的对象名字                                      |
| DB_LINK      | VARCHAR2(128) | NO             | 如果引用的是 DB Link，则表示 DB Link 名字。目前同义词暂不支持 DB LINK |

## 查询示例

查看当前租户下的所有同义词。

```shell
obclient [SYS]> SELECT * FROM SYS.DBA_SYNONYMS;
```

查询结果如下：

```shell
+----------+--------------+-------------+------------+---------+
| OWNER    | SYNONYM_NAME | TABLE_OWNER | TABLE_NAME | DB_LINK |
+----------+--------------+-------------+------------+---------+
| __public | SYN_PUB      | SYS         | TEST       | NULL    |
| SYS      | S1           | SYS         | TEST       | NULL    |
+----------+--------------+-------------+------------+---------+
2 rows in set
```
