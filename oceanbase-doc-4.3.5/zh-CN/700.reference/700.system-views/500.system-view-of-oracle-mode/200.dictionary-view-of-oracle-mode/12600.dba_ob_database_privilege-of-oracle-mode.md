# DBA_OB_DATABASE_PRIVILEGE

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

展示当前租户所拥有的控制当前用户的权限。

## 字段说明

| 字段名称          | 类型          | 是否可以为 NULL | 描述              |
|-------------------|-----------------------------------|------|-----|
| USER_ID           | NUMBER(38)                        | NO   |   用户 ID      |
| USERNAME          | VARCHAR2(128)                     | NO   |   用户名称      |
| DATABASE_NAME     | VARCHAR2(128)                     | NO   |   当前用户名称       |
| GMT_CREATE        | TIMESTAMP(6) WITH LOCAL TIME ZONE | NO   |   创建时间       |
| GMT_MODIFIED      | TIMESTAMP(6) WITH LOCAL TIME ZONE | NO   |   修改时间      |
| PRIV_ALTER        | VARCHAR2(3)                       | NO   |   是否有修改表的权限       |
| PRIV_CREATE       | VARCHAR2(3)                       | NO   |   是否有创建表的权限      |
| PRIV_DELETE       | VARCHAR2(3)                       | NO   |   是否有删除表记录的权限      |
| PRIV_DROP         | VARCHAR2(3)                       | NO   |   是否有删除表的权限      |
| PRIV_GRANT_OPTION | VARCHAR2(3)                       | NO   |   是否有转授权限      |
| PRIV_INSERT       | VARCHAR2(3)                       | NO   |   是否有插入记录的权限      |
| PRIV_UPDATE       | VARCHAR2(3)                       | NO   |   是否有更新记录的权限      |
| PRIV_SELECT       | VARCHAR2(3)                       | NO   |   是否有查询记录的权限      |
| PRIV_INDEX        | VARCHAR2(3)                       | NO   |   是否有给表加索引的权限      |
| PRIV_CREATE_VIEW  | VARCHAR2(3)                       | NO   |   是否有创建视图的权限      |
| PRIV_SHOW_VIEW    | VARCHAR2(3)                       | NO   |   是否有查看视图的权限      |

## 查询示例

用户租户查看本租户所拥有的控制当前用户的权限。

obclient [SYS]> SELECT * FROM SYS.DBA_OB_DATABASE_PRIVILEGE;
