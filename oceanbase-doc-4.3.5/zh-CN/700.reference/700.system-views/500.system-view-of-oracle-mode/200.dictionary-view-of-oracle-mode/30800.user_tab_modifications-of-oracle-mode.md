# USER_TAB_MODIFICATIONS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

展示当前用户拥有的表的累计修改信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

| 字段名称 | 类型 | 是否可以为 NULL | 描述 |
| --- | --- | --- | --- |
| TABLE_NAME | VARCHAR2(128) | NO | 修改表名 |
| PARTITION_NAME | VARCHAR2(128) | NO | 修改分区的名称 |
| SUBPARTITION_NAME | VARCHAR2(128) | NO | 修改子分区的名称 |
| INSERTS | NUMBER | NO | 大致插入次数 |
| UPDATES | NUMBER | NO | 大致更新次数 |
| DELETES | NUMBER | NO | 大致删除次数 |
| TIMESTAMP | DATE | NO | 表示上次修改表的时间 |
| TRUNCATED | VARCHAR2(3) | NO | 目前无实际含义，默认为 NULL |
| DROP_SEGMENTS | NUMBER | NO | 目前无实际含义，默认为 NULL |

## 查询示例

查看当前用户拥有的表中，表 `T_SUBPART` 的累计修改信息。
