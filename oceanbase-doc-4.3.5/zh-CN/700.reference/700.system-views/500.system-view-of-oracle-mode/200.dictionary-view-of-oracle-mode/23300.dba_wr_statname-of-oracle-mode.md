# DBA_WR_STATNAME

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该视图从 V4.2.1 版本开始引入。</p>
</main>

## 功能描述

展示本租户的系统统计项的名称。

## 字段说明

| **字段** | **类型** | **是否为 NULL** | **描述** |
| --- | --- | --- | --- |
| CLUSTER_ID | NUMBER(38) | NO | 集群 ID |
| TENANT_ID | NUMBER(38) | NO | 租户 ID |
| STAT_ID | NUMBER(38) | NO | 统计项 ID |
| STAT_NAME | VARCHAR2(64) | NO | 统计项名字 |

## 查询示例

查询本租户的系统统计项的名称，展示第 1 条记录。

```shell
obclient [SYS]> SELECT * FROM SYS.DBA_WR_STATNAME WHERE ROWNUM <= 1\G
```

查询结果如下：

```shell
*************************** 1. row ***************************
CLUSTER_ID: 10001
 TENANT_ID: 1004
   STAT_ID: 10000
 STAT_NAME: rpc packet in
1 row in set (0.063 sec)
```
