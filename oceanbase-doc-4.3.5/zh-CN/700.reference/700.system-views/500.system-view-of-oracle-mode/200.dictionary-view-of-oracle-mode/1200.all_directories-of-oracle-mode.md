# ALL_DIRECTORIES

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V3.2.1 版本开始引入。</p>
</main>

## 功能描述

展示当前用户可访问的所有目录。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|    **字段名称**    |     **类型**     | **是否可以为 NULL** |   **描述**    |
|----------------|----------------|----------------|-------------|
| OWNER          | VARCHAR2(128)  | NO             | 目录所属的 Owner |
| DIRECTORY_NAME | VARCHAR2(128)  | NO             | 目录名称        |
| DIRECTORY_PATH | VARCHAR2(4000) | NO             | 目录路径        |
| ORIGIN_CON_ID  | NUMBER(38)     | NO             | 租户 ID       |

## 查询示例

查看当前用户可访问的所有目录。

```shell
obclient [SYS]> SELECT * FROM SYS.ALL_DIRECTORIES;
```

查询结果如下：

```shell
+-------+----------------+----------------------+---------------+
| OWNER | DIRECTORY_NAME | DIRECTORY_PATH       | ORIGIN_CON_ID |
+-------+----------------+----------------------+---------------+
| SYS   | SQL_FILE_DIR   | /home/admin/sqldump  |          1004 |
+-------+----------------+----------------------+---------------+
1 row in set
```
