# ALL_SUBPART_KEY_COLUMNS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.77 版本开始引入。</p>
</main>

## 功能描述

展示用户可访问的所有分区对象中子分区键包含的列。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|              **字段名称**              |     **类型**     | **是否可以为 NULL** |                                                              **描述**                                                              |
|------------------------------------|----------------|----------------|----------------------------------------------------------------------------------------------------------------------------------|
| OWNER                              | VARCHAR2(128)  | NO             | 分区表或分区索引的拥有者                                  |
| NAME                               | VARCHAR2(128)  | NO             | 分区表或分区索引的名字                                   |
| OBJECT_TYPE                        | CHAR(5)        | NO             | 分区对象的类型： <li> TABLE   <li> INDEX    |
| COLUMN_NAME                        | VARCHAR2(4000) | NO             | 列的名字                                          |
| COLUMN_POSITION                    | NUMBER         | NO             | 列在分区键中的位置                                     |
| COLLATED_COLUMN_ID | NUMBER         | NO             | 目前暂未支持该字段，当前该字段默认为NULL                        |

## 查询示例

查看当前用户可访问的分区表 `TBL2_F_RL` 中子分区建包含的列。

```shell
obclient [SYS]> SELECT * FROM SYS.ALL_SUBPART_KEY_COLUMNS WHERE NAME='TBL2_F_RL';
```

查询结果如下：

```shell
+-------+-----------+-------------+-------------+-----------------+--------------------+
| OWNER | NAME      | OBJECT_TYPE | COLUMN_NAME | COLUMN_POSITION | COLLATED_COLUMN_ID |
+-------+-----------+-------------+-------------+-----------------+--------------------+
| SYS   | TBL2_F_RL | TABLE       | COL2        |               1 |               NULL |
+-------+-----------+-------------+-------------+-----------------+--------------------+
1 row in set
```
