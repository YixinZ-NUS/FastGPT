# ALL_OB_EXTERNAL_TABLE_FILES

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.2.0 版本开始引入。</p>
</main>

## 功能描述

展示当前用户所拥有访问权限的外部表的文件列表。

## 字段说明

| **字段** | **类型** | **是否为 NULL** | **描述** |
| --- | --- | --- | --- |
| TABLE_NAME | VARCHAR2(256) | YES | 外表表名 |
| OWNER | VARCHAR2(128) | YES | 外表所在数据库名 |
| PARTITION_NAME | CHAR(2) | NO | 外表的 PARTITION NAME |
| FILE_URL | VARBINARY(16384) | NO | 外表的文件 URL |
| FILE_SIZE | NUMBER(38) | NO | 该文件的大小 |

## 查询示例

查看当前用户拥有访问权限的外部表的文件列表。

```shell
obclient [SYS]> SELECT * FROM SYS.ALL_OB_EXTERNAL_TABLE_FILES;
```

查询结果如下：

```shell
+------------+-------+----------------+------------------------------+-----------+
| TABLE_NAME | OWNER | PARTITION_NAME | FILE_URL                     | FILE_SIZE |
+------------+-------+----------------+------------------------------+-----------+
| EXT_T3     | SYS   | P0             | xx.xx.xx.208:2882%data.csv   |        29 |
+------------+-------+----------------+------------------------------+-----------+
1 row in set
```
