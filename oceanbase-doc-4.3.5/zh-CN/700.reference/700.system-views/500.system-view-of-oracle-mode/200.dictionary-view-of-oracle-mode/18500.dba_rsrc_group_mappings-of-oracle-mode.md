# DBA_RSRC_GROUP_MAPPINGS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V3.2.4 版本开始引入。</p>
</main>

## 功能描述

展示本租户中用户和资源组之间的绑定关系。

## 字段说明

|      字段类型      |       类型       | 是否可以为 NULL |                描述                 |
|----------------|----------------|------------|-----------------------------------|
| ATTRIBUTE      | VARCHAR2(128)  | NO         | 类别，当前版本仅支持 `ORACLE_USER`。         |
| VALUE          | VARBINARY(128) | NO         | 值，当类别为 `ORACLE_USER` 时，记录的是登录用户名。 |
| CONSUMER_GROUP | VARCHAR2(128)  | YES        | 用户绑定到的资源组的名称。                     |
| STATUS         | VARCHAR2(128)  | NO         | 资源组的状态，仅用于兼容，目前值固定为 `NULL`。       |

## 查询示例

用户租户查询本租户中用户和资源组之间的绑定关系。

```shell
obclient [SYS]> SELECT * FROM SYS.DBA_RSRC_GROUP_MAPPINGS;
```

查询结果如下：

```shell
+-----------+-------+-------------------+--------+
| ATTRIBUTE | VALUE | CONSUMER_GROUP    | STATUS |
+-----------+-------+-------------------+--------+
| user      | sys   | interactive_group | NULL   |
+-----------+-------+-------------------+--------+
1 row in set
```
