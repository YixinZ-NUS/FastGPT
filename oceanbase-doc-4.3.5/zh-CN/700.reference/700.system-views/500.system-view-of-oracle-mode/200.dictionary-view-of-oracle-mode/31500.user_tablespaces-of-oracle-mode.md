# USER_TABLESPACES

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

展示当前用户可以访问的表空间。其列与 `DBA_TABLESPACES` 中的列相同，去除了 `PLUGGED_IN` 字段。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 查询示例

查看当前用户可以访问的所有表空间。

```shell
obclient [SYS]> SELECT TABLESPACE_NAME, ENCRYPTED FROM SYS.USER_TABLESPACES;
```

查询结果如下：

```shell
+-----------------+-----------+
| TABLESPACE_NAME | ENCRYPTED |
+-----------------+-----------+
| SECTEST_TS1     | YES       |
| TS2             | NO        |
+-----------------+-----------+
2 rows in set
```
