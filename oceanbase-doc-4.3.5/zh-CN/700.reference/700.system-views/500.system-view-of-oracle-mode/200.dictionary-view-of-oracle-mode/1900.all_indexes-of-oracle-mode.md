# ALL_INDEXES

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

ALL_INDEXES 用于查看用户可访问的所有表的索引的索引列信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

```shell
obclient [SYS]> SELECT * FROM SYS.ALL_INDEXES WHERE table_name='TBL2';
```
