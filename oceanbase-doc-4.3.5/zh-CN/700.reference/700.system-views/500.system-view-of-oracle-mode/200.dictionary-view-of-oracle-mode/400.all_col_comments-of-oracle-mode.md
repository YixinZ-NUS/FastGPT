# ALL_COL_COMMENTS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

展示用户可访问的视图和表的所有列注释。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|  **字段名称**   |     **类型**     | **是否可以为 NULL** | **描述** |
|-------------|----------------|----------------|--------|
| OWNER       | VARCHAR2(128)  | NO             | 对象的拥有者 |
| TABLE_NAME  | VARCHAR2(128)  | NO             | 对象名字   |
| COLUMN_NAME | VARCHAR2(128)  | NO             | 列名     |
| COMMENTS    | VARCHAR2(4000) | YES            | 列注释    |

## 查询示例

查看当前用户可访问的所有表中，表 `T1` 的所有列注释。

```shell
obclient [SYS]> SELECT * FROM SYS.ALL_COL_COMMENTS WHERE TABLE_NAME='T1';
```

查询结果如下：

```shell
+-------+------------+----------------+----------+
| OWNER | TABLE_NAME | COLUMN_NAME    | COMMENTS |
+-------+------------+----------------+----------+
| SYS   | T1         | __pk_increment | NULL     |
| SYS   | T1         | ID1            | score    |
| SYS   | T1         | ID2            | NULL     |
+-------+------------+----------------+----------+
3 rows in set
```
