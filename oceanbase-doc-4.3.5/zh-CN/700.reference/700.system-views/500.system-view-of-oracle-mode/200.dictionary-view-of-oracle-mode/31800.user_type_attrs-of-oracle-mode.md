# USER_TYPE_ATTRS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

列出当前用户拥有的用户自定义类型的属性信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|      **字段名称**      |    **类型**     | **是否可以为 NULL** |          **描述**          |
|--------------------|---------------|----------------|--------------------------|
| TYPE_NAME          | VARCHAR2(64)  | NO             | 类型名                      |
| ATTR_NAME          | VARCHAR2(64)  | NO             | 类型属性名                    |
| ATTR_TYPE_MOD      | VARCHAR2(7)   | NO             | 目前暂未使用该字段  |
| ATTR_TYPE_OWNER    | VARCHAR2(128) | NO             | 属性的类型拥有者                 |
| ATTR_TYPE_NAME     | VARCHAR2(324) | NO             | 属性的类型名                   |
| LENGTH             | NUMBER(38)    | NO             | 属性类型长度                   |
| NUMBER_PRECISION   | NUMBER(38)    | NO             | 属性数值类型的长度                |
| SCALE              | NUMBER(38)    | NO             | 数值类型小数点右边的数据长度           |
| CHARACTER_SET_NAME | CHAR(7)       | NO             | 属性类型的字符集名称：<ul><li>`CHAR_CS`</li> <li> `NCHAR_CS` </li></ul>   |
| ATTR_NO            | NUMBER(38)    | NO             | 属性编号                     |
| INHERITED          | CHAR(2)       | NO             | 该属性是否继承自父类型              |
| CHAR_USED          | CHAR(1)       | YES            | 表示该属性是按照字节长度（B）还是字符长度（C）定义 |

## 查询示例

查看当前用户拥有的用户自定义类型的属性信息。

```shell
obclient [SYS]> SELECT * FROM SYS.USER_TYPE_ATTRS;
```

查询结果如下：

```shell
+-------------+--------------+---------------+-----------------+----------------+--------+------------------+-------+--------------------+---------+-----------+-----------+
| TYPE_NAME   | ATTR_NAME    | ATTR_TYPE_MOD | ATTR_TYPE_OWNER | ATTR_TYPE_NAME | LENGTH | NUMBER_PRECISION | SCALE | CHARACTER_SET_NAME | ATTR_NO | INHERITED | CHAR_USED |
+-------------+--------------+---------------+-----------------+----------------+--------+------------------+-------+--------------------+---------+-----------+-----------+
| PROJECT_OBJ | PROJECT_NAME | NULL          | NULL            | VARCHAR2       |    100 |                2 |    -1 | CHAR_CS            |       1 | NO        | C         |
| PROJECT_OBJ | STATUS       | NULL          | NULL            | VARCHAR2       |     20 |                2 |    -1 | CHAR_CS            |       2 | NO        | C         |
+-------------+--------------+---------------+-----------------+----------------+--------+------------------+-------+--------------------+---------+-----------+-----------+
2 rows in set
```
