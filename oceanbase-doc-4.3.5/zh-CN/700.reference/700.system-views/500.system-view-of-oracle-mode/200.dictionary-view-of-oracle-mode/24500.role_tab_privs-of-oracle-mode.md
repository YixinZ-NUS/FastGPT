# ROLE_TAB_PRIVS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.77 版本开始引入。</p>
</main>

## 功能描述

描述授予角色的表特权。仅提供关于用户可以访问的角色的信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|  **字段名称**   |    **类型**     | **是否可以为 NULL** |       **描述**       |
|-------------|---------------|----------------|--------------------|
| ROLE        | VARCHAR2(128) | NO             | 角色名称               |
| OWNER       | VARCHAR2(128) | NO             | 对象的所有者             |
| TABLE_NAME  | VARCHAR2(128) | NO             | 对象的名称              |
| COLUMN_NAME | VARCHAR2(128) | NO             | 列的名称（如果使用）         |
| PRIVILEGE   | VARCHAR2(40)  | NO             | 授予角色的对象权限          |
| GRANTABLE   | VARCHAR2(3)   | NO             | 该角色是否被授予了 ADMIN 选项 |

## 查询示例

查看用户可以访问的角色中，角色 `ROLE1` 被授予的表权限。

```shell
obclient [SYS]> SELECT * FROM SYS.ROLE_TAB_PRIVS WHERE ROLE = 'ROLE1';
```

查询结果如下：

```shell
+-------+-------+------------+-------------+-----------+-----------+
| ROLE  | OWNER | TABLE_NAME | COLUMN_NAME | PRIVILEGE | GRANTABLE |
+-------+-------+------------+-------------+-----------+-----------+
| ROLE1 | SYS   | TBL2       | NULL        | UPDATE    | NO        |
| ROLE1 | SYS   | TBL2       | NULL        | SELECT    | NO        |
+-------+-------+------------+-------------+-----------+-----------+
2 rows in set
```
