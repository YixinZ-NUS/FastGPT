# ROLE_SYS_PRIVS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.77 版本开始引入。</p>
</main>

## 功能描述

描述授予角色的系统特权。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|   **字段名称**   |    **类型**     | **是否可以为 NULL** |       **描述**       |
|--------------|---------------|----------------|--------------------|
| ROLE         | VARCHAR2(128) | NO             | 角色名称               |
| PRIVILEGE    | VARCHAR2(40)  | NO             | 授予该角色的系统权限         |
| ADMIN_OPTION | VARCHAR2(3)   | NO             | 该角色是否被授予了 ADMIN 选项 |

## 查询示例

查看角色 `ROLE2` 被授予的系统权限。

```shell
obclient [SYS]> SELECT * FROM ROLE_SYS_PRIVS WHERE ROLE = 'ROLE2';
```

查询结果如下：

```shell
+-------+-----------------+--------------+
| ROLE  | PRIVILEGE       | ADMIN_OPTION |
+-------+-----------------+--------------+
| ROLE2 | CREATE SEQUENCE | NO           |
+-------+-----------------+--------------+
1 row in set
```
