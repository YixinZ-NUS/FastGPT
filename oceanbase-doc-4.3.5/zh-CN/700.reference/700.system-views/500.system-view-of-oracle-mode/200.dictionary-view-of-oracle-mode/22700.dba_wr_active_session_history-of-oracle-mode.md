# DBA_WR_ACTIVE_SESSION_HISTORY

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该视图从 V4.2.1 版本开始引入。</p>
</main>

## 功能描述

视图 `DBA_WR_ACTIVE_SESSION_HISTORY` 用于展示本租户持久化后的 ASH 数据。
