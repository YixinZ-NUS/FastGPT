# DBA_TAB_COMMENTS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

展示当前租户下所有表、视图的注释。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|  **字段名称**  |     **类型**     | **是否可以为 NULL** |  **描述**  |
|------------|----------------|----------------|----------|
| OWNER      | VARCHAR2(128)  | NO             | 表、视图的拥有者 |
| TABLE_NAME | VARCHAR2(128)  | NO             | 表、视图的名字  |
| TABLE_TYPE | VARCHAR2(11)   | NO             | 对象类型     |
| COMMENTS   | VARCHAR2(4000) | NO             | 对象注释     |

## 查询示例

查看当前租户下的表 `STUDENT` 的注释信息。

```shell
obclient [SYS]> SELECT * FROM SYS.DBA_TAB_COMMENTS WHERE TABLE_NAME='STUDENT';
```

返回结果如下：

```shell
+-------+------------+------------+---------------------------+
| OWNER | TABLE_NAME | TABLE_TYPE | COMMENTS                  |
+-------+------------+------------+---------------------------+
| SYS   | STUDENT    | TABLE      | student information table |
+-------+------------+------------+---------------------------+
1 row in set
```
