# TENANT_VIRTUAL_SESSION_VARIABLE

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.77 版本开始引入。</p>
</main>

## 功能描述

展示当前租户下的所有 Session 变量信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|   **字段名称**    |     **类型**      | **是否可以为 NULL** |   **描述**    |
|---------------|-----------------|----------------|-------------|
| VARIABLE_NAME | VARCHAR2(128)   | NO             | Session 变量名 |
| VALUE         | VARCHAR2(65536) | YES            | Session 变量值 |

## 查询示例

查看当前租户下的 Session 变量 `ob_query_timeout` 的值。

```shell
obclient[SYS]> SELECT * FROM SYS.TENANT_VIRTUAL_SESSION_VARIABLE WHERE VARIABLE_NAME = 'ob_query_timeout';
```

查询结果如下：

```shell
+------------------+----------+
| VARIABLE_NAME    | VALUE    |
+------------------+----------+
| ob_query_timeout | 10000000 |
+------------------+----------+
1 row in set
```
