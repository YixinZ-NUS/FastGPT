# USER_TAB_STATS_HISTORY

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V3.2.1 版本开始引入。</p>
</main>

## 功能描述

展示当前用户拥有的所有表的表统计修改历史记录。

## 字段说明

|     **字段名称**      |           **类型**            | **是否可以为 NULL** |  **描述**   |
|-------------------|-----------------------------|----------------|-----------|
| TABLE_NAME        | VARCHAR2(128)               | NO             | 表名        |
| PARTITION_NAME    | VARCHAR2(128)               | YES            | 分区名称      |
| SUBPARTITION_NAME | VARCHAR2(128)               | YES            | 子分区名称     |
| STATS_UPDATE_TIME | TIMESTAMP(6) WITH TIME ZONE | YES            | 更新统计信息的时间 |

## 查询示例

查看当前用户拥有的表中，表 `T_SUBPART` 的表统计修改历史记录。

```shell
obclient [SYS]> SELECT * FROM SYS.USER_TAB_STATS_HISTORY WHERE TABLE_NAME='T_SUBPART';
```
