# NLS_DATABASE_PARAMETERS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

展示 DataBase 级别（OceanBase 数据库租户级）的 NLS 参数。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

| **字段名称**  |     **类型**      | **是否可以为 NULL** | **描述** |
|-----------|-----------------|----------------|--------|
| PARAMETER | VARCHAR2(128)   | NO             | 参数名称   |
| VALUE     | VARCHAR2(65536) | YES            | 参数取值   |

## 查询示例

查看当前租户下 DataBase 级别的 NLS 参数。

```shell
obclient> SELECT * FROM SYS.NLS_DATABASE_PARAMETERS;
```

查询结果如下：

```shell
+-------------------------+----------------------------------+
| PARAMETER               | VALUE                            |
+-------------------------+----------------------------------+
| NLS_DATE_FORMAT         | YYYY-MM-DD HH24:MI:SS            |
| NLS_TIMESTAMP_FORMAT    | YYYY-MM-DD HH24:MI:SS.FF         |
| NLS_TIMESTAMP_TZ_FORMAT | YYYY-MM-DD HH24:MI:SS.FF TZR TZD |
| NLS_TERRITORY           | AMERICA                          |
| NLS_SORT                | BINARY                           |
| NLS_COMP                | BINARY                           |
| NLS_CHARACTERSET        | AL32UTF8                         |
| NLS_NCHAR_CHARACTERSET  | AL16UTF16                        |
| NLS_DATE_LANGUAGE       | AMERICAN                         |
| NLS_LENGTH_SEMANTICS    | BYTE                             |
| NLS_NCHAR_CONV_EXCP     | FALSE                            |

| NLS_NUMERIC_CHARACTERS  | .,                               |
+-------------------------+----------------------------------+
13 rows in set
```
