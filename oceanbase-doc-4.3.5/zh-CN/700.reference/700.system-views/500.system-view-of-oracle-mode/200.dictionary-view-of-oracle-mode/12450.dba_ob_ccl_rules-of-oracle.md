# DBA_OB_CCL_RULES

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.3.5 BP3 版本开始引入。</p>
</main>

## 功能描述

用于记录用户租户当前全部的限流规则。

## 字段描述

| 字段名称 | 类型 | 是否可以为 NULL | 描述 |
| --- | --- | --- | --- |
| TENANT_ID                | NUMBER(38)        | NO   |租户 ID |
| CCL_RULE_ID              | NUMBER(38)        | NO   |限流规则 ID|
| CCL_RULE_NAME            | VARCHAR2(128)     | NO   |限流规则名称 |
| AFFECT_USER_NAME         | VARCHAR2(64)      | NO   |限流生效的用户|
| AFFECT_HOST              | VARCHAR2(128)     | NO   |限流生效的 IP 地址 |
| AFFECT_FOR_ALL_DATABASES | NUMBER(38)        | NO   |是否对全部数据库生效|
| AFFECT_FOR_ALL_TABLES    | NUMBER(38)        | NO   |是否对全部表生效|
| AFFECT_DATABASE          | VARCHAR2(128)     | YES  |限流生效的数据库 |
| AFFECT_TABLE             | VARCHAR2(256)     | YES  |限流生效的表|
| AFFECT_DML               | NUMBER(38)        | NO   |限流生效的 DML 类型语句 |
| AFFECT_SCOPE             | NUMBER(38)        | NO   |限流生效粒度:<li>规则级别: 0 </li><li>Format SQLID 级别:1</li>|
| CCL_KEYWORDS             | VARCHAR2(1048576) | NO   |限流关键字|
| MAX_CONCURRENCY          | NUMBER(38)        | NO   |限流的并发度上限|

## 查询示例

用户租户查询本租户的限流规则。

```shell
obclient > SELECT * FROM DBA_OB_CCL_RULES;
```
