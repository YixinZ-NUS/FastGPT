# DBA_CONTEXT

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该视图从 V3.2.4 版本开始引入。</p>
</main>

## 功能描述

展示当前租户下的所有上下文命名空间信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|     字段名称      |      类型       | 是否可以为 NULL |          描述           |
|---------------|---------------|------------|-----------------------|
| NAMESPACE     | VARCHAR2(128) | NO         | 上下文命名空间的名称            |
| SCHEMA        | VARCHAR2(128) | NO         | 可以使用此命名空间设置属性所属包的用户名称 |
| PACKAGE       | VARCHAR2(128) | NO         | 可以使用此命名空间设置属性的指定包的包名  |
| TYPE          | VARCHAR2(22)  | NO         | 上下文创建的类型              |
| ORIGIN_CON_ID | NUMBER        | NO         | 租户 ID                 |
| TRACKING | VARCHAR2(3)        | NO         | 仅用于兼容，目前值固定为 `YES`                |

## 查询示例

查看当前租户下的所有上下文命名空间信息。

```shell
obclient [SYS]> SELECT * FROM SYS.DBA_CONTEXT;
```

查询结果如下：

```shell
+------------+--------+-------------------+------------------+---------------+----------+
| NAMESPACE  | SCHEMA | PACKAGE           | TYPE             | ORIGIN_CON_ID | TRACKING |
+------------+--------+-------------------+------------------+---------------+----------+
| SALES_CTX  | SYS    | SET_SALES_CTX_PKG | ACCESSED LOCALLY |          1004 | YES      |
| REGION_CTX | SYS    | REGION_CTX_PKG    | ACCESSED LOCALLY |          1004 | YES      |
| DEPT_CTX   | SYS    | DEPT_CTX_PKG      | ACCESSED LOCALLY |          1004 | YES      |
+------------+--------+-------------------+------------------+---------------+----------+
3 rows in set
```
