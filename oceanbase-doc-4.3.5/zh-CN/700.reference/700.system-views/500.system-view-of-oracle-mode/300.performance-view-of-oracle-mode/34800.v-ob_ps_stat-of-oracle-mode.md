# V$OB_PS_STAT

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

监控当前租户在当前 OBServer 节点上的 PS Cache 的整体状态，包括命中率、内存占用、缓存的 Item 数目。

## 字段说明

|   **字段名称**   |    **类型**    | **是否可以为 NULL** |       **描述**        |
|--------------|--------------|----------------|---------------------|
| SVR_IP       | VARCHAR2(46) | NO             | 服务器 IP 地址           |
| SVR_PORT     | NUMBER(38)   | NO             | 服务器端口号              |
| STMT_COUNT   | NUMBER(38)   | NO             | PS Cache 中目前缓存了多少语句 |
| HIT_COUNT    | NUMBER(38)   | NO             | PS Cache 总的命中次数     |
| ACCESS_COUNT | NUMBER(38)   | NO             | 访问 PS Cache 的总次数    |
| MEM_HOLD     | NUMBER(38)   | NO             | PS Cache 占用了多少内存    |

## 查询示例

用户租户下，查看本租户在当前节点上的 PS Cache 的整体状态。

```shell
obclient [SYS]> SELECT * FROM SYS.V$OB_PS_STAT;
```

查询结果如下：

```shell
+-----------+----------------+----------+------------+-----------+--------------+----------+
| TENANT_ID | SVR_IP         | SVR_PORT | STMT_COUNT | HIT_COUNT | ACCESS_COUNT | MEM_HOLD |
+-----------+----------------+----------+------------+-----------+--------------+----------+
|      1004 | 11.xxx.xxx.xxx |    28825 |          1 |         0 |            1 |    16384 |
+-----------+----------------+----------+------------+-----------+--------------+----------+
1 row in set
```
