# V$NLS_PARAMETERS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.30 版本开始引入。</p>
</main>

## 功能描述

展示 Session 级别的所有 NLS 参数。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

| **字段名称**  |     **类型**      | **是否可以为 NULL** | **描述** |
|-----------|-----------------|----------------|--------|
| PARAMETER | VARCHAR2(128)   | NO             | 参数名称   |
| VALUE     | VARCHAR2(65536) | YES            | 参数取值   |
| CON_ID    | NUMBER          | NO             | 无实际含义  |

## 查询示例

查看当前租户下 Session 级别的所有 NLS 参数。

```shell
obclient> select * from V$NLS_PARAMETERS;
```
