# V$INSTANCE

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V2.2.77 版本开始引入。</p>
</main>

## 功能描述

该视图展示了当前数据库实例的状态。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该视图仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 字段说明

|   **字段名称**   |   **类型**   | **是否可以为 NULL** |                        **描述**                        |
|------------------|--------------|----------------|-------------------------------------------------------------|
| INST_ID          | NUMBER       | NO             | 数据库实例对应的 ID                                          |
| INSTANCE_NUMBER  | NUMBER       | NO             | 目前暂不支持该字段，当前该字段默认为 NULL      |
| INSTANCE_NAME    | VARCHAR2(16) | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                        |
| HOST_NAME        | VARCHAR2(64) | NO             | 实例所在的 Server 地址                                      |
| VERSION          | VARCHAR2(17) | NO             | 数据库实例的版本                                             |
| STARTUP_TIME     | DATE         | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                      |
| STATUS           | VARCHAR2(12) | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                        |
| PARALLEL         | VARCHAR2(3)  | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                                                                     |
| THREAD#          | NUMBER       | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                     |
| ARCHIVER         | VARCHAR2(7)  | NO             | 目前暂不支持该字段，当前该字段默认为 NULL   |
| LOG_SWITCH_WAIT  | VARCHAR2(15) | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                   |
| LOGINS           | VARCHAR2(10) | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                              |
| SHUTDOWN_PENDING | VARCHAR2(3)  | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                              |
| DATABASE_STATUS  | VARCHAR2(17) | NO             | 数据库的状态                                               |
| INSTANCE_ROLE    | VARCHAR2(18) | NO             | 目前暂不支持该字段，当前该字段默认为 NULL |
| ACTIVE_STATE     | VARCHAR2(9)  | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                      |
| BLOCKED          | VARCHAR2(2)  | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                                                                   |
| CON_ID           | NUMBER       | NO             | 目前暂不支持该字段，当前该字段默认为 NULL           |
| INSTANCE_MODE    | VARCHAR2(11) | NO             | 目前暂不支持该字段，当前该字段默认为 NULL           |
| EDITION          | VARCHAR2(7)  | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                      |
| FAMILY           | VARCHAR2(80) | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                       |
| DATABASE_TYPE    | VARCHAR2(15) | NO             | 目前暂不支持该字段，当前该字段默认为 NULL                     |
