# GV$OB_SS_LOCAL_CACHE

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.3.4 版本开始引入。</p>
</main>

## 功能描述

展示共享存储模式下当前租户在每个 OBServer 节点的本地缓存中各类缓存数据的相关指标。

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图仅适用于共享存储（Shared-Storage，SS）模式。无共享（Shared-Nothing，SN）模式下，该视图的查询结果为空。</p>
</main>

## 字段说明

|      字段名称    |         类型          | 是否可以为 NULL |                       描述      |
|-----------------|-----------------------|----------------|---------------------------------|
| SVR_IP          | VARCHAR2(46)          | NO             |  OBServer 节点的 IP 地址    |
| SVR_PORT        | NUMBER(38)            | NO             |  OBServer 节点的端口号      |
| TENANT_ID       | NUMBER(38)            | NO             |  租户 ID                   |
| CACHE_NAME      | VARCHAR2(128)         | NO             |  本地缓存中某类数据缓存的名称，主要包括以下几类：<ul><li>`ss_micro_cache`：微块缓存</li> <li>`ss_local_cache_tmpfile`：临时文件缓存</li> <li>`ss_local_cache_major_macro`：基线宏块缓存</li></ul>    |
| PRIORITY        | bigint(20)            | NO             |  缓存优先级，值越大优先级越高。当前该字段暂未使用，值默认为 `0`     |
| HIT_RATIO       | NUMBER(38,3)          | NO             |  缓存命中率                    |
| TOTAL_HIT_CNT   | NUMBER(38)            | NO             |  缓存命中的总次数               |
| TOTAL_MISS_CNT  | NUMBER(38)            | NO             |  缓存未命中的总次数             |
| HOLD_SIZE       | NUMBER(38)            | NO             |  缓存中所存放的对象的总量，单位为字节    |
| ALLOC_DISK_SIZE | NUMBER(38)            | NO             |  已分配的磁盘空间，单位为字节    |
| USED_DISK_SIZE  | NUMBER(38)            | NO             |  已使用的磁盘空间（可能存在磁盘空间分配后暂未使用的情况），单位为字节    |
| USED_MEM_SIZE   | NUMBER(38)            | NO             |  已使用的内存空间，单位为字节                 |
