# GV$OB_RESULT_CACHE_OBJECTS

<main id="notice" type='explain'>
 <h4>说明</h4>
 <ul>对于 V4.3.x 版本，该视图从 V4.3.5 BP4 版本开始引入。</li></ul>
</main>

## 功能描述

该视图用于展示所有 OBServer 节点上标记 `result_cache` 的 `udf` 执行期建立的 `result cache` 中缓存的对象信息。

## 字段描述

| **字段名称** | **类型**  | **是否可以为 NULL** | **描述**                               |
|------------|-------------|---------------------|----------------------------------------|
|TENANT_ID|bigint(20)|NO|租户 ID<ul><li>1：表示系统租户 ID。</li><li>其他值：表示用户租户或者 Meta 租户 ID。</li></ul>|
|SVR_IP|varchar(46)|NO|副本所在的节点地址。|
|SVR_PORT|bigint(20)|NO|副本所在的节点端口。|
|CACHE_OBJECT_ID|bigint(20)|NO|lib cache 分配的缓存对象 ID。|
|UDF_NAME|longtext|NO|UDF 对象名称。|
|FIRST_LOAD_TIME|timestamp(6)|NO|缓存对象第一次被加载时间。|
|LAST_ACTIVE_TIME| timestamp(6)|NO| 上一次访问缓存时间。|
|HIT_COUNT| bigint(20)| NO| 缓存命中次数。|
|CACHE_OBJ_SIZE| bigint(20)|NO| 缓存对象占用内存大小。|
|BUILD_TIME| bigint(20)|NO| UDF 执行耗时。|
|OBJECT_TYPE| longtext|NO| 缓存对象类型，对 result cache 而言目前只有 UDF_RESULT。|
|OBJECT_ID| bigint(20)|NO| udf routine id。|
|DB_ID| bigint(20)|NO| UDF 所属 DB ID。|
|HASH| bigint(20)|NO| 缓存对象在 hash 表中的 hash 值。|
|SYS_VARS| varchar(4096)|NO| 影响 UDF 执行的系统变量集合。|
|CONFIGS| varchar(4096)|NO| 影响 UDF 执行的配置项集合。|

## 查询示例

执行如下命令 查询 `result cache` 中缓存的对象信息。

```shell
obclient [SYS]> SELECT * FROM SYS.GV$OB_RESULT_CACHE_OBJECTS;
```
