# V$OB_STORAGE_CACHE_TASKS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>对于 V4.3.5 版本，该视图从 V4.3.5 BP2 版本开始引入。</p>
</main>

## 功能描述

展示当前租户在当前 OBServer 节点上的缓存任务信息。

<main id="notice" type='explain'>
<h4>说明</h4>
<p>仅适用于共享存储（Shared-Storage，SS）模式。无共享（Shared-Nothing，SN）模式下，该视图的查询结果为空。</p>
</main>

## 字段说明

|      字段名称    |         类型          | 是否可以为 NULL |                       描述      |
|-----------------|-----------------------|----------------|---------------------------------|
| SVR_IP          | varchar(46)           | NO             |  OBServer 节点的 IP 地址    |
| SVR_PORT        | bigint(20)            | NO             |  OBServer 节点的端口号      |
| TENANT_ID       | bigint(20)            | NO             |  Tablet 所属的租户 ID    |
| TABLET_ID       | bigint(20)            | NO             |  Tablet ID   |
| STATUS          | varchar(64)           | NO             |  缓存任务的状态：<ul><li>`WAITING`：任务正在初始化</li> <li>`DONIG`：任务正在执行中</li> <li>`FINISHED`：任务已执行完成</li> <li>`SUSPENDED`：任务暂停</li> <li>`CANCELED`：任务被取消</li> <li>`FAILED`：任务执行失败</li></ul>    |
| SPEED           | varchar(29)           | NO             |  Tablet 的拉取速度     |
| START_TIME      | timestamp(6)          | NO             |  缓存任务开始时间    |
| COMPLETE_TIME   | timestamp(6)          | NO             |  缓存任务结束时间    |
| RESULT          | bigint(20)            | NO             |  如果任务执行失败，展示对应的错误码    |
| COMMENT         | varchar(4096)         | NO             |  任务失败后的详细错误描述，包含 `trace_id` 信息    |

## 查询示例

SS 模式下，用户租户查看本租户在当前 OBServer 节点上的缓存任务信息。

```shell
obclient> SELECT * FROM oceanbase.V$OB_STORAGE_CACHE_TASKS;
```
