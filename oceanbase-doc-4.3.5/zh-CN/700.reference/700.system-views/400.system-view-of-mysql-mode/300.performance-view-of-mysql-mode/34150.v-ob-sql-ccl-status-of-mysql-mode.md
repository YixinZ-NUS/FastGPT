# V$OB_SQL_CCL_STATUS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.3.5 BP3 版本开始引入。</p>
</main>

## 功能描述

用于记录当前时刻本租户的当前节点命中对应限流规则的 SQL 数量，用于观测当前系统已设置的限流规则的影响。

## 字段说明

| 字段名称 | 类型 | 是否可以为 NULL | 描述 |
|---------|-------|-------------|--------|
| CON_ID              | bigint(20)  | NO   |租户 ID      |
| SVR_IP              | varchar(46) | NO   |OBServer 节点的 IP 地址     |
| SVR_PORT            | bigint(20)  | NO   |BServer 节点的端口号      |
| CCL_RULE_ID         | bigint(20)  | NO   | 生效的限流规则 ID     |
| FORMAT_SQLID        | varchar(32) | NO   |限流生效粒度:<li>规则级别: 0 </li><li>SQL_ID 级别:1</li>      |
| CURRENT_CONCURRENCY | bigint(20)  | NO   | 当前正在执行中的，命中该限流规则的 SQL 个数     |
| MAX_CONCURRENCY     | bigint(20)  | NO   |  此限流规则的并发度上限    |

## 查询示例

```shell
obclient> SELECT * FROM oceanbase.GV$OB_SQL_CCL_STATUS;
```
