# information_schema.CONNECTION_CONTROL_FAILED_LOGIN_ATTEMPTS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V3.2.1 版本开始引入。</p>
</main>

## 功能描述

展示不同用户登录失败的尝试次数。

## 字段说明

|    **字段名称**     |    **类型**    | **是否可以为 NULL** |                             **描述**                             |
|-----------------|--------------|----------------|----------------------------------------------------------------|
| USERHOST        | varchar(261) | NO             | 表示登录失败的用户名和主机信息的组合，其格式为 `<user_name>@<hostname or ip address>` |
| FAILED_ATTEMPTS | bigint(20)   | NO             | 登录失败的尝试次数                                                      |

## 查询示例

查看用户登录失败的尝试次数。

```shell
obclient> SELECT * FROM information_schema.CONNECTION_CONTROL_FAILED_LOGIN_ATTEMPTS;
```

查询结果如下：

```shell
+-------------+-----------------+
| USERHOST    | FAILED_ATTEMPTS |
+-------------+-----------------+
| 'test'@'%'  |               5 |
+-------------+-----------------+
1 row in set
```
