# oceanbase.DBA_OB_DYNAMIC_PARTITION_TABLES

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该视图从 V4.3.5 BP2 版本开始引入。</p>
</main>

## 功能描述

展示本租户的动态分区表信息。

## 字段说明

|        字段        |      类型      | 是否为 NULL | 描述 |
|--------------------|---------------|-------------|------|
| DATABASE_NAME      | varchar(128)  | NO          | 动态分区表所在 Database 名。|
| TABLE_NAME         | varchar(256)  | NO          | 动态分区表表名。|
| TABLE_ID           | bigint(20)    | NO          | 动态分区表 ID。|
| MAX_HIGH_BOUND_VAL | varchar(4096) | NO          | 最大 Range 分区上界。|
| ENABLE             | varchar(4096) | NO          | 动态分区管理是否开启。|
| TIME_UNIT          | varchar(4096) | NO          | 动态分区管理的 `time_unit`。|
| PRECREATE_TIME     | varchar(4096) | NO          | 动态分区管理的 `precreate_time`。|
| EXPIRE_TIME        | varchar(4096) | NO          | 动态分区管理的 `expire_time`。|
| TIME_ZONE          | varchar(4096) | NO          | 动态分区管理的 `time_zone`。|
| BIGINT_PRECISION   | varchar(4096) | NO          | 动态分区管理的 `bigint_precision`。|

## 查询示例

查看本租户的一条动态分区表信息。

```shell
obclient> SELECT * FROM oceanbase.DBA_OB_DYNAMIC_PARTITION_TABLES LIMIT 1\G
```

查询结果如下：

```shell
*************************** 1. row ***************************
     DATABASE_NAME: db_test
        TABLE_NAME: tbl2
          TABLE_ID: 502062
MAX_HIGH_BOUND_VAL: '2025-04-15 14:00:00'
            ENABLE: TRUE
         TIME_UNIT: HOUR
    PRECREATE_TIME: 3HOUR
       EXPIRE_TIME: 1DAY
         TIME_ZONE: +8:00
  BIGINT_PRECISION: NONE
1 row in set
```
