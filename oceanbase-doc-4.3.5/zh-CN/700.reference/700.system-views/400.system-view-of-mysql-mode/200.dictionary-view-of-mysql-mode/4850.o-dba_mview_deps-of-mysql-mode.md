# oceanbase.DBA_MVIEW_DEPS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.3.5 BP2 版本开始引入。</p>
</main>

## 功能描述

该视图用于展示当前租户物化视图的依赖关系。

## 字段说明

| **字段名称** |   **类型**  | **是否可以为 NULL** | **描述** |
|-------------|--------------|---------------------|---------|
| MVIEW_OWNER | varchar(128) | NO   | 物化视图所有者名称。|
| MVIEW_NAME  | varchar(256) | NO   | 物化视图名称。|
| DEP_OWNER   | varchar(128) | NO   | 依赖对象所有者名称。|
| DEP_NAME    | varchar(256) | NO   | 依赖对象名称。|
| DEP_TYPE    | varchar(64)  | NO   | 依赖对象类型：<ul><li>`MV`：物化视图</li><li>`TABLE`：普通表</li><li>`EXTERNAL TABLE`：外表</li></ul>|

## 查询示例

```shell
obclient [(none)]> SELECT * FROM oceanbase.DBA_MVIEW_DEPS;
```

返回结果如下：

```shell
+-------------+------------------+-----------+---------------+----------------+
| MVIEW_OWNER | MVIEW_NAME       | DEP_OWNER | DEP_NAME      | DEP_TYPE       |
+-------------+------------------+-----------+---------------+----------------+
| db_test     | mv_mv1_tbl3_tbl4 | db_test   | mv1_tbl3_tbl4 | MV             |
| db_test     | mv1_tbl3_tbl4    | db_test   | tbl4          | TABLE          |
| db_test     | mv1_tbl3_tbl4    | db_test   | tbl3          | TABLE          |
| db_test     | mv_ext_tbl1      | db_test   | ext_tbl1      | EXTERNAL TABLE |
+-------------+------------------+-----------+---------------+----------------+
4 rows in set
```
