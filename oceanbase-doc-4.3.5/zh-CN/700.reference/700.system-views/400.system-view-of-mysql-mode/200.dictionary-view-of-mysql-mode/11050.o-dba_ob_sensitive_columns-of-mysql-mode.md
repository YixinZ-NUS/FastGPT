# oceanbase.DBA_OB_SENSITIVE_COLUMNS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.3.5 BP3 版本开始引入。</p>
</main>

## 功能描述

展示当前租户下的所有受敏感规则保护的列。

## 字段说明

|        字段名称         |      类型       | 是否可以为 NULL |                      描述                                |
|---------------------|---------------|------------|-----------------------------------------------------------------|
| RULE_NAME         | varchar(256)    | NO         | 敏感规则名称                                           |
| DATABASE_NAME      | varchar(128)    | NO         | 数据库名称                                        |
| TABLE_NAME         | varchar(128)    | NO         | 表名称                                      |
| COLUMN_NAME        | varchar(128)    | NO         | 列名称                                                     |

## 查询示例

查看当前租户下的所有受敏感规则保护的列。

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.DBA_OB_SENSITIVE_COLUMNS;
```

查询结果如下：

```shell
+-----------+---------------+------------+-------------+
| RULE_NAME | DATABASE_NAME | TABLE_NAME | COLUMN_NAME |
+-----------+---------------+------------+-------------+
| r1        | test          | t2         | c           |
| r1        | test          | t1         | a           |
+-----------+---------------+------------+-------------+
2 row in set
```
