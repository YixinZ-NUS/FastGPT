# oceanbase.DBA_OB_AUTO_INCREMENT

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

展示当前租户下所有自增列的信息。

## 字段说明

| 字段名称              | 类型                | 是否可以为 NULL | 描述              |
|----------------------|---------------------|----------------|-------------------|
| CREATE_TIME          | datetime(6)         | YES            | 自增列创建时间    |
| MODIFY_TIME          | datetime(6)         | YES            | 自增列修改时间    |
| AUTO_INCREMENT_KEY   | bigint(20)          | NO             | 自增列所在表的 KEY  |
| COLUMN_ID            | bigint(20)          | NO             | 列 ID              |
| AUTO_INCREMENT_VALUE | bigint(20) unsigned | YES            | 下一个可用的自增值   |
| SYNC_VALUE           | bigint(20) unsigned | NO             | 同步值，即通过 `Insert` 语句显式指定的自增列的值，且该值大于自增列的当前值。|

## 查询示例

查看本租户的所有自增列信息。

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.DBA_OB_AUTO_INCREMENT;
```

查询结果如下：

```shell
+----------------------------+----------------------------+--------------------+-----------+----------------------+------------+
| CREATE_TIME                | MODIFY_TIME                | AUTO_INCREMENT_KEY | COLUMN_ID | AUTO_INCREMENT_VALUE | SYNC_VALUE |
+----------------------------+----------------------------+--------------------+-----------+----------------------+------------+
| 2025-02-20 14:37:01.322987 | 2025-02-20 14:37:07.197611 |             500006 |        16 |              1000001 |          0 |
| 2025-02-20 14:39:18.505040 | 2025-02-20 14:39:18.505040 |             500008 |        16 |                    1 |          0 |
+----------------------------+----------------------------+--------------------+-----------+----------------------+------------+
2 rows in set
```
