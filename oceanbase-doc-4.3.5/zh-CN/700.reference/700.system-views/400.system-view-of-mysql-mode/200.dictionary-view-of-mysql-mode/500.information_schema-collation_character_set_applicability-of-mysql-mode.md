# information_schema.COLLATION_CHARACTER_SET_APPLICABILITY

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V1.4 版本开始引入。</p>
</main>

## 功能描述

记录 character 和 collation 之间的对应关系。

## 字段说明

|      **字段名称**      |    **类型**    | **是否可以为 NULL** | **描述** |
|--------------------|--------------|----------------|--------|
| COLLATION_NAME     | varchar(128) | NO             | 比对方法名称 |
| CHARACTER_SET_NAME | varchar(128) | NO             | 字符集名称  |

## 查询示例

查看数据库中 Character 和 Collation 之间的对应关系，展示前 10 条记录。

```shell
obclient [infotest]> SELECT * FROM information_schema.COLLATION_CHARACTER_SET_APPLICABILITY LIMIT 10;
```

查询结果如下：

