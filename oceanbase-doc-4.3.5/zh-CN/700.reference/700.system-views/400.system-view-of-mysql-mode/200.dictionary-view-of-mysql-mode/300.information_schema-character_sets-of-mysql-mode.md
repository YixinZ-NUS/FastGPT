# information_schema.CHARACTER_SETS

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V1.4 版本开始引入。</p>
</main>

## 功能描述

展示 OceanBase 数据库的字符集信息。

## 字段说明

|       **字段名称**       |    **类型**    | **是否可以为 NULL** | **描述**  |
|----------------------|--------------|----------------|---------|
| CHARACTER_SET_NAME   | varchar(128) | NO             | 字符集名称   |
| DEFAULT_COLLATE_NAME | varchar(128) | NO             | 字符集比对方法 |
| DESCRIPTION          | varchar(64)  | NO             | 字符集描述 |
| MAXLEN               | bigint(20)   | NO             | 字符集最大长度 |

## 查询示例

查看 OceanBase 数据库的字符集信息。

```shell
obclient [oceanbase]> SELECT * FROM information_schema.CHARACTER_SETS;
```
