# 查找某个租户中执行全表扫描的 SQL

运行如下语句可以查询某个租户中执行全表扫描的 SQL：

```shell
obclient> SELECT query_sql
FROM oceanbase.GV$OB_SQL_AUDIT
WHERE table_scan = 1 AND tenant_id = 1001
GROUP BY sql_id;
```

返回结果如下：

```shell
+-------------------------------+
| query_sql                     |
+-------------------------------+
| SHOW VARIABLES LIKE 'version' |
+-------------------------------+
```
