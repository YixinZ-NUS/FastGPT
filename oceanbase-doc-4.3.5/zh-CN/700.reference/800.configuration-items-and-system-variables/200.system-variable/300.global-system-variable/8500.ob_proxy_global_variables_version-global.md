# ob_proxy_global_variables_version

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该变量从 V2.2.77 版本开始引入。</p>
</main>

## 功能描述

`ob_proxy_global_variables_version` 在 obproxy 与 OBServer 节点内部通信时使用。用于设置 OBServer 节点返回给 obproxy 所需的全局变量的最新版本信息。

## 属性说明

| **属性** |                                                 **描述**                                                  |
|--------|---------------------------------------------------------------------------------------------------------|
| 参数类型   | Int                  |
| 默认值    | 0                    |
| 取值范围   | \[0, INT64_MAX)      |
| 生效范围   | Session              |
| 是否可修改  | 否，该变量支持在创建租户时指定，租户创建后不能通过 SQL 语句修改。 |

## 使用说明

不支持用户通过 `SHOW VARIABLES` 语句或视图 `SYS.TENANT_VIRTUAL_SESSION_VARIABLE`（Oracle 模式）及视图 `information_schema.SESSION_VARIABLES`（MySQL 模式）查询该变量的值。用户可以通过查询 `SYS.DBA_OB_SYS_VARIABLES` 视图（Oracle 模式）或视图 `oceanbase.DBA_OB_SYS_VARIABLES`（MySQL 模式）查询该变量的值。

或者，用户也可以通过 `SELECT` 语句返回当前全局变量的最新版本号：

```shell
obclient> SELECT @@ob_proxy_global_variables_version;
```

返回结果如下：

```shell
+-------------------------------------+
| @@ob_proxy_global_variables_version |
+-------------------------------------+
|                                   0 |
+-------------------------------------+
1 row in set
```
