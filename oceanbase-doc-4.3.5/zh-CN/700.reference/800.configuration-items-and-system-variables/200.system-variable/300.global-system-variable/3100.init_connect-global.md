# init_connect

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效。</p>
</main>

## 功能描述

`init_connect` 用于设置服务器为每个连接的客户端执行的字符串，该字符串由一条或多条 SQL 语句组成，用分号分隔。

## 属性说明

| **属性**  |    **描述**  |
|-----------|---------------|
| 参数类型     | Varchar |
| 生效范围     | Global   |
| 是否可修改   | 是，支持通过 `SET` 语句修改。|
