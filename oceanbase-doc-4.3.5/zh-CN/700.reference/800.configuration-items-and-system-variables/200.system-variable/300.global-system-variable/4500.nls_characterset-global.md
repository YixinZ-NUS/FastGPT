# nls_characterset

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该变量从 V2.2.77 版本开始引入。</p>
</main>

## 功能描述

`nls_characterset` 用于查看数据库中 CHAR、VARCHAR2、CLOB 等数据类型的默认字符集。

<main id="notice">
  <h4>功能适用性</h4>
  <p>该变量仅适用于 OceanBase 数据库 Oracle 模式。</p>
</main>

## 权限要求

* 查询变量

  `sys` 租户和所有用户租户均可以使用 `SHOW VARIABLES` 语句或视图 `SYS.TENANT_VIRTUAL_GLOBAL_VARIABLE`（Oracle 模式）查看 Global 系统变量的值。

* 修改变量

  该变量不支持修改。

## 属性说明

| **属性**  | **描述**  |
|---------|---------|
| 参数类型    | Varchar |
| 默认值      | AL32UTF8 |
| 生效范围    | Global |
| 是否可修改  | 否，该变量仅支持只读。|
