# server_uuid

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该变量从 V3.2.4 版本开始引入。</p>
</main>

## 功能描述

`server_uuid` 用于查看 OBServer 节点UUID。

OBServer 节点启动时，会自动创建 UUID。例如：`182d0ebb-1209-11ed-b01e-a237b1d47168`。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该变量仅 MySQL 模式下支持。</p>
</main>

## 权限要求

* 查询变量

  `sys` 租户和所有用户租户均可以使用 `SHOW VARIABLES` 语句或视图 `information_schema.GLOBAL_VARIABLES`（MySQL 模式）查看 Global 系统变量的值。

* 修改变量

  该变量不支持修改。

## 属性说明

| **属性**  |      **描述**      |
|---------|------------------|
| 参数类型    | Varchar             |
| 生效范围    | Global           |
| 是否可修改 | 否，该变量仅支持只读。                 |
