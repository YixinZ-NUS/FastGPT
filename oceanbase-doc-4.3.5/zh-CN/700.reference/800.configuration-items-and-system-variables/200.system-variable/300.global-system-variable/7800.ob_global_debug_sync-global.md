# ob_global_debug_sync

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该变量从 V1.4 版本开始引入。</p>
</main>

## 功能描述

`ob_global_debug_sync` 用于 debug 时设置同步点，设置此变量会影响所有 Session。

## 属性说明

| **属性** |   **描述**   |
|--------|------------|
| 参数类型   | Varchar    |
| 默认值    | 空字符串       |
| 生效范围   | Session     |
| 是否可见   | 否，所有租户均不可见 |

## 使用说明

不支持用户通过 `SHOW VARIABLES` 语句或视图 `SYS.TENANT_VIRTUAL_SESSION_VARIABLE`（Oracle 模式）及视图 `information_schema.SESSION_VARIABLES`（MySQL 模式）查询该变量的值。用户可以通过查询 `SYS.DBA_OB_SYS_VARIABLES` 视图（Oracle 模式）或视图 `oceanbase.DBA_OB_SYS_VARIABLES`（MySQL 模式）查询该变量的值。

仅当 [debug_sync_timeout](../../100.system-configuration-items/300.cluster-level-configuration-items/4900.debug_sync_timeout.md) 配置项的值不为 0 时，该设置才有效。
