# socket

<main id="notice" type='explain'>
  <h4>说明</h4>
  <ul><li>对于 V4.3.x 版本，该变量从 V4.3.5 BP1 版本开始引入。</li><li>对于 V4.2.x 其他版本，该变量从 V4.2.5 BP1 版本开始引入；对于 V4.2.1 版本，该变量从 V4.2.1 BP10 版本开始引入。</li></ul>
</main>

## 功能描述

`socket` 用于查询本地客户端连接 `Socket` 文件的名称。

<main id="notice">
  <h4>功能适用性</h4>
  <p>该变量仅适用于 OceanBase 数据库 MySQL 模式。</p>
</main>

## 权限要求

* 查询变量

  `sys` 租户和所有用户租户均可以使用 `SHOW VARIABLES` 语句或视图 `information_schema.GLOBAL_VARIABLES`（MySQL 模式）查看 Global 系统变量的值。

* 修改变量

  该变量不支持修改。

## 属性说明

| **属性**  |    **描述**  |
|-----------|---------------|
| 参数类型    | Varchar              |
| 默认值      | /run/sql.sock   |
| 生效范围    |  Global  |
| 是否可修改  | 否，该变量仅支持只读。        |

## 使用示例

查询本地客户端连接 `socket` 文件的名称。

```shell
obclient> SHOW VARIABLES LIKE 'socket';
```

查询结果如下：

```shell
+---------------+------------------------------------+
| Variable_name | Value                              |
+---------------+------------------------------------+
| socket        | /home/admin/oceanbase/run/sql.sock |
+---------------+------------------------------------+
1 row in set (0.003 sec)
```
