# query_cache_type

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该变量从 V2.1 版本开始引入。</p>
</main>

## 功能描述

`query_cache_type` 用于设置查询缓存类型。

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该变量当前暂不生效。</p>
</main>

## 属性说明

| **属性**  |                           **描述**                             |
|-----------|----------------------------------------------------------------|
| 参数类型   | Varchar                                                     |
| 默认值     | OFF                                                         |
| 取值范围   | <li> OFF：不缓存或检索结果。   <li> ON：缓存除了 `SELECT SQL_NO_CACHE ...`查询的所有结果。 <li> DEMAND：只缓存 `SELECT SQL_CACHE ...` 查询的结果。    |
| 生效范围    | Global |
| 是否可修改  | 否，该变量仅支持在创建租户时指定，租户创建后不能通过 SQL 语句修改。 |
