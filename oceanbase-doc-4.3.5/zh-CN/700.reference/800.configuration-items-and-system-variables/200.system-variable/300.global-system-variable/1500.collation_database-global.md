# collation_database

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该变量从 V1.4 版本开始引入，该变量仅用于兼容 MySQL，相关功能并不支持，可以查询及设置，设置行为不报错但不生效。</p>
</main>

## 功能描述

`collation_database` 用于指定创建数据库时的默认字符集和字符序。

<main id="notice">
  <h4>功能适用性</h4>
  <p>该变量仅适用于 OceanBase 数据库 MySQL 模式。</p>
</main>

## 权限要求

* 查询变量

  * Global 级别

    `sys` 租户和所有用户租户均可以使用 `SHOW VARIABLES` 语句或视图 `information_schema.GLOBAL_VARIABLES`（MySQL 模式）查看 Global 系统变量的值。

  * Session 级别

    `sys` 租户和所有用户租户均可以使用 `SHOW VARIABLES` 语句或视图 `information_schema.SESSION_VARIABLES`（MySQL 模式）查看 Session 系统变量的值。

## 属性说明

| **属性** |    **描述**   |
|----------|---------------|
| 参数类型    |  String       |
| 默认值      | utf8mb4_general_ci     |
| 取值范围    | <li> utf8mb4_general_ci   <li> utf8mb4_bin   <li> utf8mb4_unicode_ci   <li> gbk_bin   <li> gbk_chinese_ci   <li> gb18030_bin   <li> gb18030_chinese_ci   <li> utf16_general_ci   <li> utf16_bin   <li> utf16_unicode_ci   <li> binary    |
| 生效范围    | <li> GLOBAL   <li> SESSION   |
| 是否可修改  | 否，该变量只支持只读。|
| 是否可以为空  | 是         |
