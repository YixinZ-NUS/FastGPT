# ob_proxy_partition_hit

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该变量从 V2.2.77 版本开始引入。</p>
</main>

## 功能描述

`ob_proxy_partition_hit` 用于设置 obproxy 是否将 SQL 语句请求的数据发送到这些数据主 Partition 所在的 OBServer 节点上。

## 属性说明

| **属性** |                                               **描述**                                                |
|--------|-----------------------------------------------------------------------------------------------------|
| 参数类型   | Bool             |
| 默认值    | 1                |
| 取值范围   | <ul><li> 1：是   </li><li> 0：否 </li></ul>    |
| 生效范围   | Session          |
| 是否可修改  | 否，该变量支持在创建租户时指定，租户创建后不能通过 SQL 语句修改                                                                    |

## 使用说明

不支持用户通过 `SHOW VARIABLES` 语句或视图查询该变量的值。用户可以通过 `SELECT` 语句返回该变量的值：

```shell
obclient> SELECT @@ob_proxy_partition_hit;
```

返回结果如下：

```shell
+--------------------------+
| @@ob_proxy_partition_hit |
+--------------------------+
|                        1 |
+--------------------------+
1 row in set
```
