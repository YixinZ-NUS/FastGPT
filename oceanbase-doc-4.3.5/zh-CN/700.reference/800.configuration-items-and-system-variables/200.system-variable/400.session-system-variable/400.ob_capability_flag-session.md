# ob_capability_flag

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该变量从 V1.4 版本开始引入。</p>
</main>

## 功能描述

`ob_capability_flag` 用于设置 OBServer 节点能够提供的能力，用于和 obproxy 连接时进行能力协商，告知 obproxy 当前 OBServer 节点支持哪些功能，不支持哪些功能。

## 属性说明

| **属性** |              **描述**              |
|--------|----------------------------------|
| 参数类型   | uint                             |
| 默认值     | 0                                |
| 取值范围   | \[0,18446744073709551615\]       |
| 生效范围   | Session                          |
| 是否可修改  | 否，该变量支持在创建租户时指定，租户创建后不能通过 SQL 语句修改 |
| 是否可见   | 否，所有租户均不可见                               |

## 使用说明

不支持用户通过 `SHOW VARIABLES` 语句或视图 `SYS.TENANT_VIRTUAL_SESSION_VARIABLE`（Oracle 模式）及视图 `information_schema.SESSION_VARIABLES`（MySQL 模式）查询该变量的值。用户可以通过查询 `SYS.DBA_OB_SYS_VARIABLES` 视图（Oracle 模式）或视图 `oceanbase.DBA_OB_SYS_VARIABLES`（MySQL 模式）查询该变量的值。
