# ob_proxy_set_trx_executed

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该变量从 V2.2.77 版本开始引入。</p>
</main>

## 功能描述

`ob_proxy_set_trx_executed` 用于设置用户是否通过 obproxy 发送 `START TRANSACTION Syntax` 给 OBServer 节点。

## 属性说明

| **属性** |                                                 **描述**                                                 |
|--------|--------------------------------------------------------------------------------------------------------|
| 参数类型   | Bool                |
| 默认值    | 0                   |
| 取值范围   | <ul><li> 0：不发送 </li> <li> 1：发送  </li></ul>  |
| 生效范围   | Session             |
| 是否可修改  | 否，该变量支持在创建租户时指定，租户创建后不能通过 SQL 语句修改              |

## 使用说明

不支持用户通过 `SHOW VARIABLES` 语句或视图查询该变量的值。用户可以通过 `SELECT` 语句返回该变量的值：

```shell
obclient> SELECT @@ob_proxy_set_trx_executed;
```

返回结果如下：

```shell
+-----------------------------+
| @@ob_proxy_set_trx_executed |
+-----------------------------+
|                           0 |
+-----------------------------+
1 row in set
```
