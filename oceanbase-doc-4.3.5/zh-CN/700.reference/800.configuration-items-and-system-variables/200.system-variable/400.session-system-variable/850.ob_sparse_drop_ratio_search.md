# ob_sparse_drop_ratio_search

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该变量从 V4.3.5 BP2 版本开始引入。</p>
</main>

## 功能描述

`ob_sparse_drop_ratio_search` 用于在稀疏向量查询时，将查询向量中最小的一部分维度值置为 `0`，以减少维度并加速查询。

<main id="notice" type='explain'>
  <h4>功能适用性</h4>
  <p>该变量仅适用于 OceanBase 数据库 MySQL 模式。</p>
</main>

## 权限要求

* 查询变量

  * `sys` 租户和所有用户租户均可以使用 `SHOW VARIABLES` 语句或视图 `information_schema.SESSION_VARIABLES` 查看 Session 系统变量的值。

* 修改变量

  * sys 租户和所有 MySQL 用户租户均可以直接修改本租户的 Session 系统变量的值。

## 属性说明

| **属性**  | **描述**  |
|---------|---------|
| 参数类型    | Uint |
| 默认值     | 0    |
| 取值范围    | [0,100]    |
| 生效范围    | Session |
| 是否可修改 | 是，支持通过 `SET` 语句修改。 |

## 配置示例

设置稀疏向量查询时，将查询向量中最小的 30% 的维度值置为 `0`。

```shell
obclient> SET ob_sparse_drop_ratio_search = 30;
```
