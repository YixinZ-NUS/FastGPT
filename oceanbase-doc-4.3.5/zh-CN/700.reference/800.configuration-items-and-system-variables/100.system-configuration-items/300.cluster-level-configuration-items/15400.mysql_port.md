# mysql_port

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从 V1.4 版本开始引入。</p>
</main>

## 功能描述

`mysql_port` 用于设置 SQL 服务协议端口号。

## 权限要求

* 查询配置项

  `sys` 租户和所有用户租户均可以使用 `SHOW PARAMETERS` 语句或视图 `GV$OB_PARAMETERS` 查询该配置项。

* 修改配置项

  仅在集群部署时设置此配置项，且集群启动后不可修改。

## 属性说明

|      **属性**      |     **描述**      |
|------------------|-----------------|
| 参数类型             | Int             |
| 默认值               | 2881 <main id="notice" type='explain'><h4>说明</h4><p>从 V3.2.4 版本开始，默认值由 2880 调整为 2881。</p></main>           |
| 取值范围             | [1025, 65535] |
| 是否可修改                | 否，不支持通过 `ALTER SYSTEM SET` 语句修改。  |
| 是否重启 OBServer 节点生效 | 否，仅支持只读。               |
