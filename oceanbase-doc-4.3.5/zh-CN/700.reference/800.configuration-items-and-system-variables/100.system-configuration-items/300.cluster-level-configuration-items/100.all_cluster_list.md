# all_cluster_list

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>由于 V2.x 版本对集群配置进行了优化调整，该配置项从 V2.2.50 版本开始删除。</p>
</main>

## 功能描述

`all_cluster_list` 用于显示访问相同 `config_url` 的服务器列表。

## 属性说明

|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | String    |
| 默认值               | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 节点生效 | 否      |
