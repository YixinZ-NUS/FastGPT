# cpu_reserved

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>由于 V4.x 版本对系统架构进行了调整，对应功能不再支持，该配置项从 V4.0.0 版本开始删除。</p>
</main>

## 功能描述

`cpu_reserved` 用于设置系统预留的 CPU 数量，其余 CPU 将被 OceanBase 数据库独占使用。

## 属性说明

|  **属性**    |  **描述**   |
|----------------|-----------|
| 参数类型            | Int        |
| 默认值              | 2         |
| 取值范围            | [0, 15] |
| 是否重启 OBServer 节点生效 | 否         |
