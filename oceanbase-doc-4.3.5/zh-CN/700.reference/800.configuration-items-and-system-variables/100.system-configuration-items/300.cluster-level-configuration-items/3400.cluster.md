# cluster

<main id="notice" type='notice'>
<h4>注意</h4>
<p>该启动配置项仅建议在创建 OceanBase 集群时设置，集群生效后，无法修改。创建 OceanBase 集群的详细介绍，请参见 <a href="../../../../400.deploy/300.deploy-oceanbase-enterprise-edition/300.deploy-through-a-graphical-interface/300.deploy-oceanbase-cluster-use-ocp/400.deploy-single-replica-oceanbase-cluster-use-ocp.md">创建 OceanBase 集群</a>。</p>
</main>

## 功能描述

`cluster` 用于设置当前 OceanBase 集群的名称。

## 权限要求

* 查询配置项

  `sys` 租户和所有用户租户均可以使用 `SHOW PARAMETERS` 语句或视图 `GV$OB_PARAMETERS` 查询该配置项。

* 修改配置项

  仅在集群部署时设置此配置项，且集群启动后不可修改。

## 属性说明

|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型                 | String       |
| 默认值                   | obcluster |
| 是否可修改                | 否，不支持通过 `ALTER SYSTEM SET` 语句修改。  |
| 是否重启 OBServer 节点生效 | 否，仅支持只读。         |
