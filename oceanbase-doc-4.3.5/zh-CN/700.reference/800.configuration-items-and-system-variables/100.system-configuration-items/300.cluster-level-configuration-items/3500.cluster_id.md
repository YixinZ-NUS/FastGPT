# cluster_id

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从 V2.2.50 版本开始引入。</p>
</main>

## 功能描述

`cluster_id` 用于设置本 OceanBase 集群的 ID。

## 权限要求

* 查询配置项

  `sys` 租户和所有用户租户均可以使用 `SHOW PARAMETERS` 语句或视图 `GV$OB_PARAMETERS` 查询该配置项。

* 修改配置项

  仅在集群部署时设置此配置项，且集群启动后不可修改。

## 属性说明

| **属性** | **描述** |
| -------- | -------- |
| 参数类型             | Int                |
| 默认值               | 0                 |
| 取值范围             | [1, 4294901759] |
| 是否可修改           | 否，不支持通过 `ALTER SYSTEM SET` 语句修改。  |
| 是否重启 OBServer 节点生效 | 否，仅支持只读。 |
