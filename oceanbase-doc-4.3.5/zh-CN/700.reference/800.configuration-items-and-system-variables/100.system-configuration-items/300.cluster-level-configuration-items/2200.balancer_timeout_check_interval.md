# balancer_timeout_check_interval

<main id="notice" type='explain'>
<h4>说明</h4>
<p>由于 V4.x 版本对存储架构进行了升级，该配置项不再适用，从 V4.0.0 版本开始删除。</p>
</main>

## 功能描述

`balancer_timeout_check_interval` 用于设置检查负载均衡等后台任务超时的时间间隔。

## 属性说明

|      **属性**    |   **描述** |
|------------------|------------|
| 参数类型            | Time      |
| 默认值              | 1m，表示 1 分钟 |
| 取值范围            | [1s, +∞)  |
| 是否重启 OBServer 节点生效 | 否          |
