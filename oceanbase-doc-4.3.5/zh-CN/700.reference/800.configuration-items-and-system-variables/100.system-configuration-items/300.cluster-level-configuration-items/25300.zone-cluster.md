# zone

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从 V2.2.50 版本开始引入。</p>
</main>

## 功能描述

`zone` 用于设置节点所在的 Zone 的名字。

## 权限要求

* 查询配置项

  `sys` 租户和所有用户租户均可以使用 `SHOW PARAMETERS` 语句或视图 `GV$OB_PARAMETERS` 查询该配置项。

* 修改配置项

  该配置项一般不需要配置。仅 `sys` 租户可以修改该配置项，用户租户不能修改该配置项。

## 属性说明

|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | String  |
| 默认值               | ""，表示空字符串     |
| 是否可修改           | 是，支持通过 `ALTER SYSTEM SET` 语句修改。|
| 是否重启 OBServer 节点生效 | 否，设置后立即生效。            |

## 配置示例

设置节点所在 Zone 的名称为 `zone1`。

```shell
obclient> ALTER SYSTEM SET `zone` = 'zone1';
```
