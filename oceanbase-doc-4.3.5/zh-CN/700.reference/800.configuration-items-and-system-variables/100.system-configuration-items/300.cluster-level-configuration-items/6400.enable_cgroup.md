# enable_cgroup

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该配置项从 V3.2.4 版本开始引入。</p>
</main>

## 功能描述

`enable_cgroup` 用于控制 OBServer 节点是否开启 cgroup 功能。

## 权限要求

* 查询配置项

  sys 租户和所有用户租户均可以使用 `SHOW PARAMETERS` 语句或视图 `GV$OB_PARAMETERS` 查询该配置项。

* 修改配置项

  仅 sys 租户可以修改该配置项，用户租户不能修改该配置项。

## 属性说明

| **属性** | **描述** |
| ------ | ------ |
| 参数类型 | Bool |
| 默认值 | True |
| 取值范围 |<ul><li>True：若检测到系统已经挂载好 cgroup 目录，就开启 cgroup 功能；若检测不到，就报 warning。</li><li>False：不开启 cgroup 功能。</li></ul>|
| 是否可修改| 是，支持通过 `ALTER SYSTEM SET` 语句修改。|
| 是否重启 OBServer 节点生效 | 否，设置后立即生效。 <main id="notice" type='explain'><h4>说明</h4><p>从 V4.3.5 版本开始，该配置项修改为动态生效。</p></main> |

## 配置示例

开启 cgroup 功能。

```shell
obclient> ALTER SYSTEM SET enable_cgroup = true;
```
