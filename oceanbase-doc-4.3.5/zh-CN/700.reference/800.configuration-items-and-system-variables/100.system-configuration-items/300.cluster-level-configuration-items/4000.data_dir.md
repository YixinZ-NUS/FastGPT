# data_dir

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从 V1.4 版本开始引入。</p>
</main>

## 功能描述

`data_dir` 用于设置存储 SSTable 等数据的目录。

## 权限要求

* 查询配置项

  `sys` 租户和所有用户租户均可以使用 `SHOW PARAMETERS` 语句查询该配置项。

* 修改配置项

  仅 `sys` 租户在集群部署时设置此配置项，且集群启动后不可修改。用户租户无权限修改此配置项。

## 属性说明

|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型            | String    |
| 默认值              | store  |
| 是否可修改          | 否    |
| 是否重启 OBServer 节点生效 | 否，仅支持只读。      |
