# query_response_time_range_base

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

`query_response_time_range_base` 用于设置 `information_schema.QUERY_RESPONSE_TIME` 视图中统计查询响应时间的时间区间基数。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>该变量的更改仅在刷新后生效。</p>
  </main>

## 权限要求

* 查询配置项

  `sys` 租户和所有用户租户均可以使用 `SHOW PARAMETERS` 语句或视图 `GV$OB_PARAMETERS` 查询该配置项。

* 修改配置项

  `sys` 租户和用户租户均可修改该配置项。

## 属性说明

|        属性      |  描述  |
|------------------|-------|
| 参数类型             | Int |
| 默认值               | 10 |
| 取值范围             | [2, 10000) |
| 是否可修改           | 是，支持通过 `ALTER SYSTEM SET` 语句修改。|
| 是否重启 OBServer 节点生效 | 否，设置后立即生效。    |

## 配置示例

设置 `information_schema.QUERY_RESPONSE_TIME` 视图中统计查询响应时间的时间区间基数为 10。

```shell
obclient> ALTER SYSTEM SET query_response_time_range_base=10;
```
