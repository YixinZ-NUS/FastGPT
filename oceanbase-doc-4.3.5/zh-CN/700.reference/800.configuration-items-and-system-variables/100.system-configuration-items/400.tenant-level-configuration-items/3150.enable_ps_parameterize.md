# enable_ps_parameterize

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>对于 V4.3.x 版本，该配置项从 V4.3.5 BP4 版本开始引入。</p>
</main>

## 功能描述

`enable_ps_parameterize` 用于控制开启或关闭 PS 语句在解析时是否参数化。

## 权限要求

* 查询配置项

  `sys` 租户和所有用户租户均可以使用 `SHOW PARAMETERS` 语句或视图 `GV$OB_PARAMETERS` 查询该配置项。

* 修改配置项

  `sys` 租户和用户租户都可以使用 `ALTER SYSTEM SET` 语句修改该配置项。

## 属性说明

| **属性** | **描述** |
| --- | --- |
| 参数类型 | Bool |
| 默认值 | <li>新建租户默认值为`False`</li> <li>原租户默认值为 `True`</li> |
| 取值范围 |<ul><li> True：开启  </li><li> False：关闭 </li></ul>|
| 是否可修改 | 是，支持通过 `ALTER SYSTEM SET` 语句修改。|
| 是否重启 OBSever 节点生效 | 否，设置后立即生效。 |

## 配置示例

关闭参数化功能。

```shell
obclient> ALTER SYSTEM SET enable_ps_parameterize = false;
```
