# query_response_time_stats

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从 V4.0.0 版本开始引入。</p>
</main>

## 功能描述

`query_response_time_stats` 用于设置是否启用 `information_schema.QUERY_RESPONSE_TIME` 视图的统计信息收集。

## 权限要求

* 查询配置项

  `sys` 租户和所有用户租户均可以使用 `SHOW PARAMETERS` 语句或视图 `GV$OB_PARAMETERS` 查询该配置项。

* 修改配置项

  `sys` 租户和用户租户均可修改该配置项。

## 属性说明

|        属性      |  描述  |
|------------------|-------|
| 参数类型             | Bool |
| 默认值               | False |
| 取值范围             | <ul><li>True：启用 </li><li>False：禁用 </li></ul> |
| 是否可修改           | 是，支持通过 `ALTER SYSTEM SET` 语句修改。|
| 是否重启 OBServer 节点生效 | 否，设置后立即生效。    |

## 配置示例

设置不启用 `information_schema.QUERY_RESPONSE_TIME` 视图的统计信息收集。

```shell
obclient> ALTER SYSTEM SET query_response_time_stats=False;
```
