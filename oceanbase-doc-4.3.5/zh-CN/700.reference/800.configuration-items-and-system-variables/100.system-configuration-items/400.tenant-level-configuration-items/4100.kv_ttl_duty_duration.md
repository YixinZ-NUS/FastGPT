# kv_ttl_duty_duration

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从 V3.1.4 版本开始引入。</p>
</main>

## 功能描述

`kv_ttl_duty_duration` 用于设置每日定时 TTL 的触发时段，TTL 任务会在该时间段内触发一次。

## 权限要求

* 查询配置项

  `sys` 租户和所有用户租户均可以使用 `SHOW PARAMETERS` 语句或视图 `GV$OB_PARAMETERS` 查询该配置项。

* 修改配置项

  `sys` 租户和用户租户均可修改该配置项。

## 属性说明

| 属性 | 描述 |
| --- | --- |
| 参数类型      | String |
| 默认值        | [00:00:00, 24:00:00] |
| 取值范围      | [00:00:00, 24:00:00] |
| 是否可修改    | 是，支持通过 `ALTER SYSTEM SET` 语句修改。|
| 是否重启 OBServer 生效 | 否，设置后立即生效。  |

## 配置示例

设置每日定时 TTL 的触发时段为 [00:00:00, 24:00:00]。

```shell
obclient> ALTER SYSTEM SET kv_ttl_duty_duration = '[00:00:00, 24:00:00]';
```
