# enable_user_defined_rewrite_rules

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从 V4.1.0 版本开始引入。</p>
</main>

## 功能描述

`enable_user_defined_rewrite_rules` 用于控制是否开启改写绑定功能。

## 权限要求

* 查询配置项

  `sys` 租户和所有用户租户均可以使用 `SHOW PARAMETERS` 语句或视图 `GV$OB_PARAMETERS` 查询该配置项。

* 修改配置项

  `sys` 租户和用户租户均可修改该配置项。

## 属性说明

| **属性** | **描述** |
| -------- | -------- |
| 参数类型             | Bool |
| 默认值               | False |
| 取值范围             |<ul><li> True：开启 </li><li>False：关闭 </li></ul>|
| 是否可修改           | 是，支持通过 `ALTER SYSTEM SET` 语句修改。|
| 是否重启 OBServer 节点生效 | 否，设置后立即生效。 |

## 使用说明

[DBMS_UDR](../../../../700.reference/600.pl-reference/200.pl-mysql/1000.pl-system-package-mysql/17800.dbms-udr-mysql/100.dbms-udr-overview-mysql.md) 系统包提供了改写绑定功能：

* 创建一条自定义改写规则：[CREATE_RULE](../../../../700.reference/600.pl-reference/200.pl-mysql/1000.pl-system-package-mysql/17800.dbms-udr-mysql/200.create-rule-mysql.md)
* 关闭一条自定义改写规则：[DISABLE_RULE](../../../../700.reference/600.pl-reference/200.pl-mysql/1000.pl-system-package-mysql/17800.dbms-udr-mysql/300.disable-rule-mysql.md)
* 开启一条自定义改写规则：[ENABLE_RULE](../../../../700.reference/600.pl-reference/200.pl-mysql/1000.pl-system-package-mysql/17800.dbms-udr-mysql/400.enable-rule-mysql.md)
* 删除一条自定义改写规则：[REMOVE_RULE](../../../../700.reference/600.pl-reference/200.pl-mysql/1000.pl-system-package-mysql/17800.dbms-udr-mysql/500.remove-rule-mysql.md)

## 配置示例

设置关闭改写绑定功能。

```shell
obclient> ALTER SYSTEM SET enable_user_defined_rewrite_rules = False;
```
