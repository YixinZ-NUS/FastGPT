# SET CATALOG

## 描述

该语句用于切换 Catalog。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>对于 OceanBase 数据库 V4.3.5 版本，从 V4.3.5 BP2 版本开始支持 <code>SET CATALOG</code> 语句。</p>
</main>

## 权限要求

执行 `SET CATALOG` 语句，需要当前用户拥有 `USE CATALOG` 权限。有关 OceanBase 数据库权限的详细介绍，参见 [MySQL 模式下的权限分类](../../../../../600.manage/500.security-and-permissions/300.access-control/200.user-and-permission/200.permission-of-mysql-mode/100.permission-classification-of-mysql.md)。

## 语法

```sql
SET CATALOG {INTERNAL | external_catalog_name};
```

## 参数说明

|        **参数**       | **描述** |
|-----------------------|----------|
| INTERNAL              | 表示切换成系统内部 Catalog。|
| external_catalog_name | 指定要切换为外部数据目录的名称。|

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>可以使用 <code>use internal.test</code> 或 <code>use odps_catalog.default</code> 一次性完成切换 Catalog 和 数据库的操作。</p>
</main>

## 示例

* 切换为外部数据目录 `test_odps_catalog`。

  ```sql
  SET CATALOG test_odps_catalog;
  ```

* 切换成系统内部 Catalog。

  ```sql
  SET CATALOG INTERNAL;
  ```
