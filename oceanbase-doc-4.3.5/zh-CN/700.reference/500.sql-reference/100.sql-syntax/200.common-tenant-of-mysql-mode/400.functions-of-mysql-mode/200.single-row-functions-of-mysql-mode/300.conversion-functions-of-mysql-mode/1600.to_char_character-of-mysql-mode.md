# TO_CHAR(character)

## 描述

该函数用于将字符型数据转换为 `VARCHAR` 类型的字符串。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>对于 OceanBase 数据库 V4.3.5 版本，该函数从 V4.3.5 BP1 版本开始支持。</p>
</main>

## 语法声明

```sql
TO_CHAR(character)
```

## 参数解释

`character`：表示要转换的字符型数据。

## 返回类型

* 返回 `VARCHAR` 类型的数据。
* `character` 值为 `NULL` 时，返回 `NULL`。

## 示例

```sql
SELECT TO_CHAR('Hello, World!'), TO_CHAR(NULL);
```

返回结果如下：

```shell
+--------------------------+---------------+
| TO_CHAR('Hello, World!') | TO_CHAR(NULL) |
+--------------------------+---------------+
| Hello, World!            |          NULL |
+--------------------------+---------------+
1 row in set
```
