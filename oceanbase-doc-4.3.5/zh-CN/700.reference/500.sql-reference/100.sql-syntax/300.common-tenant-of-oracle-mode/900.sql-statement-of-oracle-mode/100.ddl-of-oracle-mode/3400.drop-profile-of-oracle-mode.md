# DROP PROFILE

## 描述

该语句用于删除配置文件。

## 语法

```sql
DROP PROFILE profile_name
```

## 参数解释

|      参数      |       描述       |
|--------------|----------------|
| profile_name | 指定要删除的配置文件的名称。 |

## 示例

1. 创建配置文件 `profile1`。

   ```sql
   obclient> CREATE PROFILE profile1 LIMIT
       FAILED_LOGIN_ATTEMPTS 5
       PASSWORD_LOCK_TIME 1
       PASSWORD_LIFE_TIME 60;
   ```

2. 删除配置文件 `profile1`。

   ```sql
   obclient> DROP PROFILE profile1;
   ```
