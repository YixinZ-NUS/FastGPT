# DROP ROLE

## 描述

该语句用于删除角色。

  <main id="notice" type='explain'>
    <h4>说明</h4>
      <p>删除角色后：</p>
      <ul>
      <li>该角色将从所有被授予此角色的用户和角色中撤销。</li>
      <li>该角色将从数据库中删除。</li>
      <li>已启用该角色的用户会话不受影响。</li>
      <li>新的用户会话将无法启用该角色。</li>
      </ul>
  </main>

## 语法

```sql
DROP ROLE role_name
```

## 参数解释

|    参数     |      描述      |
|-----------|--------------|
| role_name | 指定要删除的角色的名称。 |

## 示例

首先创建角色 `sales_manager`，然后删除该角色。

```sql
obclient> CREATE ROLE sales_manager;
obclient> DROP ROLE sales_manager;
```
