# DROP CONTEXT

## 描述

`DROP CONTEXT` 语句用于从数据库中删除上下文命名空间。

删除上下文命名空间后，当前会话中已设置的上下文值会保留。但是，后续尝试设置该上下文时会失败，因为命名空间已不存在。

  <main id="notice" type='notice'>
    <h4>注意</h4>
    <p>您必须具有 <code>DROP ANY CONTEXT</code> 系统权限。</p>
  </main>

## 语法

```sql
DROP CONTEXT context_name
```

## 参数说明

`context_name` 是指定要删除的上下文名称。您不能删除内置的命名空间 `USERENV`。

有关 `USERENV` 上下文的信息，请参见函数 [SYS_CONTEXT](../../500.functions-of-oracle-mode/200.single-row-functions-of-oracle-mode/900.environment-and-identifier-functions-of-oracle-mode/100.lnnvl-of-oracle-mode.md)。

## 示例

删除在 `CREATE CONTEXT` 中创建的上下文 `dev_context`。

```sql
obclient> DROP CONTEXT dev_context;
```
