# CREATE TABLESPACE

## 描述

该语句用来创建 TableSpace 逻辑对象，TableSpace 属性支持加密。

## 语法

```sql
CREATE TABLESPACE tablespace_name
    [ ENCRYPTION USING 'algorithm' ]

algorithm:
    'AES-256' | 'AES-128' | 'AES-192' | 'SM4-CBC'
```

## 参数解释

|                 参数                 |           描述            |
|------------------------------------|-------------------------|
| tablespace_name                    | 指定要创建的 TableSpace 对象名称。 |
| ENCRYPTION USING                   | 加密所使用的关键字。              |
| AES-256\|AES-128\|AES-192\|SM4-CBC | 选择所使用的加密算法。             |

## 示例

* 创建一个 TableSpace 对象。

  ```sql
  obclient> CREATE TABLESPACE tblspace1;
  ```

* 创建一个 TableSpace 对象并使用加密。

  ```sql
  -- 步骤1：设置 TDE 方法为内部密钥管理
  obclient> ALTER SYSTEM SET TDE_METHOD = 'INTERNAL';

  -- 步骤2：创建密钥库
  obclient> ADMINISTER KEY MANAGEMENT CREATE KEYSTORE my_keystore IDENTIFIED BY abcCBAK123;

  -- 步骤3：打开密钥库
  obclient> ADMINISTER KEY MANAGEMENT SET KEYSTORE OPEN IDENTIFIED BY abcCBAK123;

  -- 步骤4：创建主密钥
  obclient> ADMINISTER KEY MANAGEMENT SET KEY IDENTIFIED BY abcCBAK123;

  -- 步骤5：创建加密表空间
  obclient> CREATE TABLESPACE tblspace2 ENCRYPTION USING 'SM4-CBC';
  ```

  密码示例中使用 `abcCBAK123`，实际使用时请替换为安全密码。
