# ALTER SYSTEM FLUSH CACHE

## 描述

删除缓存。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>对于 Oracle 租户，如果要清除租户下指定 SQL 或 Schema 的计划缓存，需要使用 <a href="../../../../../600.pl-reference/300.pl-oracle/1400.pl-system-package-oracle/12500.dbms-plan-cache-oracle/100.dbms-plan-cache-overview-oracle.md">DBMS_PLAN_CACHE</a> 系统包。</p>
  </main>

## 语法

```sql
ALTER SYSTEM FLUSH {PLAN
                  | LIB } CACHE [GLOBAL]
```

## 参数解释

| 参数 | 描述 |
|---------|------|
| PLAN | 清空执行计划缓存 |
| LIB | 清空 Lib Cache |

## 示例

```sql
obclient> ALTER SYSTEM FLUSH PLAN CACHE GLOBAL;
```
