# DROP CONCURRENT_LIMITING_RULE

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该语句从 OceanBase 数据库 V4.3.5 BP3 版本开始引入。</p>
</main>

## 描述

该语句用于删除限流规则。

## 注意事项

OceanBase 数据库 Oracle 模式下执行 `DROP CONCURRENT_LIMITING_RULE` 语句需要专用权限 `drop any concurrent_limiting_rule`。

## 语法

```sql
DROP CONCURRENT_LIMITING_RULE [ IF EXISTS ] "ccl_rule_name1" [, "ccl_rule_name2", ...]
```

其中，`ccl_rule_name` 表示限流规则的名称。

## 示例

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>被删除的限流规则会立即失效，此时该规则下等待队列中的 SQL 语句全部会被正常执行。</p>
</main>

执行如下命令删除限流规则 `sql1_keywords_ccl_rule`，`sql2_keywords_ccl_rule`

```shell
obclient> DROP CONCURRENT_LIMITING_RULE "sql1_keywords_ccl_rule"，"sql2_keywords_ccl_rule";
```
