# 全局 Hint 类型下的 Transform Hint

全局 Hint 类型下的 Transform Hint 用于告诉优化器如何处理或转换整个查询的提示。

| Hint 名称 | 解释 |
| --- | --- |
| `NO_QUERY_TRANSFORMATION` | 禁止查询进行任何改写。 |
| `NO_COST_BASED_QUERY_TRANSFORMATION` | 禁止查询进行基于代价改写。 |

## `NO_QUERY_TRANSFORMATION` Hint

`NO_QUERY_TRANSFORMATION` Hint 禁止对当前查询进行任何查询改写，区别于 Query Block Hint 中的 `NO_REWRITE` Hint，使用 `NO_QUERY_TRANSFORMATION` 无法使用 Query Block Hint 开启一些改写。

查询改写相关介绍见 [查询改写概述](../../../../../../../../1000.performance-tuning-guide/500.sql-optimization/400.sql-optimization/500.query-rewrite/100.query-rewrite-overview.md)。

### 语法

```sql
/*+ NO_QUERY_TRANSFORMATION */
```

### 示例

```sql
--使用 `NO_QUERY_TRANSFORMATION` Hint 禁止任何查询改写。
SELECT /*+ NO_QUERY_TRANSFORMATION */ *
FROM (SELECT * FROM t1) v WHERE v.c1 = 3;
```

## `NO_COST_BASED_QUERY_TRANSFORMATION` Hint

`NO_COST_BASED_QUERY_TRANSFORMATION` Hint 禁止对当前查询进行任何基于代价的查询改写，基于代价查询改写详见[基于代价的改写介绍](../../../../../../../../1000.performance-tuning-guide/500.sql-optimization/400.sql-optimization/500.query-rewrite/300.cost-based-query-rewriting.md)。

### 语法

```sql
/*+ NO_COST_BASED_QUERY_TRANSFORMATION */
```

### 示例

```sql
--使用 `NO_COST_BASED_QUERY_TRANSFORMATION` Hint 禁止任何基于代价的查询改写。
CREATE TABLE t1(c1 int, c2 int, c3 int, index idx1(c1), index idx2(c2));
SELECT /*+ NO_COST_BASED_QUERY_TRANSFORMATION */ *
FROM t1
WHERE c1 = 1 OR c2 = 1;
```
