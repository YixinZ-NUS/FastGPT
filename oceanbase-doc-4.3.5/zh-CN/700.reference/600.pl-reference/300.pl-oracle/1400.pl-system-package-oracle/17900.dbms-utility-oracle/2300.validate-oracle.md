# VALIDATE

`VALIDATE` 用于验证或重新编译单个数据库对象的有效性。

<main id="notice" >
  <h4>功能适用性</h4>
  <p>该内容仅适用于 OceanBase 数据库 Oracle 模式。</p>
</main>

## 语法

```sql
DBMS_UTILITY.VALIDATE(
    OBJECT_ID NUMBER,
    IS_RECOMPILE BOOLEAN DEFAULT FALSE);
```

## 参数说明

|   参数   |      解释       |
|--------|---------------|
| OBJECT_ID | 数据库对象 ID。 |
| IS_RECOMPILE | 内部后台重编译的标记，不需要主动设置，保持 `false` 即可。|

## 使用示例

```sql
obclient> BEGIN
  DBMS_UTILITY.VALIDATE（52939）
  END;
```
