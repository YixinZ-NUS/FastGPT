# RCPT

`RCPT` 用于执行 `rcpt` 命令，指定电子邮件收件人。

<main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 语法

```sql
UTL_SMTP.RCPT (
   c           IN OUT NOCOPY     connection,
   recipient   IN                VARCHAR2,
   parameters  IN                VARCHAR2 DEFAULT NULL)
RETURN reply;

UTL_SMTP.RCPT (
   c           IN OUT NOCOPY     connection,
   recipient   IN                VARCHAR2,
   parameters  IN                VARCHAR2 DEFAULT NULL);
```

## 参数解释

| **参数** |         **解释**         |
|--------|------------------------|
|c|SMTP 连接。|
|recipient|收件人邮箱。|
|parameters|RFC1869 中定义了 SMTP 协议的格式。它必须遵循“XXX=XXX （XXX=XXX ....）”的格式。|

## 返回值

| **返回值** |          **描述**          |
|---------|--------------------------|
|reply|command 的回复。如果有多个回复，则返回最后一个回复。|

## 使用说明

* 每次调用 `RCPT` 指定单个收件人邮箱地址，可以通过多次调用 `RCPT` 来指定多个收件人。
* 需要在 `mail` 后调用。
* 预期返回是 `250` 或 `251` 开头的回复信息，表示 SMTP 服务器成功响应并接受了指定的收件人地址。
