# EHLO

`EHLO` 用于执行 `EHLO` 命令，与 SMTP 服务进行初始握手。

<main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 语法

```sql
UTL_SMTP.EHLO (
   c       IN OUT NOCOPY connection,
   domain  IN)
RETURN replies;

UTL_SMTP.EHLO (
   c       IN OUT NOCOPY connection,
   domain  IN);
```

## 参数解释

| **参数** |         **解释**         |
|--------|------------------------|
|c|SMTP 连接。|
|domain|发送主机的域名，用于识别。|

## 返回值

| **返回值** |          **描述**          |
|---------|--------------------------|
|reply|command 的回复。如果有多个回复，则返回最后一个回复。|

## 使用说明

`ehlo` 命令是 `helo` 命令的扩展，允许服务器返回有关其配置信息的描述性信息，为了兼容性，每行信息以状态码 `250` 开头。
