# MAIL

`MAIL` 用于执行 MAIL 命令，与 SMTP 服务启动邮件事务，声明发件人邮箱。

<main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 语法

```sql
UTL_SMTP.MAIL (
   c           IN OUT NOCOPY   connection,
   sender      IN              VARCHAR2,
   parameters  IN              VARCHAR2 DEFAULT NULL)
RETURN reply;

UTL_SMTP.MAIL (
   c           IN OUT NOCOPY   connection,
   sender      IN              VARCHAR2,
   parameters  IN              VARCHAR2 DEFAULT NULL);
```

## 参数解释

| **参数** |         **解释**         |
|--------|------------------------|
|c|SMTP 连接。|
|sender|发件人邮箱。|
|parameters|RFC1869 中定义了 SMTP 协议的格式。它必须遵循“XXX=XXX （XXX=XXX ....）”的格式。|

## 返回值

| **返回值** |          **描述**          |
|---------|--------------------------|
|reply|command 的回复。如果有多个回复，则返回最后一个回复。|

## 使用说明

`mail` 命令用于向服务器说明启动了一个新的邮件事务，需要由后续调用 `rcpt` 和 `data` 命令来完成事务，与 SMTP 服务器的连接必须处于打开状态，且已经执行了 `helo` 或 `ehlo` 命令。
