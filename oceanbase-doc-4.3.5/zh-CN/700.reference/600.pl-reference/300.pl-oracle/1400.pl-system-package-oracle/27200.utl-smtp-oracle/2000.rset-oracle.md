# RSET

用于发送 `RSET` 命令，用于终止当前电子邮件事务。

<main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 语法

```sql
UTL_SMTP.RSET (
   c  IN OUT NOCOPY connection)
RETURN reply;

UTL_SMTP.RSET (
   c  IN OUT NOCOPY connection);
```

## 参数解释

| **参数** |         **解释**         |
|--------|------------------------|
|c|SMTP 连接。|

## 返回值

| **返回值** |          **描述**          |
|---------|--------------------------|
|reply|command 的回复。如果有多个回复，则返回最后一个回复。|

## 使用说明

* `REST` 命令允许取消正在撰写的电子邮件，不发送邮件。可以在 `open_connection` 和 `data` 或 `open_data` 之间使用来取消邮件事务。但一旦邮件开始发送，就无法阻止发送电子邮件了。
* 预期返回是 `250` 开头的回复信息，表示当前电子邮件事务已终止。
