# COMMAND_REPLIES

`COMMAND_REPLIES` 用于执行通用的 SMTP 命令，接收多个回复行。

<main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 语法

```sql
UTL_SMTP.COMMAND_REPLIES (
   c     IN OUT NOCOPY    connection,
   cmd   IN               VARCHAR2,
   arg   IN               VARCHAR2 DEFAULT NULL)
RETURN replies;
```

## 参数解释

| **参数** |         **解释**         |
|--------|------------------------|
|c|SMTP 连接。|
|cmd|发送给服务器的 `command` 命令。|
|arg|SMTP 命令参数，实际发送时会在 `cmd` 和 `arg` 中间加一个空格进行拼接。|

## 返回值

| **返回值** |          **描述**          |
|---------|--------------------------|
|replies|command 的回复。|

## 使用说明

此函数返回多个回复行。
