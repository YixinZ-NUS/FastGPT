# QUIT

向服务器发送 `QUIT` 命令，终止 SMTP 会话，并断开与服务器的连接。

<main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 语法

```sql
UTL_SMTP.QUIT (
   c  IN OUT NOCOPY connection)
RETURN reply;

UTL_SMTP.QUIT (
   c  IN OUT NOCOPY connection);
```

## 参数解释

| **参数** |         **解释**         |
|--------|------------------------|
|c|SMTP 连接。|

## 返回值

| **返回值** |          **描述**          |
|---------|--------------------------|
|reply|command 的回复。如果有多个回复，则返回最后一个回复。|

## 使用说明

* 向服务器发送 `QUIT` 命令告知服务器终止会话。然后关闭连接，需要在调用 `open_connection` 之后调用。
* 如果邮件事务正在进行中调用 `QUIT`，等同于 `REST` 方式取消事务。
* 预期返回 `221` 开头的消息，表示 SMTP 服务器已完成处理并关闭连接。
