# UTL_TCP 概述

`UTL_TCP` 是一个通过 TCP/IP 协议与外部服务器进行通信的系统包。

<main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## UTL_TCP 子程序概览

下表列出了 OceanBase 数据库当前版本所支持的 `UTL_TCP` 子程序以及简要描述。

|                       **子程序**             |                **描述**                       |
|----------------------------------------------|-----------------------------------------------|
|[OPEN_CONNECTION](200.open-connection-oracle.md)|打开一个与指定服务的 TCP/IP 连接。|
|[CLOSE_CONNECTION](300.utl-tcp-close-connection-oracle.md)|关闭打开的 TCP/IP 连接。|
|[CLOSE_ALL_CONNECTIONS](400.close-all-connections-oracle.md)|关闭打开的 TCP/IP 连接。|
|[WRITE_LINE](500.write-line-oracle.md)|将文本行传输到打开连接的服务。|
|[WRITE_TEXT](600.write-text-oracle.md)|将文本消息传输到打开连接的服务。|
|[WRITE_RAW](700.write-raw-oracle.md)|将二进制传输到打开连接的服务。|
|[GET_LINE](800.get-line-of-oracle.md)|返回读取的文本行数据，直到读取到行尾或输入结束时返回。|
|[GET_TEXT](900.get-text-oracle.md)|返回读取的文本数据。|
