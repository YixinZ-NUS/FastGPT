# WRITE_LINE

`WRITE_LINE` 用于将文本行传输到打开连接的服务。在消息末尾将追加 `newline` 字符序列。

<main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 语法

```sql
UTL_TCP.WRITE_LINE (
   c    IN OUT NOCOPY connection,
   data IN            VARCHAR2 DEFAULT NULL CHARACTER SET ANY_CS)
  RETURN PLS_INTEGER;
```

## 参数解释

| **参数** |         **解释**         |
|--------|------------------------|
|c| TCP/IP 连接。|
|data|包含要发送数据的缓冲区。|

## 返回值

返回传输数据的实际字符数。

## 使用说明

* 必须是已经通过 `open_connection` 打开过的连接。
* 在传输前将文本字符集转换为指定的在线字符集。
