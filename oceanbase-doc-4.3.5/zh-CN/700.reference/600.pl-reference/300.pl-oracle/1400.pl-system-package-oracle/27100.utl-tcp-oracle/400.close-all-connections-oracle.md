# CLOSE_ALL_CONNECTIONS

`CLOSE_ALL_CONNECTIONS` 用于关闭 Session 内打开的全部的 TCP/IP 连接。

<main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 语法

```sql
UTL_TCP.CLOSE_ALL_CONNECTIONS;
```
