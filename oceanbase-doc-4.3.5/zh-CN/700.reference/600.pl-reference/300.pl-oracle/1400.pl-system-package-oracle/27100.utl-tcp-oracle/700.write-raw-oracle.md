# WRITE_RAW

`WRITE_RAW` 用于将二进制传输到打开连接的服务。

<main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库 Oracle 模式。</p>
  </main>

## 语法

```sql
UTL_TCP.WRITE_RAW (
   c    IN OUT NOCOPY connection,
   data IN            RAW,
   len  IN            PLS_INTEGER DEFAULT NULL)
 RETURN PLS_INTEGER;
```

## 参数解释

| **参数** |         **解释**         |
|--------|------------------------|
|c| TCP/IP 连接。|
|data|包含要发送数据的缓冲区。|
|len|要传输的数据的字节数。为 `NULL` 时，传输全部数据。|

## 返回值

返回传输数据的实际字节数。

## 使用说明

* 必须是已经通过 `open_connection` 打开过的连接。
* 如果 `len` 大于 `data` 数据量，抛出 `bad argument` 异常。
