# DBMS_SPM 概述


`DBMS_SPM` 系统包用于管理 SQL 计划，支持加载、更改以及删除 Plan Baseline 信息。


## DBMS_STATS 子程序概览

下表列出了 OceanBase 数据库当前版本所支持的 `DBMS_STATS` 子程序和简要描述。

|                   **子程序**                                 |                  **描述**                |
|-------------------------------------------------------------|------------------------------------------|
| [ACCEPT_SQL_PLAN_BASELINE](200.accept-sql-plan-baseline-oracle.md)  | 根据演进计划的结果接受一个计划作为基线。        |
| [ALTER_SQL_PLAN_BASELINE](300.alter-sql-plan-baseline-oracle.md)              | 修改基线中一个或一组计划的属性。返回值表示被修改的基线数量。              |
| [CANCEL_EVOLVE_TASK](400.cancel-evolve-task-oracle.md)            | 取消一个正在演进的任务。           |
| [CONFIGURE](500.configure-oracle.md)             | 设置 SPM 的一些参数。                         |
| [DROP_EVOLVE_TASK](600.drop-evolve-task-oracle.md)             | 丢弃一个演进任务。                              |
| [DROP_SQL_PLAN_BASELINE](700.drop-sql-plan-baseline-oracle.md)            | 丢弃一个或多个计划基线。    |
| [LOAD_PLANS_FROM_CURSOR_CACH](800.load-plans-from-cursor-cache-oracle.md)            | 从 Cursor Cache 中读取一条或多条 SQL 计划作为计划基线。         |
