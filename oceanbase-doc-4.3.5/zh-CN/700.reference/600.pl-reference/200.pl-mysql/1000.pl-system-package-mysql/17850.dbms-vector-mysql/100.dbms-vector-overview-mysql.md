# DBMS_VECTOR 概述

`DBMS_VECTOR` 系统包用于管理向量索引。

## DBMS_VECTOR 子程序概览

下表列出了 OceanBase 数据库当前版本所支持的 `DBMS_VECTOR` 子程序和简要描述。

| **子程序** | **描述** |
| --- | --- |
| [REFRESH_INDEX](200.refresh-index-mysql.md) | 刷新向量索引。 |
| [REBUILD_INDEX](300.rebuild-index-mysql.md) | 重建向量索引。 |
| [INDEX_VECTOR_MEMORY_ESTIMATE](400.index-vector-memory-estimate-mysql.md) | 向量索引内存使用情况估算。<main id="notice" type='notice'><h4>说明</h4><p>该功能从 V4.3.5 BP3 版本开始支持。</p></main> |
| [INDEX_VECTOR_MEMORY_ADVISOR](500.index-vector-memory-advisor-mysql.md) | 向量索引内存使用情况分析。<main id="notice" type='notice'><h4>说明</h4><p>该功能从 V4.3.5 BP3 版本开始支持。</p></main> |
