# DBMS_PARTITION 概述

`DBMS_PARTITION` 系统包用于提供动态分区管理接口。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>对于 OceanBase 数据库 V4.3.5 版本，从 V4.3.5 BP2 版本开始支持系统包 <code>DBMS_PARTITION</code>。</p>
</main>

## 权限说明

仅支持在用户租户下进行操作。

## 子程序概览

下表列出了 OceanBase 数据库当前版本所支持的 `DBMS_PARTITION` 子程序和简要描述。

|                                   子程序                                 |           描述          |
|--------------------------------------------------------------------------|------------------------|
| [MANAGE_DYNAMIC_PARTITION](200.manage_dynamic_partition-of-mysql-mode.md)| 用于执行一次动态分区管理任务，为当前租户下动态分区表预创建未来分区、删除过期分区。|
