# 查看动态分区表

动态分区表创建成功后，您可以通过 `SHOW CREATE TABLE` 语句查看表的定义和视图查看数据库中的动态分区表信息。

## 查看表的定义

表创建成功后，您可以使用 `SHOW CREATE TABLE table_name;` 语句查看表的定义。

更多有关 `SHOW` 语句的介绍信息，参见 [SHOW](../../../../500.sql-reference/100.sql-syntax/300.common-tenant-of-oracle-mode/900.sql-statement-of-oracle-mode/300.dcl-of-oracle-mode/3600.show-of-oracle-mode.md)。

**示例如下：**

查看表 `test_tbl1` 的定义。

```sql
SHOW CREATE TABLE test_tbl1;
```

查询结果如下：

```shell
+-----------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| TABLE     | CREATE TABLE                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
+-----------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| TEST_TBL1 | CREATE TABLE "TEST_TBL1" (
  "COL1" NUMBER(*,0),
  "COL2" TIMESTAMP(6)
) COMPRESS FOR ARCHIVE REPLICA_NUM = 1 BLOCK_SIZE = 16384 USE_BLOOM_FILTER = FALSE ENABLE_MACRO_BLOCK_BLOOM_FILTER = FALSE TABLET_SIZE = 134217728 PCTFREE = 0 DYNAMIC_PARTITION_POLICY = (ENABLE = TRUE, TIME_UNIT = 'HOUR', PRECREATE_TIME = '3HOUR', EXPIRE_TIME = '1DAY', TIME_ZONE = '+8:00', BIGINT_PRECISION = 'NONE')
 partition by range(col2)
(partition "P0" values less than (Timestamp '2024-11-11 13:30:00.000000')) |
+-----------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
1 row in set
```

## 使用视图查看动态分区表信息

| 视图名 | 功能描述 |
|--------|---------|
| [sys.DBA_OB_DYNAMIC_PARTITION_TABLES](../../../../700.system-views/500.system-view-of-oracle-mode/200.dictionary-view-of-oracle-mode/12850.dba_ob_dynamic_partition_tables-of-oracle-mode.md)| 展示本租户的动态分区表信息。<main id="notice" type='explain'><h4>说明</h4><p>该视图从 V4.3.5 BP2 版本开始引入。</p></main>|
| [sys.V$OB_DYNAMIC_PARTITION_TABLES](../../../../700.system-views/500.system-view-of-oracle-mode/300.performance-view-of-oracle-mode/31810.v-ob_dynamic_partition_tables-of-oracle-mode.md)| 展示当前租户的动态分区表信息。该视图从内存中读取动态分区表相关属性，读取性能较好。<main id="notice" type='explain'><h4>说明</h4><p>该视图从 V4.3.5 BP2 版本开始引入。</p></main>|

**示例如下：**

```sql
SELECT * FROM sys.DBA_OB_DYNAMIC_PARTITION_TABLES
WHERE TABLE_NAME = 'TEST_TBL1';
```

返回结果如下：

```shell
+---------------+------------+----------+----------------------------------------+--------+-----------+----------------+-------------+-----------+------------------+
| DATABASE_NAME | TABLE_NAME | TABLE_ID | MAX_HIGH_BOUND_VAL                     | ENABLE | TIME_UNIT | PRECREATE_TIME | EXPIRE_TIME | TIME_ZONE | BIGINT_PRECISION |
+---------------+------------+----------+----------------------------------------+--------+-----------+----------------+-------------+-----------+------------------+
| SYS           | TEST_TBL1  |   507940 | Timestamp '2024-11-11 13:30:00.000000' | TRUE   | HOUR      | 3HOUR          | 1DAY        | +8:00     | NONE             |
+---------------+------------+----------+----------------------------------------+--------+-----------+----------------+-------------+-----------+------------------+
1 row in set
```
