# 删除 Catalog

当一个 Catalog 不再使用时，可以使用 `DROP CATALOG [IF EXISTS] external_catalog_name;` 语句将其删除。

在删除 Catalog 时，如果指定了 `IF EXISTS`，即使要删除的 Catalog 不存在也不会报错；如果不指定 `IF EXISTS`，则会报错。

**示例如下：**

```shell
obclient> DROP CATALOG IF EXISTS test_odps_catalog;
```
