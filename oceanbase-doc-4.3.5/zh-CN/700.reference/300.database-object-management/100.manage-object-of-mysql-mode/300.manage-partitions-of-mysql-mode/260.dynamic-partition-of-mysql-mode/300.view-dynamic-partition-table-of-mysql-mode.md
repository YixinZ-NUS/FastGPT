# 查看动态分区表

动态分区表创建成功后，您可以通过 `SHOW CREATE TABLE` 语句查看表的定义和视图查看数据库中的动态分区表信息。

## 查看表的定义

表创建成功后，您可以使用 `SHOW CREATE TABLE table_name;` 语句查看表的定义。

更多有关 `SHOW` 语句的介绍信息，参见 [SHOW](../../../../500.sql-reference/100.sql-syntax/200.common-tenant-of-mysql-mode/600.sql-statement-of-mysql-mode/8700.show-of-mysql-mode.md)。

**示例如下：**

查看表 `test_tbl1` 的定义。

```sql
SHOW CREATE TABLE test_tbl1;
```

查询结果如下：

```shell
+-----------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Table     | Create Table                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
+-----------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| test_tbl1 | CREATE TABLE `test_tbl1` (
  `col1` int(11) DEFAULT NULL,
  `col2` datetime DEFAULT NULL
) ORGANIZATION INDEX DEFAULT CHARSET = utf8mb4 ROW_FORMAT = DYNAMIC COMPRESSION = 'zstd_1.3.8' REPLICA_NUM = 1 BLOCK_SIZE = 16384 USE_BLOOM_FILTER = FALSE ENABLE_MACRO_BLOCK_BLOOM_FILTER = FALSE TABLET_SIZE = 134217728 PCTFREE = 0 DYNAMIC_PARTITION_POLICY = (ENABLE = TRUE, TIME_UNIT = 'HOUR', PRECREATE_TIME = '3HOUR', EXPIRE_TIME = '1DAY', TIME_ZONE = '+8:00', BIGINT_PRECISION = 'NONE')
 partition by range columns(col2)
(partition `P0` values less than ('2025-04-15 13:30:00')) |
+-----------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
1 row in set
```

## 使用视图查看动态分区表信息

| 视图名 | 功能描述 |
|--------|---------|
| [oceanbase.DBA_OB_DYNAMIC_PARTITION_TABLES](../../../../700.system-views/400.system-view-of-mysql-mode/200.dictionary-view-of-mysql-mode/8650.o-dba_ob_dynamic_partition_tables-of-mysql-mode.md)| 展示本租户的动态分区表信息。<main id="notice" type='explain'><h4>说明</h4><p>该视图从 V4.3.5 BP2 版本开始引入。</p></main>|
| [oceanbase.V$OB_DYNAMIC_PARTITION_TABLES](../../../../700.system-views/400.system-view-of-mysql-mode/300.performance-view-of-mysql-mode/30950.v-ob_dynamic_partition_tables-of-mysql-mode.md)| 展示当前租户的动态分区表信息。该视图从内存中读取动态分区表相关属性，读取性能较好。<main id="notice" type='explain'><h4>说明</h4><p>该视图从 V4.3.5 BP2 版本开始引入。</p></main>|

**示例如下：**

```sql
SELECT * FROM oceanbase.DBA_OB_DYNAMIC_PARTITION_TABLES
WHERE TABLE_NAME = 'test_tbl1';
```

返回结果如下：

```shell
+---------------+------------+----------+-----------------------+--------+-----------+----------------+-------------+-----------+------------------+
| DATABASE_NAME | TABLE_NAME | TABLE_ID | MAX_HIGH_BOUND_VAL    | ENABLE | TIME_UNIT | PRECREATE_TIME | EXPIRE_TIME | TIME_ZONE | BIGINT_PRECISION |
+---------------+------------+----------+-----------------------+--------+-----------+----------------+-------------+-----------+------------------+
| db_test       | test_tbl1  |   511391 | '2025-04-15 13:30:00' | TRUE   | HOUR      | 3HOUR          | 1DAY        | +8:00     | NONE             |
+---------------+------------+----------+-----------------------+--------+-----------+----------------+-------------+-----------+------------------+
1 row in set
```
