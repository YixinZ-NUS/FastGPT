# 租户总览

在左侧导航栏中单击 **租户管理** 进入 **租户管理** 页面，单击 **租户列表** 中租户名可进入租户 **总览** 页面，本文介绍 **总览** 页面的信息及该页面可进行的操作。

## 基本信息

**基本信息** 模块展示了租户的名称、模式、租户所在集群使用的 OceanBase 数据库版本号、字符集、Collation、连接字符串、表名大小写敏感、时区、锁定状态、创建时间以及备注。

您可单击 **连接字符串** 后的 <span style="vertical-align: middle;"><img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.icon/100.ob-dashboard-tenant-01.png" alt="复制" style="height: 20px;"></span> 图标复制连接串以备后续使用；单击 **锁定状态** 后的 <span style="vertical-align: middle;"><img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.icon/200.ob-dashboard-tenant-overview-01.png" alt="修改" style="height: 20px;"></span> 图标可修改租户的锁定状态，选择对应锁定状态后单击 **确定** 即可，锁定租户会导致用户无法访问该租户，请谨慎操作。

![基本信息](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/oceanbase-database-standalone/V4.3.5/obshell/V4.3.2/200.ob-dashboard-tenant-overview-01.png)


## 副本详情

**副本详情** 页面可查看副本所属的 **Zone 名称**、**副本类型**、**Unit 规格** 和 **Unit 数量**，支持修改副本和删除副本操作。

![副本详情](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.2.5/zh-CN/700.reference/1500.Components-and-Tools/100.obshell/400.tenant/200.ob-dashboard-tenant-overview-02.png)

* 修改副本

  单击 **操作** 列下的 **编辑** 字段可修改对应副本类型和 Unit 规格。对应 Unit 配置下会显示当前可配置的范围值，请根据取值范围进行配置，修改后单击 **操作** 列的 **确定**，二次确认后即可修改当前副本。

  您也可单击 **副本详情** 模块右上角的 **修改 Unit** 按钮修改所有 Zone 下的 Unit 数量和 Unit 规格。修改完成后单击 **确定**，obshell Dashboard 将生成批量修改 Unit 的任务，您可单击 **查看任务** 按钮查看任务的详细执行信息。

* 删除副本

  单击 **操作** 列下的 **删除** 字段，在弹出的窗口中单击 **删除**，obshell Dashboard 将生成删除副本的任务，您可单击 **查看任务** 按钮查看任务的详细执行信息。

  您也可勾选不同 Zone 前的勾选框批量删除多个副本，单击 **批量删除** 后，在弹出的窗口中单击 **删除**，obshell Dashboard 将生成批量删除副本的任务，您可单击 **查看任务** 按钮查看任务的详细执行信息。勾选副本后单击 **取消选择**，可取消全部副本的选择，即全部恢复为未勾选状态。

  <main id="notice" type='notice'>
    <h4>注意</h4>
    <p>删除副本数据将不可恢复，请谨慎操作。</p>
  </main>

## Zone 优先级

**Zone 优先级** 模块展示了租户中主副本分布的优先级。您可单击模块右侧的 **修改** 按钮修改租户中的 Zone 优先级。

![修改优先级](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/400.tenant/200.ob-dashboard-tenant-overview-03.png)

最高优先级的 Zone 会被视为 Primary Zone，且只能有一个。如不指定，默认以第一个 Zone 为第一优先级。排序方法如下：

1. 勾选左侧列表框中的一个或多个 Zone。左侧列表框中显示了当前集群中所有可选的 Zone。

2. 单击中间的 **>** 按钮。此时勾选的 Zone 就会被移动到 <b>优先级排序</b> 列表中。同时勾选的多个 Zone 具有相同的优先级。

3. 重复 1、2 动作，添加低一优先级的 Zone。

4. 如需调整优先级，可在 <b>优先级排序</b> 列表中拖拽以调整顺序。列表中从上到下，优先级依次递减。

5. 单击 Zone 后的删除按钮，或单击 **恢复为默认** 可将 <b>优先级排序</b> 列表中的 Zone 重新移回左侧列表。

修改后单击 **确定** 按钮，obshell Dashboard 将生成修改 Zone 优先级的任务，您可单击 **查看任务** 按钮查看任务的详细执行信息。

## 白名单

**白名单** 模块展示了能够连接到租户的地址列表。您可单击模块右侧的 **修改** 按钮修改白名单。

![修改白名单](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/400.tenant/200.ob-dashboard-tenant-overview-04.png)

支持自定义设置可访问 IP，或设置为所有 IP 都可访问。选择自定义设置时，可在 **自定义 IP** 下的输入框中指定允许登陆的客户端列表，输入 IP 后单击键盘 Enter 键或输入英文逗号（`,`）可配置多个节点 IP。光标悬停在 **查看配置说明** 上可查看配置 IP 的说明，配置 IP 时支持的格式如下：

* IP 地址，示例：127.0.0.10,127.0.0.11

* 子网/掩码，示例：127.0.0.0/24

* 模糊匹配，示例：127.0.0.% 或 127.0.0._

* 多种格式混合，示例：127.0.0.10,127.0.0.11,127.0.0.%,127.0.0._,127.0.0.0/24

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>设置为 <code>%</code> 表示所有客户端都可以连接，存在访问安全风险，请谨慎操作。</p>
</main>

## 合并管理

合并管理 模块展示了当前租户的 **状态**、**最近 1 次合并开始时间** 和 **最近 1 次合并结束时间**。单击 **状态** 后的 **发起合并**，在弹出的窗口中单击 **确定** 即可发起合并。

![发起合并](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/400.tenant/200.ob-dashboard-tenant-overview-05.png)

## 新增副本

单击 **总览** 页面右上角 **新增副本** 按钮，在弹出的窗口中配置副本信息，配置完成后单击 **确定** 按钮，obshell Dashboard 将生成新增副本的任务，您可单击 **查看任务** 按钮查看任务的详细执行信息。

![新增副本](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/400.tenant/200.ob-dashboard-tenant-overview-10.png)

| 参数  | 说明  |
|-------|------|
| 目标 Zone | 可通过下拉框中选择在哪个 Zone 中新增副本。 |
| 副本类型  | 可通过下拉框选择副本类型，支持配置为 **全能型副本** 和 **只读型副本**。 |
| Unit 规格 | 设置 Unit 的 CPU 规格上限和内存规格。  |
| Unit 数量 | 展示各 Zone 的 Unit 数量，不支持修改。  |

## 删除租户

单击 **总览** 页面右上角 **删除租户** 按钮，在弹出的窗口输入栏中输入 `delete` 后单击 **删除** 按钮，obshell Dashboard 将生成删除租户的任务，您可单击 **查看任务** 按钮查看任务的详细执行信息。

![删除租户](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/400.tenant/200.ob-dashboard-tenant-overview-06.png)
