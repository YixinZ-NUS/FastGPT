# 数据库管理

在左侧导航栏中单击 **租户管理**，在 **租户管理** 页面单击 **租户列表** 中租户名，进入对应租户 **总览** 页面后单击左侧导航栏 **数据库管理**，即可进入 **数据库管理** 页面，本文介绍 **数据库管理** 页面的信息及该页面可进行的操作。

![总览](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/400.tenant/300.ob-dashboard-tenant-db-01.png)

## 新建数据库

单击 **数据库管理** 页面右上角 **新建数据库** 按钮，在 **新建数据库** 页面中配置 **数据库名**、**字符集**、**Collation** 以及是否只读。设置完成后单击 **提交** 即完成新建。

![新建数据库](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/400.tenant/300.ob-dashboard-tenant-db-03.png)

| 参数  | 说明  |
|-------|------|
| 数据库名 | 设置新建数据库的名称，数据库名需满足如下要求：<ul><li>长度为 2 ~ 128 个字符。</li><li>可包含小写英文字母、数字和下划线，但需以英文字母开头。</li></ul> |
| 字符集 | 可通过下拉框中选择字符集。 |
| Collation  | 可通过下拉框中选择合适的 Collation。 |
| 只读 | 勾选设置待新建数据库是否为只读数据库。  |

## 修改数据库

单击 **操作** 列下的 **编辑** 字段即可修改对应数据库的 **字符集**、**Collation** 和是否只读。修改内容后单击 **提交** 即可完成对数据库的修改。

![编辑数据库](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/400.tenant/300.ob-dashboard-tenant-db-02.png)

## 删除数据库

单击 **操作** 列下的 **删除** 字段，在弹出的窗口输入栏中输入 `delete` 后单击 **删除** 按钮即可删除对应数据库。

![删除数据库](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/400.tenant/300.ob-dashboard-tenant-db-04.png)
