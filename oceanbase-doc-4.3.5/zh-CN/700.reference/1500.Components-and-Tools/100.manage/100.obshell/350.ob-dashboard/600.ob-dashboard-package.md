# 软件包管理

单击左侧导航栏 **系统配置** > **软件包管理** 进入 **软件包** 页面。本文介绍如何通过 obshell Dashboard 查看或上传软件包。

## 查看软件包

在软件包页面展示了 obshell Dashboard 中已上传的软件包的 **软件包名**、**版本**、**硬件架构**、**大小（MB）**、**MD5** 和 **上传时间**。

您可通过 **版本** 和 **硬件架构** 后的 <span style="vertical-align: middle;"><img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.icon/100.ob-dashboard-tenant-02.png" alt="筛选" style="height: 20px;"></span> 图标按照软件包的版本和硬件架构筛选查看软件包，也可通过 **软件包名** 和 **上传时间** 后的 <span style="vertical-align: middle;"><img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.icon/100.ob-dashboard-tenant-03.png" alt="排序" style="height: 20px;"></span> 图标选择是否按照升序或降序排列软件包。若已上传的软件包过多，可通过页面左上角搜索框搜索是否存在指定软件包。

![查看](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/600.ob-dashboard-package-02.png)

## 上传软件包

单击 **软件包** 页面右上角 **上传软件包** 按钮，单击 **选择软件包** 从本地上传安装包。若本地无对应安装包，可单击 **RPM 包下载地址** 地方访问开源镜像站下载对应软件包。

![上传软件包](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/600.ob-dashboard-package-01.png)
