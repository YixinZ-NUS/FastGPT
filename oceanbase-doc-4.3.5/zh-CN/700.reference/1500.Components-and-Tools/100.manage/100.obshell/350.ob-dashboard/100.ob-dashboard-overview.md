# obshell Dashboard

obshell Dashboard 是 obshell 提供的基于 Web 的交互式管理页面，专为 OceanBase 集群和租户资源的管理而设计。它提供了关键资源和性能指标的监控功能，帮助用户高效管理 OceanBase 集群。

obshell Dashboard 服务完全集成在 obshell 进程中，用户可以通过访问任何部署了 obshell 的服务地址来管理 OceanBase 集群。这种设计不仅简化了管理流程，还确保了极低的资源开销，为用户提供简单易用的管控体验。

## 特性介绍

obshell Dashboard 目前支持 OceanBase 集群管理的以下特性：

* 集群管理：查看 [集群信息](200.ob-dashboard-cluster-overview.md)，包含状态信息、资源消耗、拓扑结构等核心信息，以及 [管理集群](300.ob-dashboard-cluster.md)、[管理参数](500.ob-dashboard-parameter.md) 等功能。

* 租户管理：管理集群下的租户及租户中的数据库、用户和参数，具体介绍可参见 [租户管理](400.ob-dashboard-tenant/100.ob-dashboard-tenant.md) 章节。

* 备份恢复：支持租户级备份恢复能力。具体介绍可参加 [备份恢复](400.ob-dashboard-tenant/500.backup-and-recovery/100.tenant-backup-and-recovery-overview.md) 章节。

* 性能监控：展示了集群维度、租户维度、主机维度、OBServer 维度、Unit 维度等多个维度的性能数据，包含数据库、主机、性能与 SQL、事务、存储与缓存等多个方面，具体介绍可参见 [性能监控](420.Performance-monitoring.md) 。

* 告警中心：obshell Dashboard 提供了告警能力，可通过告警中心查看告警事件、设置告警屏蔽和告警规则，以便及时响应和处理异常情况，确保系统稳定运行，具体介绍可参加 [告警中心](450.alarm-center/100.alarm-overview.md) 章节。

* 软件包管理：上传软件包，升级集群和 obshell 需保证已上传目标版本的软件包。具体操作可参见 [软件包管理](600.ob-dashboard-package.md)。

## 登录与退出

自 obshell V4.3.0.0 起支持通过图形化界面交互式管理集群，在成功安装 obshell 后，使用 obshell 节点 IP 和 obshell 端口即可访问 obshell Dashboard（obshell 交互式管理页面）。访问时默认使用 sys 租户下的 root 用户登录，密码为部署 OceanBase 集群时设置的 root@sys 用户密码，若 root@sys 密码为空，可直接单击 **登录** 按钮登陆。

登录 obshell Dashboard 控制台后，光标悬停在页面右上角 <span style="vertical-align: middle;"><img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.icon/100.ob-dashboard-overview-01.png" alt="用户" style="height: 20px;"></span> 图标处，单击出现的 **退出登录**，在弹出的窗口单击 **确定** 即可退出登录。

## 查看版本

登录 obshell Dashboard 控制台后，单击页面右上角 **帮助** > **关于 obshell Dashboard**，即可查看 obshell Dashboard 的版本。
