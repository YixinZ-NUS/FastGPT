# 告警概述

obshell Dashboard 告警中心用于集中管理和监控 OceanBase 集群中的告警事件。您可通过该页面查看告警事件、设置告警屏蔽和告警规则，以便及时响应和处理异常情况，确保系统稳定运行。

## 访问路径

登录 obshell Dashboard 控制台后，输入 Alertmanager 地址、用户名、密码，单击 **开启** 即可进入 **告警中心** 页面，您可以使用已有的 Alertmanager 地址开启告警中心，若暂未部署，建议使用 obd 安装部署。在告警中心页面上方点击 **告警事件**、**告警屏蔽**、**告警规则** 即可进入对应页签。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>仅首次登录告警中心页面需输入 Alertmanager 地址、用户名、密码等信息。</p>
</main>

## 告警事件

在 **告警事件** 页签中，支持根据  **对象类型**、 **告警对象**、 **告警等级**、 **关键词**、**开始时间**、**结束时间** 等条件进行精确查找。

告警事件列表中可查看具体告警的信息，包括 **告警事件**、**告警对象**、**告警等级**、**告警状态**、**产生时间**、**结束时间**、**操作** 。支持根据 **告警等级**、**告警状态**、**产生时间**、**结束时间** 进行排序。

更多介绍可参见 [查看告警事件](200.view-alarm-events.md)。

![告警事件](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.4.1/obshell/V4.3.2/450.alarm-center/100.alarm-overview-1.png)

## 告警屏蔽

在 **告警屏蔽** 页签查询区域输入查询条件，可根据 **对象类型**、**屏蔽对象**、**关键词** 进行屏蔽条件精确查找，并可对指定屏蔽内容进行 **编辑** 和 **解除屏蔽** 操作。

在屏蔽列表中，您可查看屏蔽条件的 **屏蔽应用/对象类型**、**屏蔽对象**、**屏蔽告警规则**、**屏蔽结束时间**、**创建人**、**状态**、**创建时间**、**备注**、**操作** 等信息，支持根据 **屏蔽结束时间**、**状态**、**创建时间** 进行排序，并可对指定屏蔽内容进行 **编辑** 和 **解除屏蔽** 操作。详情参见：[管理告警屏蔽](300.alert-shielding-management.md)。

![告警屏蔽](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.4.1/obshell/V4.3.2/450.alarm-center/100.alarm-overview-02.png)

## 告警规则

在 **告警规则** 页签，可通过 **对象类型**、**告警等级**、**关键词** 进行精确查找。

在规则列表中，可查看 **告警规则名**、**触发规则**、**持续时间**、**对象类型**、**告警等级**、**类型** 等信息。支持根据 **告警等级** 进行排序，支持通过 **类型** 筛选 **自定义** 或 **默认** 类型的告警规则。

![告警规则](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/oceanbase-database-standalone/V4.3.5/obshell/V4.3.2/100.alarm-overview-03.png)
