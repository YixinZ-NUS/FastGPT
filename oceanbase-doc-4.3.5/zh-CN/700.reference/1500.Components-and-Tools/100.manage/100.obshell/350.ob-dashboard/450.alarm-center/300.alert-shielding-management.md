# 管理告警屏蔽

## 新建屏蔽

### 操作步骤

1. 登录 obshell Dashboard 控制台。

2. 在左侧导航栏中选择 **告警中心** 。

3. 单击 **告警屏蔽** 页签。

4. 在 **告警屏蔽** 页签中，单击 **新建屏蔽** 。

5. 在 **屏蔽条件** 页面配置屏蔽条件。

   | 参数  | 说明  |
   |-------|------|
   | 屏蔽对象类型 | 依据实际需要，点击屏蔽对象的下拉框选择具体的告警对象。<ul><li>当屏蔽对象类型是 obcluster 时，屏蔽对象需要选择目标集群。</li><li>当屏蔽对象类型是 obtenant 时，屏蔽对象需要选择对应目标集群和目标租户（全部租户或指定目标租户）。</li><li>当屏蔽对象类型是 observer 时，屏蔽对象需要选择对应目标集群和目标主机（全部主机或指定 IP 主机）。 |
   | 屏蔽告警规则 | 在对应下拉框中选择需要屏蔽的告警，可多选。告警规则和 **告警规则**页签的规则列表保持一致，您可到该页签查看详情。 |
   | 标签 | 支持对指定的指标进行屏蔽。按照标签匹配条件屏蔽告警，支持值匹配或者正则表达式，当所有条件都满足时告警才会被屏蔽。为可选项。 |
   | 屏蔽结束时间 | 配置告警屏蔽的结束时间。在结束时间前，被屏蔽的告警不会被发送。为必填项。有如下两种方式配置屏蔽结束时间：<ul><li>可单击方框内自定义结束时间。</li><li>也可单击方框后的时间，如单击 6 小时 ，系统会将当前时间 + 6 小时填充到方框中。</li></ul> |
   | 备注信息 | 可添加屏蔽规则说明，方便后续查看识别以及重复使用。为必填项。 |

   <main id="notice" type='explain'>
      <h4>说明</h4>
      <p>如果需要屏蔽某条告警，建议使用 <b>告警事件</b> 页面事件列表中右侧的 <b>屏蔽</b> 按钮，将自动跳转到 <b>告警屏蔽</b> 页签下的 <b>屏蔽条件</b> 页面，并自动填充与与该告警事件关联的屏蔽对象类型、屏蔽告警规则、标签 信息，您只需配置告警屏蔽的结束时间和添加备注信息，即可成功屏蔽相应告警。</p>
   </main>

   ![屏蔽条件](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.4.1/obshell/V4.3.2/450.alarm-center/300.alert-shielding-management-01.png)

6. 单击 **提交** 。系统返回 **屏蔽列表** ，且列表中会新增一条数据即新建的屏蔽。


## 编辑屏蔽

本小节为您介绍如何对已创建的告警屏蔽条件进行编辑。

### 前提条件

屏蔽列表中存在待修改的屏蔽信息。

### 操作步骤

1. 登录 obshell Dashboard 控制台。

2. 在左侧导航栏中选择 **告警中心** 。

3. 单击 **告警屏蔽** 页签。

4. 在 **告警屏蔽** 页签的查询区域输入查询条件，支持对屏蔽进行精确查找，**屏蔽列表** 中将返回所有符合查询条件的屏蔽信息。

5. 选择需要编辑的屏蔽，在其对应的 **操作** 列单击 **编辑**。

   ![编辑](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.4.1/obshell/V4.3.2/450.alarm-center/300.alert-shielding-management-02.png)

6. 进入 **屏蔽条件** 页面，可以对相应屏蔽的屏蔽条件进行编辑，包括屏蔽对象类型、屏蔽对象、屏蔽告警规则、标签、屏蔽时间、备注。具体参数可参照上文 [新建屏蔽](#新建屏蔽) 中的参数说明进行配置。

   ![屏蔽条件](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.4.1/obshell/V4.3.2/450.alarm-center/300.alert-shielding-management-03.png)

7. 单击 **提交**，即成功编辑屏蔽。

## 解除屏蔽

本小节为您介绍如何解除已创建的告警屏蔽。

### 前提条件

已新建屏蔽。

### 操作步骤

1. 登录 obshell Dashboard 控制台。

2. 在左侧导航栏中选择 **告警中心** 。

3. 单击 **告警屏蔽** 页签。

4. 在 **告警屏蔽** 页签的查询区域输入查询条件，支持对屏蔽进行精确查找，**屏蔽列表** 中将返回所有符合查询条件的屏蔽信息。

5. 选择需要解除的屏蔽，在其对应的 **操作** 列单击 **解除屏蔽**。

   ![解除屏蔽](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.4.1/obshell/V4.3.2/450.alarm-center/300.alert-shielding-management-04.png)

6. 在跳出的二次确认框中单击 **解除**，即成功解除屏蔽。

   ![单击解除](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.4.1/obshell/V4.3.2/450.alarm-center/300.alert-shielding-management-05.png)

   <main id="notice" type='explain'>
     <h4>说明</h4>
     <p>屏蔽解除后不可恢复，请谨慎操作。</p>
   </main>

7. 解除屏蔽后，屏蔽列表对应屏蔽对象的状态将由 **活跃** 变为 **过期**，操作列的 **编辑** 和 **解除屏蔽** 按钮将变为灰色，不可继续进行任何操作。

   ![屏蔽后页面](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.4.1/obshell/V4.3.2/450.alarm-center/300.alert-shielding-management-06.png)
