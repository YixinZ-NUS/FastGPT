# 参数管理

本文介绍如何通过 obshell Dashboard 查看并修改 OceanBase 集群的参数。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>通过 <b>集群管理</b> 页面查看到的参数均为 OceanBase 集群的配置项（集群级和租户级），通过 <b>租户管理</b> 页面查看到的参数为 OceanBase 集群的租户级配置项和 Global 系统变量。</p>
</main>

## 集群管理页面

单击左侧导航栏 **集群管理**，在 **集群管理** 页面单击右上角 **参数管理** 进入 **参数管理** 页面。

**参数管理** 页面展示了配置项的 **参数名称**、**类别**、**当前值**、**说明**、**是否重启生效**、是否 **只读**，以及可对参数进行的操作。如需查看指定的配置项，您可在页面右上角搜索框中搜索对应配置项。您也可以通过页面中不同列后的筛选按钮按照不同类型查看配置项，目前支持按照参数类别（**类别**）、生效范围（**当前值**）、生效方式（**是否重启生效**）、是否可修改（**只读**）几种类型查看配置项。

![集群](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.ob-dashboard-parameter-01.png)

单击 **操作** 列的 **修改值** 字段可修改对应配置项的 **值**、**生效范围** 和 **生效对象**。如需设置不同生效范围的值，可单击 **+ 添加值** 设置多个值。设置完成后单击 **确定** 按钮，在确定值修改信息后单击 **确定** 即可完成参数的修改。

![修改参数](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.ob-dashboard-parameter-02.png)

## 租户管理页面

在左侧导航栏中单击 **租户管理** 进入 **租户管理** 页面，单击 **租户列表** 中租户名，进入租户 **总览** 页面后单击左侧导航栏 **参数管理** 即可进入 **参数管理** 页面。

**参数管理** 页面展示了配置项的 **参数名称**、**当前值**、**说明**、是否 **只读**，以及可对参数进行的操作。如需查看指定的配置项，您可在页面右上角搜索框中搜索对应配置项。您也可以通过页面 **只读** 后的 <span style="vertical-align: middle;"><img src="https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.icon/100.ob-dashboard-tenant-02.png" alt="筛选" style="height: 20px;"></span> 图标按照参数是否可修改筛选查看配置项。

![租户](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.ob-dashboard-parameter-03.png)

单击 **操作** 列的 **修改值** 字段可修改对应配置项的值。设置完成后单击 **提交** 按钮即可完成参数的修改。

![修改参数](https://obbusiness-private.oss-cn-shanghai.aliyuncs.com/doc/img/observer-enterprise/V4.3.5/700.reference/1500.Components-and-Tools/100.obshell/500.ob-dashboard-parameter-04.png)
