# 下载 RPM 包

本文介绍如何通过 obshell-sdk-python 下载所需的 RPM 包。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>建议先查看 <a href='../100.quickstart-of-python.md'>obshell-sdk-python 快速上手</a> 内容，了解如何使用 obshell-sdk-python。</p>
</main>

## 示例代码

:::tab

tab 下载与本机同架构的 RPM 包

* 下载最新版本的 RPM 包

  ```python
  from obshell import download_package

  # 下载与执行机器同架构和系统的最新的 obshell 包到指定目录
  def download_same_arch():
      dest = download_package("/root/download", "obshell")
      print (dest)
  ```

* 下载指定版本的 RPM 包
  
  ```python
  from obshell import download_package

  # 下载与执行机器同架构和系统的指定版本的社区版 OceanBase 数据库包
  def download_same_arch_version():
      dest = download_package("/root/download", "oceanbase-ce", version="4.3.5.0")
      print (dest)
  ```

tab 下载指定架构或系统的 RPM 包

* 下载最新版本的 RPM 包
  
  ```python
  from obshell.mirror import BASE_COMMUNITY_MIRROR, EL8, AARCH64

  # 下载与执行机器不同架构的最新的 obshell 包
  def download_diff_arch():
      # 指定架构和系统分别为 aarch64，el8
      mirror = BASE_COMMUNITY_MIRROR.get_mirror(EL8, AARCH64)
      dest = mirror.download("/root/download", "obshell")
      print (dest)
  ```

* 下载指定版本的 RPM 包

  ```python
  from obshell.mirror import BASE_COMMUNITY_MIRROR, EL8, AARCH64

  # 下载与执行机器不同架构的指定版本的obshell包
  def download_diff_arch_version():
      # 指定架构和系统分别为 aarch64，el8
      mirror = BASE_COMMUNITY_MIRROR.get_mirror(EL8, AARCH64)
      dest = mirror.download("/root/download", "obshell", version="4.2.4.0")
      print (dest)
  ```

:::
