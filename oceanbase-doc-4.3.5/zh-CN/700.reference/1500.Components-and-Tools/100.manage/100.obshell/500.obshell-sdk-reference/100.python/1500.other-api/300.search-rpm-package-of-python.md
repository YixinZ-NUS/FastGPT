# 搜索 RPM 包

本文介绍如何通过 obshell-sdk-python 搜索所需的 RPM 包。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>建议先查看 <a href='../100.quickstart-of-python.md'>obshell-sdk-python 快速上手</a> 内容，了解如何使用 obshell-sdk-python。</p>
</main>

## 示例代码

:::tab

tab 搜索本机同架构的 RPM 包

* 搜索最新版本的 RPM 包

  ```python
  from obshell import search_package

  # 搜索与执行机器同架构和系统的最新的 obshell 包
  def search_same_arch():
      pkgs = search_package("obshell")
      print (pkgs)
  ```

* 搜索指定版本的 RPM 包
  
  ```python
  from obshell import search_package

  # 搜索与执行机器同架构和系统的指定版本的社区版 OceanBase 数据库包
  def search_same_arch_version():
      pkgs = search_package("oceanbase-ce", version="4.3.5.0")
      print (pkgs)
  ```

tab 搜索指定架构或系统的 RPM 包

* 搜索最新版本的 RPM 包
  
  ```python
  from obshell.mirror import BASE_COMMUNITY_MIRROR, EL8, AARCH64

  # 搜索与执行机器不同架构的最新的 obshell 包
  def search_diff_arch():
      # 指定架构和系统分别为 aarch64，el8 
      mirror = BASE_COMMUNITY_MIRROR.get_mirror(EL8, AARCH64)
      pkgs = mirror.search("obshell")
      print (pkgs)
  ```

* 搜索指定版本的 RPM 包

  ```python
  from obshell.mirror import BASE_COMMUNITY_MIRROR, EL8, AARCH64

  # 搜索与执行机器不同架构的指定版本的 obshell 包
  def search_diff_arch_version():
      # 指定架构和系统分别为 aarch64，el8
      mirror = BASE_COMMUNITY_MIRROR.get_mirror(EL8, AARCH64)
      pkgs = mirror.search("obshell", version="4.2.4.0")
      print (pkgs)
  ```

:::
