# 接管集群

本文介绍如何通过 obshell-sdk-python 接管集群。

  <h4>说明</h4>
  <p>建议先查看 <a href='../100.quickstart-of-python.md'>obshell-sdk-python 快速上手</a> 内容，了解如何使用 obshell-sdk-python。</p>

## 注意事项

在接管集群前请确保所需的 obshell 已被解压安装到每个机器上。可以手动解压，也可以使用 obshell-sdk-python 解压安装，使用 obshell-sdk-python 解压安装的操作可参见 [安装 obshell](../110.obshell-management/300.install-obshell-of-python.md)。

## 示例代码

```python
from obshell import takeover, NodeConfig

def init_nodes():
    ips = [
        "10.10.10.1",
        "10.10.10.2",
        "10.10.10.3",
    ]
    work_dir = "/data/ob" # OceanBase 数据库的工作目录
    nodes_config = []
    for _, ip in enumerate(ips):
        node = NodeConfig(ip, work_dir)
        nodes_config.append(node)

    # 接管
    # 参数说明： 
    # password: 待被接管集群 root@sys 密码
    # configs: 节点配置信息
    takeover("pwd", nodes_config)
```
