# 接管集群

本文介绍如何通过 obshell-sdk-go 接管集群。

  <h4>说明</h4>
  <p>建议先查看 <a href='../100.quickstart-of-go.md'>obshell-sdk-go 快速上手</a> 内容，了解如何使用 obshell-sdk-go。</p>

## 注意事项

在接管集群前请确保所需的 obshell 已被解压安装到每个机器上。可以手动解压，也可以使用 obshell-sdk-go 解压安装，使用 obshell-sdk-go 解压安装的操作可参见 [安装 obshell](../110.obshell-management/300.install-obshell-of-go.md)。

## 示例代码

```go
func main() {
    ips := []string{
        "10.10.10.1",
        "10.10.10.2",
        "10.10.10.3",
    }

    workDir := "/data/ob" // OceanBase 数据库的工作目录
    nodeConfigs := make([]util.NodeConfig, 0)
    for _, ip := range ips {
        nodeConfigs = append(nodeConfigs, util.NewNodeConfig(ip, workDir, 2886))
    }

    // 参数说明： 
    // password: 待被接管集群 root@sys 密码
    // configs: 节点配置信息
    if err := util.Takeover("pwd", nodeConfigs...); err != nil {
        panic(err)
    }
}
```
