# 启动 obshell

本文介绍如何通过 obshell-sdk-go 启动 obshell 进程。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>建议先查看 <a href='../100.quickstart-of-go.md'>obshell-sdk-go 快速上手</a> 内容，了解如何使用 obshell-sdk-go。</p>
</main>

## 注意事项

在执行启动 obshell 之前请先确保所有节点的工作目录下已安装 obshell。可以手动下载 RPM 并解压安装，也可以使用obshell-sdk-go 进行节点初始化，初始化过程中会安装 obshell，使用 obshell-sdk-go 进行节点初始化的详细介绍可参见 [初始化节点](../200.cluster-management/550.initialize-node-of-go.md)。

## 示例代码

```go
package main

import (
    "github.com/oceanbase/obshell-sdk-go/util"
)

func startObshell() error {
    ips := []string{
        "10.10.10.1",
        "10.10.10.2",
        "10.10.10.3",
    }

    workDir := "/data/ob" // OceanBase 数据库的工作目录
    nodeConfigs := make([]util.NodeConfig, 0)
    for _, ip := range ips {
        nodeConfigs = append(nodeConfigs, util.NewNodeConfig(ip, workDir, 2886))
    }

    // 启动 obshell
    return util.StartObshell(nodeConfigs...)
}
```
