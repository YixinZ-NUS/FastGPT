# 删除 Zone

本文介绍如何通过 obshell-sdk-go 删除集群内指定的 Zone 以及 Zone 内的节点。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>建议先查看 <a href='../100.quickstart-of-go.md'>obshell-sdk-go 快速上手</a> 内容，了解如何使用 obshell-sdk-go。</p>
</main>

## 注意事项

对集群进行缩容前，需注意以下要求：

* 请求的 client 对应的节点需在待缩容的集群中，且不能位于待删除的 Zone 内。

* obshell 需正常运行。

* 待删除的 Zone 上不存在任何 unit。

## 示例代码

:::tab

tab 任务同步执行

```go
package main

import (
    "github.com/oceanbase/obshell-sdk-go/services"
)

func main() {
    // 创建 client 实例，节点地址为 '10.10.10.1'，端口为 2886。
    // 所在集群的 root@sys 密码为 '****'。
    client, err := services.NewClientWithPassword("10.10.10.1", 2886, "****")
    if err != nil {
        panic(err)
    }

    // 将 zone3 从集群中删除。
    req := client.V1().NewDeleteZoneRequest("zone3")
    if _, err := client.V1().DeleteZoneSyncWithRequest(req); err != nil {
        panic(err)
    }
}
```

tab 任务异步执行

```go
package main

import (
    "github.com/oceanbase/obshell-sdk-go/services"
)

func main() {
    // 创建 client 实例，节点地址为 '10.10.10.1'，端口为 2886。
    // 所在集群的 root@sys 密码为 '****'。
    client, err := services.NewClientWithPassword("10.10.10.1", 2886, "****")
    if err != nil {
        panic(err)
    }

    // 将 zone3 从集群中删除。
    req := client.V1().NewDeleteZoneRequest("zone3")
    dag, err := client.V1().DeleteZoneWithRequest(req)
    if err != nil {
        panic(err)
    }

    // 等待任务成功。
    if dag != nil && dag.GenericDTO != nil {
        if _, err = client.V1().WaitDagSucceed(dag.GenericID); err != nil {
            panic(err)
        }
    }
}

```
