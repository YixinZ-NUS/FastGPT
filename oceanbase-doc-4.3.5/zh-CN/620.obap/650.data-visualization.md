# 数据可视化

OceanBase 作为分布式数据库，擅长处理海量数据的存储与计算，通过集成主流 BI 工具（Superset、Power BI 等），OceanBase 用户可快速将复杂数据转化为直观图表，支持业务决策、实时监控与多维分析。  

## 支持的 BI 工具

### Apache Superset

- **灵活自定义**：支持 SQL 直接查询 OceanBase 表，自定义仪表盘与数据看板。  
- **SQL查询优化**：利用 OceanBase 的 HTAP 特性，直接在 Superset 中执行复杂分析 SQL。

更多信息，参见[使用 Superset 和 OceanBase 进行数据分析](../680.ecological-integration/700.visualization/100.superset-mysql-connection-oceanbase-sample-program.md)。

### Power BI

- **企业级协作**：支持权限分层、数据集共享与实时刷新。  
- **ODBC 驱动**：安装 OceanBase ODBC 驱动，通过 Power BI Desktop 连接集群。  
- **实时分析**：利用 Power BI 的 DirectQuery 模式，直接关联 OceanBase 进行动态查询。  

更多信息，参见[在 Power BI 中连接 OceanBase 并获取数据](../680.ecological-integration/700.visualization/300.power-bi.md)。

### Quick BI

- **直连 OceanBase**：通过阿里云控制台配置数据源，无需额外驱动。  
- **跨产品协同**：无缝对接 DataWorks、MaxCompute，构建端到端分析链路。  

更多信息，参见[在 Quick BI 中连接 OceanBase 并获取数据](../680.ecological-integration/700.visualization/400.quick-bi.md)。

### Tableau

- **企业级部署**：提供 Server 版本支持大规模团队协作。  
- **JDBC/ODBC 驱动**：安装 Tableau 连接器，配置 OceanBase 集群地址与认证信息。  
- **数据提取（Extract）**：离线存储数据加速复杂分析。  

更多信息，参见[在 Tableau 中连接 OceanBase 数据库](../680.ecological-integration/700.visualization/500.tableau.md)。

### 观远 BI

- **AI 增强分析**：内置预测模型（如销售趋势预测）与异常检测功能。   
- **直连OceanBase**：通过观远 BI 的数据库连接器配置集群IP、端口与账号。  
- **预置模板**：提供电商、金融等行业的 OceanBase 数据看板模板。  

更多信息，参见[在观远 BI 中连接 OceanBase 数据库](../680.ecological-integration/700.visualization/600.guanyuan.md)。

### 永洪 BI

- **高性能处理**：支持亿级数据秒级响应，适配 OceanBase 的 HTAP 架构。   
- **JDBC驱动**：配置 OceanBase 的 JDBC 连接参数，支持分布式查询优化。  
- **数据治理**：内置数据清洗工具，自动适配 OceanBase 的分区表与索引结构。  

更多信息，参见[在 vividime 中连接 OceanBase 数据库](../680.ecological-integration/700.visualization/700.yonghong.md)。
