# 数据加工工具

数据加工工具是连接原始数据与 OceanBase 数据库的核心桥梁。以下简要介绍主流工具。

## OceanBase Developer Center (ODC)

OceanBase 开发者中心（OceanBase Developer Center，ODC）是数据库图形化开发工具，也是数据研发和生产变更管控协同平台。ODC 有桌面版、Web 版两种产品形态。桌面版侧重数据库开发工具能力，支持 Windows、Mac、Linux 操作系统，具有轻量化和易部署的特性。Web 版在提供工具能力的同时还提供了管控、协同能力，侧重数据库变更的安全、合规和效率。

如需了解如何使用 ODC，参见[OceanBase 开发者中心（ODC）](../../700.reference/1400.platform-products/400.oceanbase-developer-center.md)。

## 生态 ETL 工具

### Flink

Flink 是一个用于大规模数据处理和分析的开源框架，Flink CDC 是在该平台上实现的一个用于捕获数据库变更事件的组件。二者结合起来，可以提供一套强大的实时数据处理和分析解决方案。

如需了解如何使用 Flink，参见[使用 Flink CDC 从 MySQL 数据库同步数据到 OceanBase 数据库](../../500.data-migration/200.migrate-data-from-mysql-database-to-oceanbase-database/600.use-flink-cdc-to-migrate-data-from-mysql-database-to-oceanbase-database.md) 和 [使用 Flink CDC 从 OceanBase 数据库迁移数据到 MySQL 数据库](../../500.data-migration/300.migrate-data-from-oceanbase-database-to-mysql-database/500.use-flink-cdc-to-migrate-data-from-oceanbase-database-to-mysql-database.md)。

### dbt（Data Build Tool）

dbt（data build tool）是一款开源数据转换工具，能够通过 SQL 实现数据转化，将命令转化为表或者视图。本文介绍如何使用 dbt-oceanbase，通过 dbt 分析 OceanBase 数据库中的数据。

如需了解如何使用 dbt，参见[使用 dbt 分析 OceanBase 数据](../../680.ecological-integration/400.data-ingestion/200.dbt.md)。

### DataWorks

DataWorks 是阿里云上的数据开发治理平台，基于 MaxCompute、Hologres、EMR、AnalyticDB 和 CDP 等大数据引擎，为数据仓库、数据湖、湖仓一体等解决方案提供统一的全链路处理能力。DataWorks 具有数据集成、数据开发、数据建模、数据分析等多种功能。
如需了解如何使用 DataWorks，参见 [DataWorks](https://www.oceanbase.com/docs/common-oceanbase-cloud-1000000001738701)。

### AWS Glue

AWS Glue ‌是一个无服务器数据集成服务，旨在帮助用户轻松发现、准备、移动和集成来自多个来源的数据，以支持分析、机器学习和应用程序开发。它提供了全面的数据集成功能，包括数据发现、现代 ETL、数据清理和转换，以及集中式编目，所有这些功能都整合在单一的服务中。AWS Glue 无需用户管理基础设施，支持 ETL、ELT 和流式传输等多种工作负载，并且可以按需扩展，以适应任何数据大小和类型。

如需了解如何使用 AWS Glue，参见[使用 AWS Glue 迁移 OceanBase 数据](https://www.oceanbase.com/docs/common-oceanbase-cloud-1000000001478166)。

#### 工具选型建议

| **工具**         | **核心优势**                          | **适用场景**                      |
|------------------|---------------------------------------|----------------------------------|
| **ODC**          | OceanBase 专用管理工具，深度集成生态   | OceanBase 数据库的开发与运维      |
| **Flink**        | 实时流处理与高吞吐批处理              | 流数据监控、实时报表生成          |
| **dbt**          | 数据建模标准化，分析流程可追溯         | 分析型数据仓库建模（如 OceanBase） |
| **DataWorks**    | 阿里云生态全栈支持，可视化开发         | 云原生数据湖与离线分析            |
| **AWS Glue**     | 无服务器 ETL，AWS 生态无缝集成         | AWS 云数据湖构建与批处理          |
