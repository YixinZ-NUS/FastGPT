# （可选）关闭防火墙和 SELinux

本文介绍如何关闭防火墙和 SELinux。如果是个人试用场景，可以跳过此步骤。如果用于生产环境，建议您参考本文关闭防火墙和 SELinux。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>您必须在每台机器上分别执行此操作。</p>
</main>

## 关闭防火墙

依次执行以下命令，关闭防火墙：

```shell
[admin@test001 ~]$ sudo systemctl disable firewalld
[admin@test001 ~]$ sudo systemctl stop firewalld
[admin@test001 ~]$ sudo systemctl status firewalld
```

## 关闭 SELinux

执行以下命令，打开 `/etc/selinux/config` 配置文件：

```shell
[admin@test001 ~]$ sudo vim /etc/selinux/config
```

在 `/etc/selinux/config` 配置文件中修改对应配置项为以下内容：

```shell
SELINUX=disabled
```

执行以下命令或重启服务器，使更改生效：

```shell
[admin@test001 ~]$ setenforce 0
```

执行以下命令，查看更改是否生效：

```shell
sestatus
```
