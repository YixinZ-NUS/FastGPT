# 准备安装包

部署前，请确认已获取各产品使用的镜像和安装包，用于后续的安装部署。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>请在 <a href="https://www.oceanbase.com/softwarecenter-enterprise">OceanBase 软件下载中心</a> 下载 OceanBase 数据库以及配套工具、驱动和中间件的安装包。如果未找到对应版本的安装包，请联系技术支持获取。</p>
</main>

**部署 OceanBase 集群使用的镜像和安装包如下表所示：**

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>使用命令行部署 OceanBase 集群无需准备部署 OceanBase 云平台相关的镜像/安装包。</p>
</main>
