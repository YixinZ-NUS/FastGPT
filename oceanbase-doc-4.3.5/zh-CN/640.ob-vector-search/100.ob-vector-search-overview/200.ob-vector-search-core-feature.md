# 向量搜索核心功能

OceanBase 提供了存储、索引、搜索 Embedding 向量数据的能力。具体包括：

|**核心功能**|**描述**|
|---|---|
| 向量数据类型 | <ul><li>支持最大 16,000 维的 float 稠密向量数据存储。</li><li>支持稀疏向量数据存储。</li></ul> |
| 向量索引 | <ul><li>支持精确搜索和近似最近邻搜索。</li><li>支持 L2 距离、内积和余弦相似度计算。</li><li>支持 HNSW/HNSW_SQ/HNSW_BQ 索引，索引列最大维度为 4096。HNSW_BQ 索引从 V4.3.5 BP2 版本开始支持。</li><li>支持 IVF/IVF_PQ 索引，索引列最大维度为 4096。</li></ul> |
| 向量搜索 SQL 运算符 | 支持向量加、减、乘、比较、聚合等基础运算操作符。 |

## 使用限制和说明

* OceanBase 默认采用 NULL first 比较模式，所以对 NULL 值进行排序时会将其放至最前，建议查询的时候加上 `NOT NULL` 条件。
* OceanBase [社区版](../../400.deploy/500.deploy-oceanbase-database-community-edition/100.deployment-overview.md)/[桌面版](../../400.deploy/500.deploy-oceanbase-database-community-edition/800.deploy-oceanbase-desktop.md)支持在 SYS 及用户租户下使用向量检索功能；企业版仅支持用户租户下使用。
