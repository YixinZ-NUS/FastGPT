# 向量搜索实践教程概述

本文档将介绍如何基于 OceanBase 构建向量搜索的典型应用场景。

## 向量搜索实践教程

### 基于 OceanBase 构建智能问答机器人

文档智能助手是一种基于 OceanBase 数据库的应用，它将文档以向量的形式批量存储。用户可以通过 UI 界面向它提问，它会使用 BGE-M3 模型将提问内容嵌入成为向量，并在数据库中搜索相似向量。当找到相似向量对应的文档内容后，应用会将它们会同用户提问一起发送给 LLM，LLM 会根据提供的文档生成更加准确的回答。

### 基于 OceanBase 构建图搜图应用

图搜图应用是另一种基于 OceanBase 数据库的应用，它将图片库以向量形式存储在数据库内。用户可以在对应的 UI 界面上传需要查询的图片，图片会被应用转换为向量，并在数据库内查询相似向量。最终，它会以图片形式，在 UI 页面上展示相似图片。

### 基于 OceanBase 构建多模融合应用

OceanBase 的多模融合技术支持构建文旅小助手应用，该应用能够同时处理多种数据类型，包括：

* 向量数据：景点类型、饮食偏好等语义信息
* 空间数据：地理位置、行程范围等空间信息
* 关系数据：景点评分、出行季节等结构化信息

通过高效的多模数据混合搜索技术，为用户提供更加精准、个性化和实时的推荐服务体验。
