# 通过 Navicat 连接 OceanBase 数据库（Oracle 模式）

Navicat 是一个功能强大的数据库管理工具，支持连接多种数据库，包括 OceanBase 数据库的 Oracle 模式。本文介绍如何使用 Navicat 连接 OceanBase 数据库的 Oracle 模式。

<main id="notice" type="explain">
    <h4>说明</h4>
    <p>本文介绍使用 Navicat 通过 Oracle 协议连接 OceanBase 数据库的方法。</p>
</main>

## 版本兼容性

* Navicat Premium 16.1.9 或更高版本支持 OceanBase 企业版（Oracle 模式）V3 或更高版本
* Navicat For Oracle 16.1.9 或更高版本支持 OceanBase 企业版（Oracle 模式）V3 或更高版本

<main id="notice" type="notice">
    <h4>注意</h4>
    <p>Navicat 的 macOS 平台版本目前不支持 OceanBase（Oracle 模式）。</p>
</main>

## 前提条件

* 已安装 Navicat Premium 16.1.9+ 或 Navicat for Oracle 16.1.9+
* 已获取 OceanBase 数据库连接信息：
  * 数据库服务器地址和端口
  * 租户名称
  * 用户名和密码

## 下载 OceanBase JDBC 驱动

1. 访问 [OceanBase 软件下载中心](https://www.oceanbase.com/softwarecenter-enterprise)，在 **驱动和中间件** 分类下下载 OceanBase JDBC 驱动（OceanBase Connector/J）。

2. 下载完成后，将 JAR 文件保存到本地目录，例如：`/path/to/oceanbase-client.jar`

## 配置 Navicat 连接

1. 打开 Navicat，点击菜单栏的 **文件** > **新建连接** > **Oracle**。

2. 在连接设置页面，填写以下信息：

   | 参数 | 说明 | 示例值 |
   |------|------|--------|
   | 主机 | OceanBase 服务器地址 | 127.0.0.1 |
   | 端口 | 数据库端口（默认 2881） | 2881 |
   | 用户名 | 数据库用户名 | user@tenant |
   | 密码 | 数据库密码 | ******** |

3. 切换到 **高级** 标签页：
   * 在 **OCI 库** 部分，选择 **使用 OCI**
   * 在 **OCI 库路径** 中，选择 Oracle 客户端的 `oci.dll` 文件（Windows）或 `libclntsh.dylib` 文件（macOS）

4. 切换到 **驱动程序** 标签页：
   * 选择 **使用自定义 JDBC 驱动**
   * 点击 **添加** 按钮，选择下载的 OceanBase JDBC 驱动 JAR 文件
   * 在 **类名** 中输入：`com.alipay.oceanbase.jdbc.Driver`
   * 在 **URL 模板** 中输入：`jdbc:oceanbase://{host}:{port}/{database}`

5. 点击 **测试连接** 按钮，验证连接是否成功。

## 使用 Navicat 管理 OceanBase 数据库

连接成功后，您可以使用 Navicat 的图形界面执行以下操作：

* 浏览数据库对象（表、视图、存储过程等）
* 执行 SQL 查询
* 导入/导出数据
* 管理数据库用户和权限
* 监控数据库性能
