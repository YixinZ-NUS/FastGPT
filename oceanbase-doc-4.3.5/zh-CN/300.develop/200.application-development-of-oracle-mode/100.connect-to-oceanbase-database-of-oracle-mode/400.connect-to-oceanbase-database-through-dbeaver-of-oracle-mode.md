
# 通过 DBeaver 连接 OceanBase 数据库（Oracle 模式）

DBeaver 是一个通用的数据库管理工具，支持连接多种数据库，包括 OceanBase 数据库的 Oracle 模式。本文介绍如何使用 DBeaver 连接 OceanBase 数据库的 Oracle 模式。

<main id="notice" type="explain">
    <h4>说明</h4>
    <p>本文使用 Generic 模式 + OceanBase 驱动创建的数据库连接，功能上存在一些局限性，例如无法在界面上修改字段信息。</p>
</main>

## 前提条件

* 已安装 DBeaver Community 或 Enterprise 版本（建议使用最新版本）
* 已获取 OceanBase 数据库连接信息：
  * 数据库服务器地址和端口
  * 租户名称
  * 用户名和密码

## 下载 OceanBase JDBC 驱动

1. 访问 [OceanBase 软件下载中心](https://www.oceanbase.com/softwarecenter-enterprise)，在 **驱动和中间件** 分类下下载 OceanBase JDBC 驱动（OceanBase Connector/J）。

2. 下载完成后，将 JAR 文件保存到本地目录，例如：`/path/to/oceanbase-client.jar`

## 创建新驱动

1. 打开 DBeaver，点击 **数据库** > **驱动管理器**。

2. 点击 **新建** 按钮创建新驱动。

3. 在 **库** 标签页中：
   * 点击 **添加文件夹**，选择下载的 OceanBase JDBC 驱动 JAR 文件
   * 点击 **确定** 保存驱动配置

## 创建新连接

1. 点击 **数据库** > **新建数据库连接**。
2. 在数据库选择界面，选择 **OceanBase Oracle** 驱动，然后点击 **下一步**。
3. 在连接设置页面，填写以下信息：

   | 参数 | 说明 | 示例值 |
   |------|------|--------|
   | 主机 | OceanBase 服务器地址 | 127.0.0.1 |
   | 端口 | 数据库端口（默认 2881） | 2881 |
   | 用户名 | 数据库用户名 | user@tenant |
   | 密码 | 数据库密码 | ******** |

4. 点击 **测试连接** 按钮，验证连接是否成功

## 使用 DBeaver 管理 OceanBase 数据库

连接成功后，您可以使用 DBeaver 的图形界面执行以下操作：

* 浏览数据库对象（表、视图、存储过程等）
* 执行 SQL 查询
* 导入/导出数据
* 管理数据库用户和权限
* 监控数据库性能
