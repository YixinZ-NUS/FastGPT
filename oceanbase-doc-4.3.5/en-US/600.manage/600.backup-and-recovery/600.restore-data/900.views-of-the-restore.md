# Views related to physical restore

OceanBase Database allows you to query views for the restore progress and results.

## CDB_OB_RESTORE_PROGRESS

The [CDB_OB_RESTORE_PROGRESS](../../../700.reference/700.system-views/300.system-view-of-sys-tenant/200.dictionary-view-of-sys-tenant/10400.o-cdb_ob_restore_progress-of-sys-tenant.md) view displays the restore progress of tenants.

## DBA_OB_RESTORE_PROGRESS

The `DBA_OB_RESTORE_PROGRESS` view displays ongoing restore jobs.

* For more information about this view in the `sys` tenant, see [DBA_OB_RESTORE_PROGRESS](../../../700.reference/700.system-views/300.system-view-of-sys-tenant/200.dictionary-view-of-sys-tenant/21200.o-dba_ob_restore_progress-of-sys-tenant.md).

* For more information about this view in an Oracle tenant, see [DBA_OB_RESTORE_PROGRESS](../../../700.reference/700.system-views/500.system-view-of-oracle-mode/200.dictionary-view-of-oracle-mode/14900.dba_ob_restore_progress-of-oracle-mode.md).

* For more information about this view in a MySQL tenant, see [DBA_OB_RESTORE_PROGRESS](../../../700.reference/700.system-views/400.system-view-of-mysql-mode/200.dictionary-view-of-mysql-mode/10900.o-dba_ob_restore_progress-of-mysql-mode.md).

## CDB_OB_RESTORE_HISTORY

The [CDB_OB_RESTORE_HISTORY](../../../700.reference/700.system-views/300.system-view-of-sys-tenant/200.dictionary-view-of-sys-tenant/10300.o-cdb_ob_restore_history-of-sys-tenant.md) view displays the restore results of tenants.

## DBA_OB_RESTORE_HISTORY

The `DBA_OB_RESTORE_HISTORY` view displays the restore jobs that have been completed.

* For more information about this view in the `sys` tenant, see [DBA_OB_RESTORE_HISTORY](../../../700.reference/700.system-views/300.system-view-of-sys-tenant/200.dictionary-view-of-sys-tenant/21100.o-dba_ob_restore_history-of-sys-tenant.md).

* For more information about this view in an Oracle tenant, see [DBA_OB_RESTORE_HISTORY](../../../700.reference/700.system-views/500.system-view-of-oracle-mode/200.dictionary-view-of-oracle-mode/14800.dba_ob_restore_history-of-oracle-mode.md).

* For more information about this view in a MySQL tenant, see [DBA_OB_RESTORE_HISTORY](../../../700.reference/700.system-views/400.system-view-of-mysql-mode/200.dictionary-view-of-mysql-mode/10800.o-dba_ob_restore_history-of-mysql-mode.md).

## DBA_OB_TENANTS

The `DBA_OB_TENANTS` view displays the basic information of tenants. The `RESTORE_DATA_MODE` column in the view records physical restore information.

* For more information about this view in the `sys` tenant, see [DBA_OB_TENANTS](../../../700.reference/700.system-views/300.system-view-of-sys-tenant/200.dictionary-view-of-sys-tenant/23100.o-dba_ob_tenants-of-sys-tenant.md).

* For more information about this view in an Oracle tenant, see [DBA_OB_TENANTS](../../../700.reference/700.system-views/500.system-view-of-oracle-mode/200.dictionary-view-of-oracle-mode/16200.dba_ob_tenants-oracle.md).

* For more information about this view in a MySQL tenant, see [DBA_OB_TENANTS](../../../700.reference/700.system-views/400.system-view-of-mysql-mode/200.dictionary-view-of-mysql-mode/12300.o-dba_ob_tenants-of-mysql-mode.md).
