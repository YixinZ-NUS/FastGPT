# CANCEL_GATHER_STATS

`CANCEL_GATHER_STATS` is used to cancels a statistics collection command.

## Syntax

```sql
DBMS_STATS.cancel_gather_stats (
      taskid         VARCHAR2
    );
```

## Parameters

|    Parameter    |        Description         |
|----------|-----------------------|
|taskid|The ID of the statistics collection task. <main id="notice" type='explain'><h4>Notice</h4><p>You can query the <code>GV$OB_OPT_STAT_GATHER_MONITOR</code> view for the <code>taskid</code>. For more information about this view, see <a href="../../../../700.system-views/400.system-view-of-mysql-mode/300.performance-view-of-mysql-mode/6100.gv_ob_opt_stat_gather_monitor-of-mysql-mode.md">GV\$OB_OPT_STAT_GATHER_MONITOR</a>. </p></main>|

## Example

```sql
obclient> CALL DBMS_STATS.CANCEL_GATHER_STATS(taskid);
```

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>Replace <code>taskid</code> with the actual value. </p>
</main>
