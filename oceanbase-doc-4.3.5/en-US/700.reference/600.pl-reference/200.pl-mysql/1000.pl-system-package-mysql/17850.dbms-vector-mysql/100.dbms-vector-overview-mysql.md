# Overview of DBMS_VECTOR

The `DBMS_VECTOR` system package is used to manage vector indexes.

## Overview of DBMS_VECTOR procedures

The following table describes the `DBMS_VECTOR` procedures supported in the current version of OceanBase Database.

| **Procedure** | **Description** |
| --- | --- |
| [REFRESH_INDEX](200.refresh-index-mysql.md) | Refreshes a vector index. |
| [REBUILD_INDEX](300.rebuild-index-mysql.md) | Rebuilds a vector index. |
| [INDEX_VECTOR_MEMORY_ESTIMATE](400.index-vector-memory-estimate-mysql.md) | Estimates the memory usage of a vector index. |
| [INDEX_VECTOR_MEMORY_ADVISOR](500.index-vector-memory-advisor-mysql.md) | Analyzes the memory usage of a vector index. |
