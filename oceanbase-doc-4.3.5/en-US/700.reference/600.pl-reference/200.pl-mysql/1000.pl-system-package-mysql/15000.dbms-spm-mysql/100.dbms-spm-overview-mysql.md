# Overview

The `DBMS_SPM` package is used for managing SQL plans. It supports operations such as loading, modifying, and dropping plan baseline information.

## Subprogram overview

The following table lists the subprograms of the `DBMS_SPM` package supported in the current version of OceanBase Database and their descriptions.

|                   **Subprogram**                                 |                  **Description**                |
|-------------------------------------------------------------|------------------------------------------|
| [ACCEPT_SQL_PLAN_BASELINE](200.accept-sql-plan-baseline-mysql.md)  | Accepts a plan as a baseline based on the evolution results.        |
| [ALTER_SQL_PLAN_BASELINE](300.alter-sql-plan-baseline-mysql.md)              | Modifies the attributes of one or more plans in a baseline. The return value indicates the number of modified baselines.              |
| [CANCEL_EVOLVE_TASK](400.cancel-evolve-task-mysql.md)            | Cancels an ongoing evolution task.           |
| [CONFIGURE](500.configure-mysql.md)             | Sets some parameters of SPM.                         |
| [DROP_EVOLVE_TASK](600.drop-evolve-task-mysql.md)             | Drops an evolution task.                              |
| [DROP_SQL_PLAN_BASELINE](700.drop-sql-plan-baseline-mysql.md)            | Drops one or more plan baselines.    |
| [LOAD_PLANS_FROM_CURSOR_CACH](800.load-plans-from-cursor-cache-mysql.md)            | Reads one or more SQL plans from the Cursor Cache as a plan baseline.         |
