# DBMS_PARTITION overview

The `DBMS_PARTITION` system package provides interfaces for dynamic partition management.

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>For OceanBase Database V4.3.5, the <code>DBMS_PARTITION</code> package is supported starting from V4.3.5 BP2. </p>
</main>

## Privileges

Operations can be performed only in user tenants.

## Subprogram overview

The following table lists the `DBMS_PARTITION` subprograms supported by the current version of OceanBase Database and provides a brief description.

|               Subprogram               |              Description              |
|----------------------------------------|---------------------------------------|
| [MANAGE_DYNAMIC_PARTITION](200.manage_dynamic_partition-of-mysql-mode.md) | Performs a dynamic partition management task: pre-create future partitions and delete expired partitions for dynamic partition tables under the current tenant. |
