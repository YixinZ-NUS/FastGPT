# AUTO_REFRESH_EXTERNAL_TABLE

`AUTO_REFRESH_EXTERNAL_TABLE` regularly refreshes the metadata of an external table.

## Syntax

```sql
DBMS_EXTERNAL_TABLE.auto_refresh_external_table(interval int)
```

## Parameters

| **Parameter** | **Description** |
|------------------|-----------------------------------------------------|
| interval int | The interval for scheduling the refresh task, in seconds. Value range: [-1,âˆž]. <ul><li>When the value is `-1`, scheduling of the refresh task is disabled. </li><li>When the value is `0`, the metadata of the external table is immediately refreshed. </li><li>When the value is greater than 0, the metadata of the external table is refreshed based on the specified interval. </li></ul>|
