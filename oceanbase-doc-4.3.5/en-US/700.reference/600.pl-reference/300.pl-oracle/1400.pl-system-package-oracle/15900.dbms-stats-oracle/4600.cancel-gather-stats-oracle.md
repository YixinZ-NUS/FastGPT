# CANCEL_GATHER_STATS

`CANCEL_GATHER_STATS` is used to cancel a statistics collection command.

## Syntax

```sql
DBMS_STATS.cancel_gather_stats (
      taskid         VARCHAR2
    );
```

## Parameters

|    Parameter    |        Description         |
|----------|-----------------------|
|taskid|The ID of the statistics collection task. <main id="notice" type='explain'><h4>Note</h4><p>You can query the <code>GV$OB_OPT_STAT_GATHER_MONITOR</code> view for the <code>taskid</code>. For more information about this view, see <a href="../../../../700.system-views/500.system-view-of-oracle-mode/300.performance-view-of-oracle-mode/2000.gv-ob_opt_stat_gather_monitor-of-oracle-mode.md">GV\$OB_OPT_STAT_GATHER_MONITOR</a>. </p></main>|

## Example

```sql
obclient> CALL DBMS_STATS.CANCEL_GATHER_STATS(taskid);
```

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>Replace <code>taskid</code> with the actual ID. </p>
</main>
