# GET_PREFS

The `GET_PREFS` function obtains the default value of the specified preference.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This topic applies only to OceanBase Database Enterprise Edition. OceanBase Database Community Edition provides only the MySQL mode. </p>
  </main>

## Syntax

```sql
DBMS_STATS.GET_PREFS (
   pname     IN   VARCHAR2,
   ownname   IN   VARCHAR2 DEFAULT NULL,
   tabname   IN   VARCHAR2 DEFAULT NULL)
 RETURN VARCHAR2;
```

## Parameters

| Parameter           | Description                                                |
|---------|-----------------------------------|
| pname   | The name of the preference.                      |
| ownname       | The username. If the parameter is set to `NULL`, the current login username is used by default. |
| tabname | The name of the table.                              |


## Exceptions

|    Error code    |      Description      |
|-----------|----------------------|
| OBE-20000 | The resource manager is not started and statistics cannot be collected. |
| OBE-20001 | The input value is invalid.              |


## Examples

Obtain the default value of the `GRANULARITY` preference for the `tbl1` table of the `testUser01` user.

```sql
obclient> SELECT DBMS_STATS.GET_PREFS ('GRANULARITY','testUser01','tbl1') FROM DUAL;
+----------------------------------------------------------+
| DBMS_STATS.GET_PREFS ('GRANULARITY','testUser01','tbl1') |
+----------------------------------------------------------+
| AUTO                                                     |
+----------------------------------------------------------+
1 row in set
```
