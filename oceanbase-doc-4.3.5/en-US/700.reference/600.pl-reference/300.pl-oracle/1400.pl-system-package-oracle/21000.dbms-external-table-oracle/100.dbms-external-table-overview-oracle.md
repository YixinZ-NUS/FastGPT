# Overview

The `DBMS_EXTERNAL_TABLE` package provides a programming interface for managing and operating external tables.

## Subprograms

The following table describes the `DBMS_EXTERNAL_TABLE` subprograms supported by the current OceanBase Database version.

|                                Subprogram                                |        Description        |
| ----------------------- | --------------------- |
| [auto_refresh_external_table](../21000.dbms-external-table-oracle/200.auto-refresh-external-table-oracle.md) | Regularly refreshes the metadata of an external table. |
