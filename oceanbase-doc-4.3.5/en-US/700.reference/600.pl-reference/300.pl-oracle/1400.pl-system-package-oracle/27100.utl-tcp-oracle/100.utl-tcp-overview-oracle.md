# Overview of UTL_TCP

`UTL_TCP` is a system package used for communication with external servers via the TCP/IP protocol.

<main id="notice">
    <h4>Applicability</h4>
    <p>This content is only applicable to the Oracle-compatible mode of OceanBase Database.</p>
</main>

## Subprograms of UTL_TCP

The following table lists the subprograms supported by the `UTL_TCP` package in the current version of OceanBase Database, along with a brief description of each subprogram.

|                       **Subprogram**             |                **Description**                       |
|----------------------------------------------|-----------------------------------------------|
|[OPEN_CONNECTION](200.open-connection-oracle.md)|Opens a TCP/IP connection with the specified service.|
|[CLOSE_CONNECTION](300.utl-tcp-close-connection-oracle.md)|Closes an opened TCP/IP connection.|
|[CLOSE_ALL_CONNECTIONS](400.close-all-connections-oracle.md)|Closes all opened TCP/IP connections.|
|[WRITE_LINE](500.write-line-oracle.md)|Transmits a text line to the service with the opened connection.|
|[WRITE_TEXT](600.write-text-oracle.md)|Transmits a text message to the service with the opened connection.|
|[WRITE_RAW](700.write-raw-oracle.md)|Transmits binary data to the service with the opened connection.|
|[GET_LINE](800.get-line-of-oracle.md)|Returns the read text line data. The return continues until the end of the line or the end of input is reached.|
|[GET_TEXT](900.get-text-oracle.md)|Returns the read text data. |
