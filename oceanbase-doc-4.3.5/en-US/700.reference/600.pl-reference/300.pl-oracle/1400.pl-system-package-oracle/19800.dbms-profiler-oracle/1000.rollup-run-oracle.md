# ROLLUP_RUN

The `ROLLUP_RUN` procedure calculates the execution duration of each unit during an execution.

 <main id="notice" >
    <h4>Applicability</h4>
    <p>This topic applies only to OceanBase Database Enterprise Edition. OceanBase Database Community Edition provides only the MySQL mode. </p>
  </main>

## Syntax

```sql
PROCEDURE ROLLUP_RUN(run_number IN number);
```

## Parameters

|       Parameter       |              Description               |
|----------------|---------------------------------|
| run_number | The run number, which helps you store and call run data later. The `IN` value behind <code>run_number</code> is sourced from the <code>run_number OUT</code> parameter of <code>start_profiler</code>. |

## Examples

```shell
obclient> call dbms_profiler.rollup_run(1);
Query OK, 0 rows affected (0.06 sec)
```
