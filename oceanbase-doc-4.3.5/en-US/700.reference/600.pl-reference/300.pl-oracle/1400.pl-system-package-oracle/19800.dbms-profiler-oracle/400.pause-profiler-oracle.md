# PAUSE_PROFILER

`PAUSE_PROFILER` pauses the profiler.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This topic applies only to OceanBase Database Enterprise Edition. OceanBase Database Community Edition provides only the MySQL mode. </p>
  </main>

## Syntax

```sql
DBMS_PROFILER.PAUSE_PROFILER
  RETURN BINARY_INTEGER;

DBMS_PROFILER.PAUSE_PROFILER;
```

 <main id="notice" type='explain'>
    <h4>Note</h4>
    <p><code>RETURN BINARY_INTEGER</code> specifies to return a binary integer. In actual scenarios, you do not need to declare <code>RETURN BINARY_INTEGER</code> separately but can directly call the <code>PAUSE_PROFILER</code> procedure. </p>
  </main>

## Examples

Run the following command to pause the profiler:

```shell
obclient> call DBMS_PROFILER.pause_profiler();
Query OK, 0 rows affected (0.012 sec)
```
