# RCPT

`RCPT` is used to execute the `rcpt` command to specify the email recipient.

<main id="notice" >
    <h4>Applicability</h4>
    <p>This content applies only to the Oracle-compatible mode of OceanBase Database. </p>
</main>

## Syntax

```sql
UTL_SMTP.RCPT (
   c           IN OUT NOCOPY     connection,
   recipient   IN                VARCHAR2,
   parameters  IN                VARCHAR2 DEFAULT NULL)
RETURN reply;

UTL_SMTP.RCPT (
   c           IN OUT NOCOPY     connection,
   recipient   IN                VARCHAR2,
   parameters  IN                VARCHAR2 DEFAULT NULL);
```

## Parameters

| **Parameter** |         **Description**         |
|--------|------------------------|
|c|The SMTP connection. |
|recipient|The recipient's email address. |
|parameters|The format of the SMTP protocol as defined in RFC1869. It must follow the format "XXX=XXX (XXX=XXX ....)".|

## Return value

| **Return value** |          **Description**          |
|---------|--------------------------|
|reply|The reply to the command. If multiple replies are returned, the last reply is returned. |

## Considerations

* Each call to `RCPT` specifies a single recipient email address. Multiple recipients can be specified by calling `RCPT` multiple times.
* Must be called after the `mail` command.
* The expected response is a message starting with `250` or `251`, indicating that the SMTP server has successfully responded and accepted the specified recipient address.
