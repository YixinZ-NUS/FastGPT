# Create a tenant

This topic describes how to create a tenant by using obshell-sdk-go.

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>We recommend that you first read <a href='../100.quickstart-of-go.md'>Get started with obshell-sdk-go</a> to learn how to use obshell-sdk-go. </p>
</main>

## Considerations

* Make sure that obshell is running.

* Make sure that the tenant exists and is in the NORMAL state.

* The node corresponding to the client for the request must be in an initialized cluster.

## Sample code

:::tab

tab Synchronous execution of tasks

```go
package main

import (
  "fmt"

  "github.com/oceanbase/obshell-sdk-go/services"
  "github.com/oceanbase/obshell-sdk-go/services/v1"
)

func main() {
  // Create a client instance, whose node address is '10.10.10.1' and port is 2886.
  client, err := services.NewClientWithPassword("10.10.10.1", 2886, "****")
  if err != nil {
    fmt.Print(err)
    return
  }

  req := client.V1().NewCreateTenantRequest("t1", []v1.ZoneParam{
    {
      Name:           "zone1",
      ReplicaType:    "FULL",
      UnitConfigName: "s1",
      UnitNum:        1,
    },
  })

  if _, err = client.V1().CreateTenantSyncWithRequest(req); err != nil {
    fmt.Print(err)
    return
  }
}
```

tab Asynchronous execution of tasks

```go
package main

import (
  "fmt"

  "github.com/oceanbase/obshell-sdk-go/services"
  "github.com/oceanbase/obshell-sdk-go/services/v1"
)

func main() {
  // Create a client instance, whose node address is '10.10.10.1' and port is 2886.
  client, err := services.NewClientWithPassword("10.10.10.1", 2886, "****")
  if err != nil {
    fmt.Print(err)
    return
  }

  req := client.V1().NewCreateTenantRequest("t1", []v1.ZoneParam{
    {
      Name:           "zone1",
      ReplicaType:    "FULL",
      UnitConfigName: "s1",
      UnitNum:        1,
    },
  })

  dag, err := client.V1().CreateTenantWithRequest(req)
  if err != nil {
    fmt.Print(err)
    return
  }

  // Wait for the task to succeed.
  if _, err = client.V1().WaitDagSucceed(dag.GenericID); err != nil {
    fmt.Print(err)
    return
  }
}
```
