# Modify tenant configurations

This topic describes how to use the API to set configurations for a specified tenant. You can set multiple configurations for a tenant at a time.

## Request parameters

| Parameter | Type | Required | Example | Description |
|------|------|-------|------|------|
| parameters | map[string]interface{} | Yes | {<br>"backup_data_file_size": "2G",<br>"arbitration_timeout": "10s"<br>} | The configurations of the tenant. |

### Request examples

PUT /api/v1/tenant/t1/parameters
