# SHRINK

## Purpose

You can use this statement to rewrite the data of a table.

## Syntax

```sql
ALTER TABLE table_name SHRINK SPACE
```

## Parameters

| Parameter | Description |
|------------|-----|
| table_name | The name of the table.  |

## Examples

1. Create the `employee` table and insert data.

    ```sql
    obclient> CREATE TABLE employee (id NUMBER, name VARCHAR2(50));
    obclient> INSERT INTO employee VALUES (1, 'John');
    obclient> INSERT INTO employee VALUES (2, 'Mary');
    ```

2. Rewrite (shrink) the data in the `employee` table.

    ```sql
    obclient> ALTER TABLE employee SHRINK SPACE;
    ```

3. Verify the result of the space shrink operation.

    ```sql
    obclient> SELECT segment_name, bytes/1024/1024 MB
    FROM user_segments
    WHERE segment_name = 'EMPLOYEE';
    ```
