# CANCEL JOB

You can use this statement to cancel an asynchronous job.

<main id="notice" type='notice'>
  <h4>Notice</h4>
  <p><ul><li>The <code>job_id</code> parameter is required. </li><li>This statement stops a running job or cancels a job that has not been started. <li>If you execute this statement on a completed job, <code> ERROR 4018 (HY000): Entry not exist</code> is returned. <li>You can still query the status of a canceled job. </li></ul></p>
</main>

## Syntax

```sql
CANCEL JOB 'job_id';
```

## Examples

```shell
obclient> CANCEL JOB '2010011125899906847397';
3 rows in set (0.00 sec)
```
