# DECODE_TRACE_ID

## Syntax

```sql
DECODE_TRACE_ID()
```

## Purpose

`DECODE_TRACE_ID()` parses a `trace_id` value and returns the IP address and port number of the server where the SQL request is initiated. You can query the [GV$OB_SQL_AUDIT](../../../../../../../700.reference/700.system-views/400.system-view-of-mysql-mode/300.performance-view-of-mysql-mode/3500.gv-ob_sql_audit-of-mysql-mode.md) view for related information.

## Examples

```sql
obclient [(none)]> SELECT DECODE_TRACE_ID(LAST_TRACE_ID());
```

The return result is as follows:

```shell
+----------------------------------+
| DECODE_TRACE_ID(LAST_TRACE_ID()) |
+----------------------------------+
| xxx.xx.xxx.xx1:2882              |
+----------------------------------+
1 row in set
```
