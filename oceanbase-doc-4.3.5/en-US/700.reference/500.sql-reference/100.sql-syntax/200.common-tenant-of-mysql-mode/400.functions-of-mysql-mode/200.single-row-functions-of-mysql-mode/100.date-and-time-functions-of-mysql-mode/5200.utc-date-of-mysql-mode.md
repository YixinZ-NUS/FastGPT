# UTC_DATE

## Syntax

```sql
UTC_DATE()
```

## Purpose

`UTC_DATE()` returns the current UTC date in the `%Y-%m-%d` format.

## Examples

```sql
obclient> SELECT UTC_DATE(), UTC_DATE() + 0, UTC_DATE() + 6;
+------------+----------------+----------------+
| UTC_DATE() | UTC_DATE() + 0 | UTC_DATE() + 6 |
+------------+----------------+----------------+
| 2021-07-14 |       20210714 |       20210720 |
+------------+----------------+----------------+
1 row in set
```
