# Overview of map functions

Map functions are a set of specialized tools provided by OceanBase Database for working with map data. These functions enable users to perform complex queries and analyses on map data efficiently.

Below is an overview of the map functions available in OceanBase Database and their purposes:

## Types of map functions

| Type                              | Description                                                                 |
|--------------------------------------------------|-----------------------------------------------------------------------------|
| [Map constructor functions](200.map-constructor-functions-of-mysql-mode.md) | Create new map data types.                                |
| [Map decision functions](300.map-decision-functions-of-mysql-mode.md)      | Evaluate map data and return Boolean results.                |
| [Map manipulation functions](600.map-manipulation-functions-of-mysql-mode.md) | Perform basic operations on map data, such as extracting keys or values, and returning the modified map data. |

## Overview of map functions

The following table lists all the map functions currently supported by the OceanBase database:

| Function type                                   | Function name   | Description                                           |
|---------|---------|------|
| [Map constructor functions](200.map-constructor-functions-of-mysql-mode.md) | `map`           | Create a map data type.                      |
| [Operators](200.map-constructor-functions-of-mysql-mode.md)                | `{}`            | Construct a map data type.                   |
| [Operators](300.map-decision-functions-of-mysql-mode.md)                  | `=` / `!=`      | Check whether two map data entries are equal. |
| [Map manipulation functions](600.map-manipulation-functions-of-mysql-mode.md) | `map_keys`      | Retrieve the list of keys from a map.               |
| [Map manipulation functions](600.map-manipulation-functions-of-mysql-mode.md) | `map_values`    | Retrieve the list of values from a map.             |
