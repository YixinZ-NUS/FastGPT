# PASSWORD

## Syntax

```sql
PASSWORD(str)
```

## Purpose

`PASSWORD` encrypts a string. It receives a plaintext string as the argument and returns an encrypted string.

## Examples

```sql
obclient [test]> SELECT PASSWORD('***1***');
```

The return result is as follows:

```shell
+--------------------------------------------+
| PASSWORD('***1***')                        |
+--------------------------------------------+
| *BA10AFC5C8D9F9D1A5704E9C274C1A63EE9CB94B  |
+--------------------------------------------+
1 row in set, 1 warning
```
