# CLEAR RESTORE SOURCE

## Purpose

You can use this statement to clear the path loaded by using the `ADD RESTORE SOURCE` statement.

## Required privileges

To execute this statement, you must have the `ALTER SYSTEM` privilege.

## Syntax

```sql
ALTER SYSTEM CLEAR RESTORE SOURCE;
```
