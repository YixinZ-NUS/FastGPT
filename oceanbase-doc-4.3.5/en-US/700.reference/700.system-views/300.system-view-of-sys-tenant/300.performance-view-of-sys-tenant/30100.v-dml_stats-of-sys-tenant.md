# V$DML_STATS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>

## Purpose

This view displays the DML information of user tables of all tenants on the current OBServer node.

## Columns

| Column | Type | Nullable? | Description |
| --- | --- | --- | --- |
| SVR_IP | varchar(46) | NO | The IP address of the server. |
| SVR_PORT | bigint(20) | NO | The port number of the server. |
| INST_ID | bigint(20) | NO | The tenant ID. |
| OBJN | bigint(20) | NO | The object ID. |
| INS | bigint(20) | NO | The approximate number of insertions since the statistics were last collected. |
| UPD | bigint(20) | NO | The approximate number of updates since the statistics were last collected. |
| DEL | bigint(20) | NO | The approximate number of deletions since the statistics were last collected. |
| DROPSEG | bigint(0) | NO | Not supported. The default value is NULL. |
| CURROWS | bigint(0) | NO | Not supported. The default value is NULL. |
| PAROBJN | bigint(20) | NO | Not supported. The default value is NULL. |
| LASTUSED | bigint(0) | NO | Not supported. The default value is NULL. |
| FLAGS | bigint(0) | NO | Not supported. The default value is NULL. |
| CON_ID | bigint(0) | NO | Not supported. The default value is NULL. |

## Sample query

View the DML information of the user table in the current OBServer node for the tenant with the tenant ID of 1004.

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.V$DML_STATS WHERE INST_ID=1004;
```
