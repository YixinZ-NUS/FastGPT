# GV$DML_STATS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>

## Purpose

This view displays the DML information of user tables in all OBServer nodes for each tenant.

## Columns

| Column | Type | Nullable? | Description |
| --- | --- | --- | --- |
| SVR_IP | varchar(46) | NO | The IP address of the server. |
| SVR_PORT | bigint(20) | NO | The port number of the server. |
| INST_ID | bigint(20) | NO | The tenant ID. |
| OBJN | bigint(20) | NO | The object ID. |
| INS | bigint(20) | NO | The approximate number of insert operations since the statistics were last collected. |
| UPD | bigint(20) | NO | The approximate number of update operations since the statistics were last collected. |
| DEL | bigint(20) | NO | The approximate number of delete operations since the statistics were last collected. |
| DROPSEG | bigint(0) | NO | This field is currently not supported and its default value is NULL. |
| CURROWS | bigint(0) | NO | This field is currently not supported and its default value is NULL. |
| PAROBJN | bigint(20) | NO | The value of the TABLET_ID field. Note that the semantic meaning (parent object ID) of this field is not supported, but technically, it will return the value of TABLET_ID. |
| LASTUSED | bigint(0) | NO | This field is currently not supported and its default value is NULL. |
| FLAGS | bigint(0) | NO | This field is currently not supported and its default value is NULL. |
| CON_ID | bigint(0) | NO | This field is currently not supported and its default value is NULL. |

## Sample query

View the DML information of user tables for the tenant with tenant ID 1004 on all OBServer nodes.

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.GV$DML_STATS WHERE INST_ID=1004;
```
