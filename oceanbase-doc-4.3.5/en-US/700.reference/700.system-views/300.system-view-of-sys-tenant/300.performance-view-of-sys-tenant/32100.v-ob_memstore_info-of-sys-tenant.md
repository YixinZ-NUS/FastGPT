# V$OB_MEMSTORE_INFO

<main id="notice" type='explain'>
<h4>Note</h4>
<p>The view <code>V$MEMSTORE_INFO</code> was renamed to <code>V$OB_MEMSTORE_INFO</code> in V4.0.0. You must use the new view name <code>V$OB_MEMSTORE_INFO</code> to query data from V4.0.0 and later. Otherwise, an error will be returned. </p>
</main>

## Purpose

This view displays the details of MemTables in all partitions of all tenants on the server.

## Columns

| **Column** | **Type** | **Required** | **Description** |
| --- | --- | --- | --- |
| SVR_IP | varchar(46) | No | The IP address of the server. |
| SVR_PORT | bigint(20) | No | The port of the server. |
| TENANT_ID | bigint(20) | No | The ID of the tenant. |
| LS_ID | bigint(20) | No | The ID of the log stream. |
| TABLE_ID | bigint(20) | No | The ID of the data shard. |
| IS_ACTIVE | varchar(3) | No | Whether the MemTable is active. |
| START_SCN | bigint(20) unsigned | No | The starting SCN of the MemTable. |
| END_SCN | bigint(20) unsigned | No | The ending SCN of the MemTable. |
| FREEZE_TS | bigint(20) | No | The time when the MemTable is frozen. |

## Sample query

The following example queries the details of MemTables in all partitions of the tenant with ID `1002` on the server.

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.V$OB_MEMSTORE_INFO WHERE TENANT_ID=1002;
```
