# GV$OB_MEMSTORE_INFO

<main id="notice" type='explain'>
<h4>Note</h4>
<p>The <code>GV$MEMSTORE_INFO</code> view has been renamed as <code>GV$OB_MEMSTORE_INFO</code> since OceanBase Database V4.0.0. If you query the <code>GV$MEMSTORE_INFO</code> view in OceanBase Database V4.0.0 or later, an error is returned. </p>
</main>

## Purpose

The `GV$OB_MEMSTORE_INFO` view displays the details about MemTables of all partitions of all tenants on all OBServer nodes.

## Columns

| Column | Type | Nullable? | Description |
| --- | --- | --- | --- |
| SVR_IP | varchar(46) | NO | The IP address of the OBServer node. |
| SVR_PORT | bigint(20) | NO | The port number of the OBServer node. |
| TENANT_ID | bigint(20) | NO | The ID of the tenant. |
| LS_ID | bigint(20) | NO | The ID of the log stream. |
| TABLE_ID | bigint(20) | NO | The ID of the tablet. |
| IS_ACTIVE | varchar(3) | NO | Indicates whether the current MemTable is active. |
| START_SCN | bigint(20) unsigned | NO | The system change number (SCN) at which the storage started. |
| END_SCN | bigint(20) unsigned | NO | The SCN at which the storage ended. |
| FREEZE_TS | bigint(20) | NO | The freeze timestamp. |

## Sample query

Query details about MemTables of all partitions of a tenant with the ID of `1002` on all OBServer nodes.

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.GV$OB_MEMSTORE_INFO WHERE TENANT_ID=1002;
```
