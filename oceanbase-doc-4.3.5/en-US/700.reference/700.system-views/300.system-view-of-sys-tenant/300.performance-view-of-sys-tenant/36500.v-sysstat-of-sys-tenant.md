# V$SYSSTAT

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is introduced since OceanBase Database V1.4. </p>
</main>

## Purpose

This view displays the statistics of each tenant on the current OBServer node.

where:

*`STATISTIC#`: the statistic number. This number may vary across versions of the OBServer node.

* `STAT_ID`: the ID of the statistic. The ID of each statistic is the same across OBServer node versions.

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>To query statistics from this view, you must enable monitoring by setting the <code>enable_perf_event</code> parameter to True. </p>
</main>

## Columns

| **Column** | **Type** | **Nullable?** | **Description** |
| --- | --- | --- | --- |
| CON_ID | bigint(20) | NO | Tenant ID |
| SVR_IP | varchar(46) | NO | IP address of the server where the information is located |
| SVR_PORT | bigint(20) | NO | Port number of the server where the information is located |
| STATISTIC# | bigint(20) | NO | Index of the statistics event |
| NAME | varchar(64) | NO | Name of the statistics event |
| CLASS | bigint(20) | NO | Type of the statistics event |
| VALUE | bigint(20) | NO | Result value of the statistics event |
| VALUE_TYPE | varchar(16) | NO | Statistics value type:<li>ADD: cumulative<li>SET: updated |
| STAT_ID | bigint(20) | NO | Statistics event ID |

## Sample query

In the sys tenant, view the statistics events of all tenants on the current node and display the first 10 records.

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.V$SYSSTAT LIMIT 10;
```
