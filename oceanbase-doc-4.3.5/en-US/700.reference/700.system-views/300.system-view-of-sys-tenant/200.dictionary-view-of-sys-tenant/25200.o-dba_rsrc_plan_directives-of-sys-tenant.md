# oceanbase.DBA_RSRC_PLAN_DIRECTIVES

<main id="notice" type='explain'>
<h4>Note</h4>
<ul><li>This view was introduced in OceanBase Database V3.1.2. </li>
<li>In the current version, we recommend that you use the <a href="21420.o-dba_ob_rsrc_directives-of-sys-tenant.md">DBA_OB_RSRC_DIRECTIVES</a> view to view the resource configuration parameters of a tenant in the Resource Manager. </li></ul>
</main>

## Purpose

This view displays the details of each resource plan directive in the database.
