# oceanbase.DBA_OB_LS_HISTORY

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is introduced in V4.2.0. </p>
</main>

## Purpose

This view displays the log streams that existed in the tenant.

## Columns

| Column | Type | Nullable? | Description |
|-------------|---------------|-------------|-------------------------------------------|
| LS_ID       | bigint(20)    | NO          |  The ID of the log stream.                                 |
| LS_GROUP_ID | bigint(20)    | NO          |  The ID of the log stream group.                               |
| STATUS      | varchar(100)  | NO          |  The current status of the log stream.  <ul><li>CREATING: The log stream is being created. </li><li>NORMAL: The normal status of a log stream. </li><li>DROPPING: The log stream is being dropped. </li><li>PRE_TENANT_DROPPING: A special status of a system log stream that indicates the deletion of all log streams in the tenant is started. </li><li>TENANT_DROPPING: The tenant is being deleted. The log streams are being deleted. </li><li>DROPPED: The log stream has been recycled or is being recycled. </li><li>WAIT_OFFLINE: The log stream is waiting for recycling. </li><li>CREATE_ABORT: The creation of the log stream is stopped. </li></ul>        |
| FLAG        | varchar(2048) | NO          |  The log stream flag.  <ul><li>Empty: A normal log stream. </li><li> DUPLICATE: A broadcast log stream. A replica table is created on this log stream. </li> <li>BLOCK_TABLET_IN: A log stream on which no tablet can be created. </li> <li>DUPLICATE\|BLOCK_TABLET_IN: A broadcast log stream on which no tablet can be created. </li></ul>                        |
| CREATE_SCN  | decimal(20,0) | NO          |  The GTS when the log stream was created. The unit is nanoseconds.              |

## Sample query

The following example shows how to query the log streams that existed in the current tenant in the sys tenant.

```sshell
obclient [oceanbase]> SELECT * FROM oceanbase.DBA_OB_LS_HISTORY;
```

The query result is as follows:

```shell
+-------+-------------+--------+------+------------+
| LS_ID | LS_GROUP_ID | STATUS | FLAG | CREATE_SCN |
+-------+-------------+--------+------+------------+
|     1 |           0 | NORMAL |      |          0 |
+-------+-------------+--------+------+------------+
1 row in set
```
