# oceanbase.CDB_TAB_COLS

## Purpose

This view displays the column information of table and view objects in all tenants.

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>
