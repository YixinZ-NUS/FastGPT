# oceanbase.DBA_OB_LS_REPLICA_TASK_HISTORY

<main id="notice" type='explain'>
  <h4>Note</h4>
  <ul><li>For OceanBase Database V4.3.x, this view was introduced in OceanBase Database V4.3.3. </li><li>For OceanBase Database V4.2.x, this view was introduced in OceanBase Database V4.2.3. </li></ul>
</main>

## Purpose

The `oceanbase.DBA_OB_LS_REPLICA_TASK_HISTORY` view displays the execution history of disaster recovery tasks.

## Columns

| Column | Type | Nullable? | Description |
| --- | --- | --- | --- |
| LS_ID | bigint(20) | NO | The ID of the log stream. |
| TASK_TYPE | varchar(64) | NO | The type of the task. Valid values: <ul><li>`MIGRATE REPLICA`: indicates a replica migration task.</li><li>`ADD REPLICA`: indicates a replica adding task.</li><li>`TYPE TRANSFORM`: indicates a replica type conversion task.</li><li>`REMOVE PAXOS REPLICA`: indicates a task that deletes a Paxos replica.</li><li>`REMOVE NON PAXOS REPLICA`: indicates a task that deletes a non-Paxos replica.</li><li>`MODIFY PAXOS REPLICA NUMBER`: indicates a task that modifies the quorum of the Paxos group.</li></ul> |
| TASK_ID | varchar(200) | NO | The ID of the task. |
| TASK_STATUS | varchar(2048) | YES | The status of the task. Valid values: <ul><li>`INPROGRESS`: The task is running or being scheduled.</li><li>`COMPLETED`: The task is completed.</li><li>`CANCELED`: The task is canceled.</li></ul> |
| PRIORITY | varchar(5) | NO | The priority of the task. Valid values: <ul><li>`LOW`: indicates a low-priority task.</li><li>`HIGH`: indicates a high-priority task.</li></ul> |
| TARGET_REPLICA_SVR_IP | varchar(46) | YES | The IP address of the OBServer node where the destination replica resides. |
| TARGET_REPLICA_SVR_PORT | bigint(20) | YES | The port number of the OBServer node where the destination replica resides. |
| TARGET_PAXOS_REPLICA_NUMBER | bigint(20) | YES | The quorum of the destination Paxos group of the log stream. |
| TARGET_REPLICA_TYPE | varchar(16) | YES | The type of the destination replica. |
| SOURCE_REPLICA_SVR_IP | varchar(46) | YES | The IP address of the OBServer node where the source replica resides. |
| SOURCE_REPLICA_SVR_PORT | bigint(20) | YES | The port number of the OBServer node where the source replica resides. |
| SOURCE_PAXOS_REPLICA_NUMBER | bigint(20) | YES | The quorum of the source Paxos group of the log stream. |
| SOURCE_REPLICA_TYPE | varchar(16) | YES | The type of the source replica. |
| DATA_SOURCE_SVR_IP | varchar(46) | YES | The IP address of the data source. |
| DATA_SOURCE_SVR_PORT | bigint(20) | YES | The port number of the data source. |
| IS_MANUAL | varchar(6) | NO | The source of the task. Value values: <ul><li>`TRUE`: The task is manually triggered by an O&M command.</li><li>`FALSE`: The task is automatically generated by the RootService layer during disaster recovery.</li></ul> |
| TASK_EXEC_SVR_IP | varchar(46) | YES | The IP address of the OBServer node where the task is executed. |
| TASK_EXEC_SVR_PORT | bigint(20) | YES | The port number of the OBServer node where the task is executed. |
| CREATE_TIME | datetime | NO | The time when the task was generated. |
| START_TIME | datetime | NO | The time when the task was scheduled. |
| MODIFY_TIME | datetime | NO | The time when the task status was updated. |
| FINISH_TIME | datetime | NO | The end time of the task. |
| EXECUTE_RESULT | varchar(2048) | YES | The task execution result. |
| COMMENT | varchar(2048)  | YES | The reason why the task was generated. |
