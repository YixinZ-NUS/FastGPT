# information_schema.QUERY_RESPONSE_TIME

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>

## Purpose

This view displays data types like RESPONSE TIME histograms for Prometheus monitoring.

## Columns

| Column | Type | Nullable? | Description |
| --- | --- | --- | --- |
| RESPONSE_TIME | bigint(14) | NO | The response time. |
| COUNT | bigint(14) | NO | The number of requests within the response time. |
| TOTAL | bigint(14) | NO | The total response time of requests. |

## Sample query

Query the RESPONSE TIME histogram under the current tenant.

```shell
obclient[test]> SELECT * FROM information_schema.information_schema.QUERY_RESPONSE_TIME;
```
