# oceanbase.CDB_OB_SERVICES

<main id="notice" type='explain'>
  <h4>Note</h4>
  <ul><li>For OceanBase Database V4.3.x, this view was introduced in OceanBase Database V4.3.3. </li><li>For OceanBase Database V4.2.x, this view was introduced in OceanBase Database V4.2.4. </li></ul>
</main>

## Purpose

The `oceanbase.CDB_OB_SERVICES` view displays all service names in all tenants.

## Columns

| Column | Type | Nullable? | Description |
|---------------------------|---------------|----------------|--------|
| TENANT_ID | bigint(20) | NO | The ID of the tenant, which can be a user tenant. |     |
| CREATE_TIME     | timestamp(6) | NO   | The time when the service was created.     |
| MODIFIED_TIME   | timestamp(6) | NO   | The time when the service information was last modified.     |
| SERVICE_NAME_ID | bigint(20)   | NO   | The ID of the service, which uniquely identifies the name of the service.     |
| SERVICE_NAME    | varchar(64)  | NO   | The name of the service.     |
| SERVICE_STATUS  | varchar(64)  | NO   | The status of the service. Valid values: <ul><li>`STARTED`: The service has been started. </li><li>`STOPPING`: The service is being stopped. </li><li>`STOPPED`: The service has been stopped. </li></ul> |

## Sample query

Query all service names in all tenants.

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.CDB_OB_SERVICES;
```

The query result is as follows:

```shell
+-----------+----------------------------+----------------------------+-----------------+------------------+----------------+
| TENANT_ID | CREATE_TIME                | MODIFIED_TIME              | SERVICE_NAME_ID | SERVICE_NAME     | SERVICE_STATUS |
+-----------+----------------------------+----------------------------+-----------------+------------------+----------------+
|      1002 | 2024-06-18 14:18:36.405999 | 2024-06-18 14:18:36.405999 |               3 | service_mysql001 | STARTED        |
+-----------+----------------------------+----------------------------+-----------------+------------------+----------------+
1 row in set (0.012 sec)
```
