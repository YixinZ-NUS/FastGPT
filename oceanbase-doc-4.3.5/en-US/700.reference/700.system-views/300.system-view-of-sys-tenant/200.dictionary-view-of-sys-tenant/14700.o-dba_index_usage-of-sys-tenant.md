# DBA_INDEX_USAGE

<main id="notice" type='explain'>
<h4>Note</h4>
<ul><li>This parameter was introduced in V4.3.0 for V4.3.x. </li> <li>This parameter was introduced in V4.2.4 for V4.2.x. </li></ul>
</main>

## Purpose

The `DBA_INDEX_USAGE` view displays index usage data.

## Columns

| **Column** | **Type** | **Nullable?** | **Description** |
| --- | --- | --- | --- |
| OBJECT_ID | bigint(20) | NO | The ID of the index table. |
| NAME | varchar(128) | NO | The name of the index table. |
| OWNER | varchar(128) | NO | The name of the database. |
| TOTAL_ACCESS_COUNT | bigint(20) | NO | The total number of accesses. |
| TOTAL_EXEC_COUNT | bigint(20) | NO | The total number of executions. |
| TOTAL_ROWS_RETURNED | bigint(20) | NO | The total number of rows returned by the index table. |
| BUCKET_0_ACCESS_COUNT | bigint(20) | NO | The number of times the index table was used 0 times. |
| BUCKET_1_ACCESS_COUNT | bigint(20) | NO | The number of times the index table was used 1 time. |
| BUCKET_2_10_ACCESS_COUNT | bigint(20) | NO | The number of times the index table was used 2 to 10 times. |
| BUCKET_2_10_ROWS_RETURNED | bigint(20) | NO | The number of rows returned by the index table when it was used 2 to 10 times. |
| BUCKET_11_100_ACCESS_COUNT | bigint(20) | NO | The number of times the index table was accessed 11 to 100 times. |
| BUCKET_11_100_ROWS_RETURNED | bigint(20) | NO | The number of rows returned by the index table when it was accessed 11 to 100 times. |
| BUCKET_101_1000_ACCESS_COUNT | bigint(20) | NO | The number of times the index table was accessed 101 to 1000 times. |
| BUCKET_101_1000_ROWS_RETURNED | bigint(20) | NO | The number of rows returned by the index table when it was accessed 101 to 1000 times. |
| BUCKET_1000_PLUS_ACCESS_COUNT | bigint(20) | NO | The number of times the index table was accessed more than 1000 times. |
| BUCKET_1000_PLUS_ROWS_RETURNED | bigint(20) | NO | The number of rows returned by the index table when it was accessed more than 1000 times. |
| LAST_USED | varchar(128) | NO | The last time the index table was used. |

## Sample query

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.DBA_INDEX_USAGE;
```
