# oceanbase.DBA_TAB_PARTITIONS

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>

## Purpose

This view displays the information about the partitions, partitioning parameters, and partition statistics that the current user has access to.
