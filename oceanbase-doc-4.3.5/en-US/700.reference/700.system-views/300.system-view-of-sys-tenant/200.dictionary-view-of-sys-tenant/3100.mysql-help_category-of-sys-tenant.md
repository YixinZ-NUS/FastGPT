# mysql.help_category

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is introduced in OceanBase Database V1.4. </p>
</main>

## Purpose

This view is used to generate the HELP command in combination with the following three tables: `mysql.help_relation`, `mysql.help_topic`, and `mysql.help_keyword`.

## Columns

| **Column** | **Type** | **Nullable?** | **Description** |
|--------------------|----------------|----------------|--------|
| help_category_id | bigint(20) | NO | The category ID. |
| name | varchar(64) | NO | The category name. |
| parent_category_id | bigint(20) | YES | The ID of the parent category. |
| url | varchar(65535) | NO | The URL address. |
