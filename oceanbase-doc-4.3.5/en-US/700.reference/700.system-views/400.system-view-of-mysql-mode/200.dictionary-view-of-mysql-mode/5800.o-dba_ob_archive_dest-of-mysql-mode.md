# oceanbase.DBA_OB_ARCHIVE_DEST

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view was introduced since OceanBase Database V4.0.0. </p>
</main>

## Purpose

This view displays the archive path configuration for the current tenant.

## Columns

| Column | Type | Nullable? | Description |
| --- | --- | --- | --- |
| DEST_NO | bigint(20) | NO | The identifier for log_archive_dest_n. |
| NAME    | varchar(1024) | NO | The name of the field. |
| VALUE   | varchar(4096) | NO | The value of the field. |

## Sample query

User tenants can view the archive path configuration for their own tenant.

```shell
obclient[oceanbase]> SELECT * FROM oceanbase.DBA_OB_ARCHIVE_DEST;
```

The query results are as follows:

```shell
+---------+-----------------------+-------------------------------------------------------------------------------------------------------------------------------+
| DEST_NO | NAME                  | VALUE                                                                                                                         |
+---------+-----------------------+-------------------------------------------------------------------------------------------------------------------------------+
|       0 | binding               | OPTIONAL                                                                                                                      |
|       0 | dest_id               | 1002                                                                                                                          |
|       0 | path                  | oss://example_patharchive?host=cn-******.aliyuncs.com&access_id=******&access_key=*****&checksum_type=md5&enable_worm=true    |
|       0 | piece_switch_interval | 1d                                                                                                                            |
|       0 | state                 | ENABLE                                                                                                                        |
+---------+-----------------------+-------------------------------------------------------------------------------------------------------------------------------+
5 rows in set
```
