# oceanbase.DBA_OB_CCL_RULES

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V4.3.5 BP3. </p>
</main>

## Purpose

This view records all the throttling rules for the current user tenant.

## Columns

| Column | Type | Nullable | Description |
| --- | --- | --- | --- |
|TENANT_ID| bigint(20)| NO| The tenant ID.|
|CCL_RULE_ID|bigint(20)|NO|The throttling rule ID.|
|CCL_RULE_NAME| varchar(128)| NO| The throttling rule name.|
|AFFECT_USER_NAME| varchar(128)| NO |The user to which the throttling applies.|
|AFFECT_HOST| varchar(128)| NO| The IP address to which the throttling applies.|
|AFFECT_FOR_ALL_DATABASES| tinyint(4)| NO| Indicates whether the throttling applies to all databases.|
|AFFECT_FOR_ALL_TABLES| tinyint(4)| NO| Indicates whether the throttling applies to all tables.|
|AFFECT_DML| bigint(20)| NO| The DML statement types to which the throttling applies.|
|AFFECT_DATABASE| bigint(20)| YES |The database to which the throttling applies.|
|AFFECT_TABLE| bigint(20)| YES| The table to which the throttling applies.|
|AFFECT_SCOPE| bigint(10)| NO |The throttling scope:<li>Rule level: 0 </li><li>Format SQLID level:1</li>|
|CCL_KEYWORDS| varchar(1048576)| NO |The throttling keywords.|
|MAX_CONCURRENCY| bigint(20)| NO |The maximum concurrency level for throttling.|

## Sample query

Query the throttling rules of the user tenant.

```shell
obclient[oceanbase]> SELECT * FROM DBA_OB_CCL_RULES;
```
