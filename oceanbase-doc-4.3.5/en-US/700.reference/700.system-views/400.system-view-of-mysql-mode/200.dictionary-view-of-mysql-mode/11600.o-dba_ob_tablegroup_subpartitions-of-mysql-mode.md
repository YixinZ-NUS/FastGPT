# oceanbase.DBA_OB_TABLEGROUP_SUBPARTITIONS

<main id="notice" type='explain'>
<h4>Note</h4>
<ul>
<li>In V4.2.0 and later, table groups do not support partitions. Therefore, this view is no longer used. You can query this view, but the query result is empty. </li>
<li>In V4.2.0 and later, you can query information about table groups by using the view <a href="11800.o-dba_ob_tablegroups-of-mysql-mode.md">DBA_OB_TABLEGROUPS</a>. For more information about table groups, see <a href="../../../300.database-object-management/100.manage-object-of-mysql-mode/400.manage-table-groups-of-mysql-mode/100.about-table-groups-of-mysql-mode.md">About table groups</a>. </li>
</ul>
</main>

## Purpose

This view displays the subpartition information of a table group that contains partitions.

## Columns

|         Column         |      Type       | Nullable? | Description |
|-----------------------|---------------|------------|----------------------------------------------------------------------|
| TABLEGROUP_NAME       | varchar(128)  | NO         | The name of the table group.    |
| PARTITION_NAME        | varchar(64)   | NO         | The name of the primary partition corresponding to the subpartition.  |
| SUBPARTITION_NAME     | varchar(64)   | NO         | The name of the subpartition.         |
| HIGH_VALUE            | varchar(4096) | NO         | The value is `NULL`.                                                       |
| HIGH_VALUE_LENGTH     | bigint(20)    | NO         | The value is `NULL`.                                                                           |
| PARTITION_POSITION    | bigint(20) unsigned  | NO  | The value is `NULL`.      |
| SUBPARTITION_POSITION | bigint(20) unsigned  | NO  | The value is `NULL`.            |
