# information_schema.SESSION_STATUS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is introduced since OceanBase Database V1.4. </p>
</main>

## Purpose

This view displays the session status, such as the number of active connections and the database active duration (uptime).

## Columns

| **Column** | **Type** | **Nullable?** | **Description** |
|----------------|---------------|----------------|-------------|
| VARIABLE_NAME | varchar(128) | NO | The name of a session status variable. |
| VARIABLE_VALUE | varchar(1024) | YES | The value of the session status variable. |

## Sample query

Query the server status variables of the session and their corresponding values.

```shell
obclient [oceanbase]> SELECT * FROM information_schema.SESSION_STATUS;
```

The query result is as follows:

```shell
+-------------------+----------------+
| VARIABLE_NAME     | VARIABLE_VALUE |
+-------------------+----------------+
| Threads_connected | 2              |
| Uptime            | 255239         |
+-------------------+----------------+
2 rows in set
```
