# oceanbase.DBA_RSRC_PLAN_DIRECTIVES

<main id="notice" type='explain'>
<h4>Note</h4>
<ul><li>The view was introduced in V3.1.2. </li>
<li>In the current version, we recommend that you use the view <a href="11020.o-dba_ob_rsrc_directives-of-mysql-mode.md">DBA_OB_RSRC_DIRECTIVES</a> to view the resource configuration parameters of a tenant in Resource Manager. </li></ul>
</main>

## Purpose

This view displays the details of resource plan directives in the database.
