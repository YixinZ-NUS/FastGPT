# oceanbase.DBA_TAB_SUBPARTITIONS

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>

## Purpose

This view displays the names of subpartitions, tables, and partitions to which subpartitions belong, as well as their storage attributes and statistics generated by the `DBMS_STATS` package.
