# oceanbase.DBA_OB_LS_HISTORY

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is introduced in V4.2.0. </p>
</main>

## Purpose

This view displays the log streams that existed in the tenant.

## Columns

| Column | Type | Nullable? | Description |
| --- | --- | --- | --- |
| LS_ID       | bigint(20)    | NO          |  Log stream ID                                 |
| LS_GROUP_ID | bigint(20)    | NO          |  Log stream group ID                               |
| STATUS      | varchar(100)  | NO          |  Current status of the log stream. Valid values: <ul><li>CREATING: the log stream is being created. </li><li>NORMAL: the log stream is in the normal state. </li><li>DROPPING: the log stream is being dropped. </li><li>PRE_TENANT_DROPPING: a special state of the system log stream, indicating the start of dropping all log streams in the tenant. </li><li>TENANT_DROPPING: the tenant is being dropped, and the log streams are being dropped. </li><li>DROPPED: the log stream has been recycled or is being recycled. </li><li>WAIT_OFFLINE: the log stream is waiting to be recycled. </li><li>CREATE_ABORT: the log stream creation is stopped. </li></ul>        |
| FLAG        | varchar(2048) | NO          |  Log stream flag. Valid values: <ul><li>Empty string: a normal log stream. </li><li>DUPLICATE: a broadcast log stream. A replica table is created on this log stream. </li> <li>BLOCK_TABLET_IN: Tablets cannot be created on this log stream. </li> <li>DUPLICATE|BLOCK_TABLET_IN: the broadcast log stream does not allow Tablets to be created. </li></ul>                        |
| CREATE_SCN  | decimal(20,0) | NO          |  GTS at which the log stream was created, in nanoseconds.              |

## Sample query

The following example shows how to query the log streams that existed in the user tenant.

```sshell
obclient [oceanbase]> SELECT * FROM oceanbase.DBA_OB_LS_HISTORY;
```

The query result is as follows:

```shell
+-------+-------------+--------+------+---------------------+
| LS_ID | LS_GROUP_ID | STATUS | FLAG | CREATE_SCN          |
+-------+-------------+--------+------+---------------------+
|     1 |           0 | NORMAL |      |                   0 |
|  1001 |        1001 | NORMAL |      | 1736152507462028000 |
+-------+-------------+--------+------+---------------------+
2 rows in set
```
