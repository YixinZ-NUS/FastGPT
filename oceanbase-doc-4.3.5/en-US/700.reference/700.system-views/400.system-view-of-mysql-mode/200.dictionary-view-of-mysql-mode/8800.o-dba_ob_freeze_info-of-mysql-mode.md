# oceanbase.DBA_OB_FREEZE_INFO

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is introduced in OceanBase Database V4.0.0. </p>
</main>

## Purpose

This view displays the information about the versions generated during major compactions in the current tenant.

## Columns

| Column | Type | Nullable? | Description |
| --- | --- | --- | --- |
| FROZEN_SCN | bigint(20) unsigned | NO | The version number of the major compaction. |
| CLUSTER_VERSION | bigint(20) | NO | The cluster version at the time of the major compaction. |
| SCHEMA_VERSION | bigint(20) | NO | The schema version at the time of the major compaction. |
| GMT_CREATE | timestamp(6) | NO | The creation time of the record. |
| GMT_MODIFIED | timestamp(6) | NO | The last modification time of the record. |

## Sample query

Query the information about the versions generated during major compactions in the current user tenant.

```shell
obclient [oceanbase]> SELECT * FROM oceanbase.DBA_OB_FREEZE_INFO;
```
