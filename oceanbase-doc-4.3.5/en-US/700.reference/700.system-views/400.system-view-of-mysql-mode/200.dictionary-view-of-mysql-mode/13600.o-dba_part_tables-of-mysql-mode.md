# oceanbase.DBA_PART_TABLES

## Purpose

This view displays the partition information of a user table.

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>

## Columns

| Column       |  Type  | Nullable? | Description |
|------------------|-----------|----------|-------------|
| OWNER            | varchar(128) | NO | The owner of the partitioned table. |
| TABLE_NAME       | varchar(128) | NO | The name of the partitioned table. |
| PARTITIONING_TYPE | varchar(13) | NO | The partitioning method. Valid values include: <li> HASH <li> RANGE <li> LIST |
| SUBPARTITIONING_TYPE | varchar(13) | NO | The subpartitioning method. Valid values include: <li> HASH <li> RANGE <li> LIST |
| PARTITION_COUNT  | bigint(20) | NO | The number of partitions in the partitioned table. |
| DEF_SUBPARTITION_COUNT | bigint(20) | NO | The number of subpartitions in the composite partitioned table. |
| PARTITIONING_KEY_COUNT | bigint(21) | NO | The number of partitioning keys. |
| SUBPARTITIONING_KEY_COUNT | bigint(21) | NO | The number of subpartitioning keys in the composite partitioned table. |
| STATUS           | varchar(8) | NO | This field is currently not supported. The default value is NULL. |
| DEF_TABLESPACE_NAME | varchar(30) | NO | The name of the tablespace to which the partitioned table belongs. |
| DEF_PCT_FREE     | bigint(0)  | NO | This field is currently not supported. The default value is NULL. |
| DEF_PCT_USED     | bigint(0)  | NO | This field is currently not supported. The default value is NULL. |
| DEF_INI_TRANS    | bigint(0)  | NO | This field is currently not supported. The default value is NULL. |
| DEF_MAX_TRANS    | bigint(0)  | NO | This field is currently not supported. The default value is NULL. |
| DEF_INITIAL_EXTENT | varchar(40) | NO | This field is currently not supported. The default value is NULL. |
| DEF_NEXT_EXTENT  | varchar(40) | NO | This field is currently not supported. The default value is NULL. |
| DEF_MIN_EXTENTS  | varchar(40) | NO | This field is currently not supported. The default value is NULL. |
| DEF_MAX_EXTENTS  | varchar(40) | NO | This field is currently not supported. The default value is NULL. |
| DEF_MAX_SIZE     | varchar(40) | NO | This field is currently not supported. The default value is NULL. |
| DEF_PCT_INCREASE | varchar(40) | NO | This field is currently not supported. The default value is NULL. |
| DEF_FREELISTS    | bigint(0)  | NO | This field is currently not supported. The default value is NULL. |
| DEF_FREELIST_GROUPS | bigint(0) | NO | This field is currently not supported. The default value is NULL. |
| DEF_LOGGING      | varchar(7) | NO | This field is currently not supported. The default value is NULL. |
| DEF_COMPRESSION  | varchar(8) | NO | Whether to enable compression. |
| DEF_COMPRESS_FOR | varchar(12) | NO | The compression method. |
| DEF_BUFFER_POOL  | varchar(7) | NO | This field is currently not supported. The default value is NULL. |
| DEF_FLASH_CACHE  | varchar(7) | NO | This field is currently not supported. The default value is NULL. |
| DEF_CELL_FLASH_CACHE | varchar(7) | NO | This field is currently not supported. The default value is NULL. |
| REF_PTN_CONSTRAINT_NAME | varchar(30) | NO | This field is currently not supported. The default value is NULL. |
| INTERVAL         | text      | YES | This field is currently not supported. The default value is NULL. |
| AUTOLIST         | varchar(3) | NO | Indicates whether the table is automatically list-partitioned: <li>YES<li>NO |
| INTERVAL_SUBPARTITION | text     | NO | The string value of the subpartitioning interval. |
| AUTOLIST_SUBPARTITION | varchar(3) | NO | Indicates whether to automatically list-partition this subpartition: <li>YES<li>NO |
| IS_NESTED        | varchar(3) | NO | This field is currently not supported. The default value is NULL. |
| DEF_SEGMENT_CREATED | varchar(4) | NO | This field is currently not supported. The default value is NULL. |
| DEF_INDEXING     | varchar(3) | NO | The index attribute of the table. Valid values include: <li>ON-INDEXING: ON is specified or the index attribute is not specified<li>OFF-INDEXING: The index attribute is specified as OFF. |
| DEF_INMEMORY     | varchar(8) | NO | The default value is NULL, which has no practical meaning. |
| DEF_INMEMORY_PRIORITY | varchar(8) | NO | The default value is NULL, which has no practical meaning. |
| DEF_INMEMORY_DISTRIBUTE | varchar(15) | NO | The default value is NULL, which has no practical meaning. |
| DEF_INMEMORY_COMPRESSION | varchar(17) | NO | The default value is NULL, which has no practical meaning. |
| DEF_INMEMORY_DUPLICATE | varchar(13) | NO | The default value is NULL, which has no practical meaning. |

| DEF_CELLMEMORY   | varchar(24) | NO | The default value is NULL, which has no practical meaning. |
| DEF_INMEMORY_SERVICE | varchar(12) | NO | The default value is NULL, which has no practical meaning. |
| DEF_INMEMORY_SERVICE_NAME | text     | NO | The default value is NULL, which has no practical meaning. |
| AUTO             | varchar(3) | NO | Indicates whether the table is automatically partitioned: <li>YES<li>NO |
