# ALL_TAB_SUBPARTITIONS

## Purpose

The ALL_TAB_SUBPARTITIONS view displays the partition-level partitioning information, storage parameters, and statistics generated by the DBMS_STATS package for partitions that the current user can access.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>
