# DBA_COL_COMMENTS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

This view displays all column comments of all views and tables in the current tenant.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

|  **Column**   |     **Type**     | **Nullable?** | **Description** |
|-------------|----------------|----------------|--------|
| OWNER       | VARCHAR2(128)  | NO             | The owner of the object. |
| OBJECT_NAME | VARCHAR2(128)  | NO             | The name of the object. |
| COLUMN_NAME | VARCHAR2(128)  | NO             | The name of the column. |
| COMMENTS    | VARCHAR2(4000) | YES            | The comment of the column. |

## Sample query

Query all column comments of the `T1` table.

```shell
obclient [SYS]> SELECT * FROM SYS.DBA_COL_COMMENTS WHERE TABLE_NAME='T1';
```

The query result is as follows:

```shell
+-------+------------+----------------+----------+
| OWNER | TABLE_NAME | COLUMN_NAME    | COMMENTS |
+-------+------------+----------------+----------+
| SYS   | T1         | __pk_increment | NULL     |
| SYS   | T1         | ID1            | score    |
| SYS   | T1         | ID2            | NULL     |
+-------+------------+----------------+----------+
3 rows in set
```
