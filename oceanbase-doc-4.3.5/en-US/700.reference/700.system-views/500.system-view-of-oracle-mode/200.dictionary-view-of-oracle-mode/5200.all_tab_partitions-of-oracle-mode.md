# ALL_TAB_PARTITIONS

## Purpose

The view displays the multi-level partitioning information, storage parameters, and statistics of all partitioned tables.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>
