# ALL_IND_PARTITIONS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

This view displays information about the index partitions that are accessible to the current user.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view is available only in Oracle mode of OceanBase Database. </p>
  </main>
