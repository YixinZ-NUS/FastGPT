# DBA_OB_LS_REPLICA_TASK_HISTORY

<main id="notice" type='explain'>
  <h4>Note</h4>
  <ul><li>This view is available starting with V4.3.3 in V4.3.x. </li><li>This view is available starting with V4.2.3 in V4.2.x. </li></ul>
</main>

## Purpose

The `DBA_OB_LS_REPLICA_TASK_HISTORY` view displays the execution history of disaster recovery tasks.

## Columns

| **Column** | **Type** | **Nullable** | **Description** |
| --- | --- | --- | --- |
| LS_ID | NUMBER(38) | NO | The ID of the log stream. |
| TASK_TYPE | VARCHAR2(64) | NO | The type of the task. Valid values: <ul><li>`MIGRATE REPLICA`: migrate a replica</li><li>`ADD REPLICA`: add a replica</li><li>`TYPE TRANSFORM`: convert the type of a replica</li><li>`REMOVE PAXOS REPLICA`: remove a Paxos member replica</li><li>`REMOVE NON PAXOS REPLICA`: remove a non-Paxos member replica</li><li>`MODIFY PAXOS REPLICA NUMBER`: modify the number of required Paxos member replicas</li></ul>|
| TASK_ID | VARCHAR2(200) | NO | The ID of the task. |
| TASK_STATUS | VARCHAR2(2048) | YES | The status of the task. Valid values: <ul><li>`INPROGRESS`: the task is being scheduled</li><li>`COMPLETED`: the task is completed</li><li>`CANCELED`: the task is canceled</li></ul>|
| PRIORITY | CHAR(5) | NO | The priority of the task. Valid values: <ul><li>`LOW`: low priority</li><li>`HIGH`: high priority</li></ul>|
| TARGET_REPLICA_SVR_IP | VARCHAR2(46) | YES | The IP address of the server where the target replica is located. |
| TARGET_REPLICA_SVR_PORT | NUMBER(38) | YES | The port number of the server where the target replica is located. |
| TARGET_PAXOS_REPLICA_NUMBER | NUMBER(38) | YES | The number of required Paxos member replicas in the log stream. |
| TARGET_REPLICA_TYPE | VARCHAR2(16) | YES | The type of the target replica. |
| SOURCE_REPLICA_SVR_IP | VARCHAR2(46) | YES | The IP address of the server where the source replica is located. |
| SOURCE_REPLICA_SVR_PORT | NUMBER(38) | YES | The port number of the server where the source replica is located. |
| SOURCE_PAXOS_REPLICA_NUMBER | NUMBER(38) | YES | The number of required Paxos member replicas in the log stream. |
| SOURCE_REPLICA_TYPE | VARCHAR2(16) | YES | The type of the source replica. |
| DATA_SOURCE_SVR_IP | VARCHAR2(46) | YES | The IP address of the data source. |
| DATA_SOURCE_SVR_PORT | NUMBER(38) | YES | The port number of the data source. |
| IS_MANUAL | CHAR(6) | NO | The source of the disaster recovery task. Valid values: <ul><li>`TRUE`: the task is triggered by a manual operation. </li><li>`FALSE`: the task is automatically generated by the RS layer. </li></ul>|
| TASK_EXEC_SVR_IP | VARCHAR2(46) | YES | The IP address of the server where the task is executed. |
| TASK_EXEC_SVR_PORT | NUMBER(38) | YES | The port number of the server where the task is executed. |
| CREATE_TIME | TIMESTAMP(6) | NO | The time when the task is generated. |
| START_TIME | TIMESTAMP(6) | NO | The time when the task is scheduled. |
| MODIFY_TIME | TIMESTAMP(6) | NO | The time when the status of the task is updated. |
| FINISH_TIME | TIMESTAMP(6) | NO | The time when the task is completed. |
| EXECUTE_RESULT | VARCHAR2(2048) | YES | The result of the task execution. |
| COMMENT | VARCHAR2(2048) | YES | The remarks about the reason for generating the task. |
