# USER_COL_PRIVS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.77. </p>
</main>

## Purpose

Describes the column privileges granted to the current user as the object owner, grantor, or grantee.

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Oracle mode of OceanBase Database does not support column privileges. </p>
</main>

## Columns

|  **Column**   |    **Type**     | **Nullable** |      **Description**      |
|-------------|---------------|----------------|------------------|
| GRANTEE     | VARCHAR2(128) | NO             | The name of the user to whom the privilege is granted.    |
| OWNER       | VARCHAR2(128) | NO             | The owner of the object.           |
| TABLE_NAME  | VARCHAR2(256) | NO             | The name of the object.            |
| COLUMN_NAME | VARCHAR2(128) | NO             | The name of the column.             |
| GRANTOR     | VARCHAR2(128) | NO             | The name of the user who granted the privilege.       |
| PRIVILEGE   | VARCHAR2(40)  | NO             | The privilege granted on the column.            |
| GRANTABLE   | VARCHAR2(3)   | NO             | Indicates whether the GRANT statement can be used to grant privileges. |
