# DBA_JOBS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V3.2.1. </p>
</main>

## Purpose

This view displays an overview of the submitted tasks.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

|  **Column**   |     **Type**     | **Nullable?** |                                                                  **Description**                                                                   |
|-------------|----------------|----------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| JOB         | NUMBER(38)     | NO             | The job identifier. The value of this parameter does not change when you import or export a job or resubmit a job.                            |
| LOG_USER    | VARCHAR2(128)  | NO             | The user who submitted the job.                                            |
| PRIV_USER   | VARCHAR2(128)  | NO             | The user with default privileges for the job.                                         |
| SCHEMA_USER | VARCHAR2(128)  | NO             | The default schema user for job parsing.                                     |
| LAST_DATE   | DATE           | YES            | The date when the job was last successfully executed.                                          |
| LAST_SEC    | VARCHAR2(32)   | YES            | The start time when the job was last successfully executed.                                        |
| THIS_DATE   | DATE           | YES            | The date when the job was started.                                            |
| THIS_SEC    | VARCHAR2(32)   | YES            | The start time when the job was last successfully executed.                                          |
| NEXT_DATE   | DATE           | NO             | The date when the job will be executed next.                                            |
| NEXT_SEC    | VARCHAR2(32)   | YES            | The start time when the job will be executed next.                                          |
| TOTAL_TIME  | NUMBER         | YES            | The total clock time (in seconds) spent by the system on the job since the job was first executed. The value is cumulative.             |
| BROKEN      | VARCHAR2(1)    | YES            | Whether the job is running: <li> `N`: The job is running.   <li> `Y`: The job is not running.    |
| INTERVAL    | VARCHAR2(200)  | NO             | The date function that records the time interval from the start of the job to the start of the next execution.                      |
| FAILURES    | NUMBER(38)     | YES            | The number of successful and failed executions since the job was last successfully executed.                                |
| WHAT        | VARCHAR2(4000) | YES            | The body of the anonymous PL/SQL block that executes the job.                                       |
| NLS_ENV     | VARCHAR2(4000) | YES            | The session parameters that describe the NLS environment of the job.                                      |
| MISC_ENV    | RAW(32)        | YES            | Other session parameters applicable to the job.                                         |
| INSTANCE    | VARCHAR2(128)  | YES            | The zone where the job is running.                                       |
