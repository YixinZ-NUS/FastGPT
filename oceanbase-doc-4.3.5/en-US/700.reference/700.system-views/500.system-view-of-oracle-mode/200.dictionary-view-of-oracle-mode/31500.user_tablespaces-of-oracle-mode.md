# USER_TABLESPACES

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

This view displays the tablespaces that the current user can access. The columns in this view are the same as those in the `DBA_TABLESPACES` view, except the `PLUGGED_IN` column.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view is available only in Oracle mode of OceanBase Database. </p>
  </main>

## Sample query

Query all tablespaces that the current user can access.

```shell
obclient [SYS]> SELECT TABLESPACE_NAME, ENCRYPTED FROM SYS.USER_TABLESPACES;
```

The query result is as follows:

```shell
+-----------------+-----------+
| TABLESPACE_NAME | ENCRYPTED |
+-----------------+-----------+
| SECTEST_TS1     | YES       |
| TS2             | NO        |
+-----------------+-----------+
2 rows in set
```
