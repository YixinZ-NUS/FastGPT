# USER_TAB_COLS

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view was introduced in V2.2.77. </p>
</main>

## Purpose

The `USER_TAB_COLS` view displays all columns of tables and views accessible to the current user.

The `USER_TAB_COLS` view differs from the [USER_TAB_COLUMNS](30500.user_tab_columns-of-oracle-mode.md) view in that the `USER_TAB_COLS` view displays columns that are marked for quick deletion, whereas the `USER_TAB_COLUMNS` view does not.

<main id="notice" >
  <h4>Applicability</h4>
  <p>This view applies only to OceanBase Database in Oracle mode. </p>
</main>
