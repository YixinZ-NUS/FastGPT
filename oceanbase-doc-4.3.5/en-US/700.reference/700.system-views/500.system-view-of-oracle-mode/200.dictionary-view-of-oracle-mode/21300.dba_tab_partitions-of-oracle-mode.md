# DBA_TAB_PARTITIONS

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view was introduced in OceanBase Database V2.2.77. </p>
</main>

## Purpose

This view displays the information about the current user's access to the multi-level partitions, partition storage parameters, and partition statistics.

|-----------|----------|----------|-------------|
