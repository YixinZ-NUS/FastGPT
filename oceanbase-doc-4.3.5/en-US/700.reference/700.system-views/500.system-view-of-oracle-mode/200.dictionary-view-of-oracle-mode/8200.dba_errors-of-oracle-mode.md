# DBA_ERRORS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

This view displays the error information of the current stored objects in the database.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

|    **Column**    |     **Type**     | **Nullable?** |                                             **Description**                                              |
|----------------|----------------|----------------|-----------------------------------|
| OWNER          | VARCHAR2(128)  | NO             | The owner of the object.      |
| NAME           | VARCHAR2(128)  | NO             | The name of the object.       |
| TYPE           | VARCHAR2(19)   | NO             | The type of the object: <li> VIEW   <li> PROCEDUCE   <li> FUNCTION   <li> PACKAGE   <li> PACKAGE_BODY   <li> TRIGGER   <li> TYPE   <li> TYPE BODY   <li> LIBRARY   <li> JAVA SOURCE   <li> JAVA CLASS   <li> DIMENSION    |
| SEQUENCE       | NUMBER         | NO             | The error number of the errors generated during the same compilation (for sorting).                                                                            |
| LINE           | NUMBER         | NO             | The line number where the error occurred.     |
| POSITION       | NUMBER         | NO             | The column number where the error occurred.     |
| TEXT           | VARCHAR2(4000) | NO             | The text of the error message.      |
| ATTRIBUTE      | VARCHAR2(9)    | NO             | Indicates whether the message is an error or a warning.                                                                           |
| MESSAGE_NUMBER | NUMBER         | NO             | The error code.        |
