# USER_OB_SOURCE

<main id="notice" type='explain'>
 <h4>Note</h4>
 <p>This view is introduced in OceanBase Database V4.3.5 BP4. </p>
</main>

## Purpose

The `USER_OB_SOURCE` view displays the source code information of the storage objects owned by the current user.

## Columns

| Column | Type | Nullable? | Description |
|------------|-------------|---------------------|----------------------------------------|
| NAME         | VARCHAR2(128)  | NO   | The name of the object. |
| TYPE          | VARCHAR2(12)   | NO   | The type of the object. |
| LINE          | NUMBER(38)     | NO   | The source code line number, which identifies the position of the line in the object definition. |
| TEXT          | VARCHAR2(4000) | NO   | The source code text, which stores the actual code content corresponding to the line. |
| ORIGIN_CON_ID | NUMBER(38)     | NO   | The tenant ID, which identifies the tenant to which the object belongs. |

## Examples

Execute the following command to view the source code information of the storage objects owned by the current user, and display the first five records.

```shell
obclient [oceanbase]>SELECT * FROM USER_OB_SOURCE WHERE rownum <=5;
```

The query result is as follows:

```shell
+------------+---------+------+--------------------------------------------------------------------+---------------+
| NAME       | TYPE    | LINE | TEXT                                                               | ORIGIN_CON_ID |
+------------+---------+------+--------------------------------------------------------------------+---------------+
| dbms_stats | PACKAGE |    1 | PACKAGE dbms_stats AUTHID CURRENT_USER                             |             1 |
| dbms_stats | PACKAGE |    2 | DECLARE DEFAULT_METHOD_OPT     VARCHAR(1) DEFAULT 'Z';             |             1 |
| dbms_stats | PACKAGE |    3 | DECLARE DEFAULT_GRANULARITY    VARCHAR(1) DEFAULT 'Z';             |             1 |
| dbms_stats | PACKAGE |    4 | DECLARE AUTO_SAMPLE_SIZE       DECIMAL(20, 10) DEFAULT 0;          |             1 |
| dbms_stats | PACKAGE |    5 | DECLARE DEFAULT_STAT_CATEGORY  VARCHAR(20) DEFAULT 'OBJECT_STATS'; |             1 |
+------------+---------+------+--------------------------------------------------------------------+---------------+
```
