# DBA_OB_DYNAMIC_PARTITION_TABLES

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view is introduced in V4.3.5 BP2. </p>
</main>

## Purpose

This view displays the information about dynamic partition tables in the current tenant.

## Columns

|        Column        |       Type       | Nullable? | Description |
|--------------------|-----------------|-------------|------|
| DATABASE_NAME      | VARCHAR2(128)   | YES         | The name of the schema where the dynamic partition table resides. |
| TABLE_NAME         | VARCHAR2(256)   | NO          | The name of the dynamic partition table. |
| TABLE_ID           | NUMBER(38)      | NO          | The ID of the dynamic partition table. |
| MAX_HIGH_BOUND_VAL | VARCHAR2(4096)  | NO          | The maximum range partition upper bound. |
| ENABLE             | VARCHAR2(16384) | NO          | Whether dynamic partition management is enabled. |
| TIME_UNIT          | VARCHAR2(16384) | NO          | The `time_unit` value in dynamic partition management. |
| PRECREATE_TIME     | VARCHAR2(16384) | NO          | The `precreate_time` value in dynamic partition management. |
| EXPIRE_TIME        | VARCHAR2(16384) | NO          | The `expire_time` value in dynamic partition management. |
| TIME_ZONE          | VARCHAR2(16384) | NO          | The `time_zone` value in dynamic partition management. |
| BIGINT_PRECISION   | VARCHAR2(16384) | NO          | The `bigint_precision` value in dynamic partition management. |

## Sample query

```sql
SELECT * FROM sys.DBA_OB_DYNAMIC_PARTITION_TABLES\G
```
