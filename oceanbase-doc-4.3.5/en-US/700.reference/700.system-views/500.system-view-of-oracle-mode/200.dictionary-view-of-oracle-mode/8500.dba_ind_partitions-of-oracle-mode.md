# DBA_IND_PARTITIONS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

This view displays information about all index partitions in the current tenant.

## Columns

|          Column           |       Type        | Nullable |                                                                        Description                                                                         |
|-------------------------|-----------------|------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| INDEX_OWNER             | VARCHAR2(128)   | NO         | The owner of the index.                                                       |
| INDEX_NAME              | VARCHAR2(128)   | NO         | The name of the index.                                                           |
| COMPOSITE               | CHAR(3)         | NO         | Indicates whether the partition is part of a composite-partitioned table's local index:<li>YES<li>NO             |
