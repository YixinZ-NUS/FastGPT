# ALL_TYPE_METHODS

  <main id="notice" type='explain'>
    <h4>Note</h4>
    <p>OceanBase Database does not support this view. </p>
  </main>

## Purpose

The view displays the methods that a user can access.

## References

* DBA_TYPE_METHODS

* USER_TYPE_METHODS

## Columns

|   **Column**   |    **Type**    | **Nullable?** |                                                                                 **Description**                                                                                  |
|--------------|--------------|----------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| OWNER        | VARCHAR2(30) | NO             | The owner of the type.                                                                               |
| TYPE_NAME    | VARCHAR2(30) | NO             | The name of the type.                                                                                |
| METHOD_NAME  | VARCHAR2(30) | NO             | The name of the method.                                                                                |
| METHOD_NO    | NUMBER       | NO             | The method number, which is used to distinguish overloaded methods.                                                                        |
| METHOD_TYPE  | VARCHAR2(6)  | NO             | The method type: <li> MAP   <li> ORDER   <li> PUBLIC    |
| PARAMETERS   | NUMBER       | NO             | The number of method parameters.                                                                              |
| RESULTS      | NUMBER       | NO             | The number of method return results.                                                                            |
| FINAL        | VARCHAR2(3)  | NO             | Indicates whether the method is final: <li> YES   <li> NO                                         |
| INSTANTIABLE | VARCHAR2(3)  | NO             | Indicates whether the method can be instantiated: <li> YES   <li> NO                                            |
| OVERRIDING   | VARCHAR2(3)  | NO             | Indicates whether the method overrides a method of a super type: <li> YES   <li> NO                                       |
| INHERITED    | VARCHAR2(3)  | NO             | Indicates whether the method is inherited from a super type: <li> YES   <li> NO                                        |
