# ALL_COL_PRIVS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.77. </p>
</main>

## Purpose

The `ALL_COL_PRIVS` view describes the following types of privileges:

* Column object privileges granted to, granted by, or owned by the current user

* Column object privileges granted to the PUBLIC role or to the current user in an enabled role

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

|  **Column**   |    **Type**     | **Nullable?** |    **Description**     |
|-------------|---------------|----------------|---------------|
| GRANTOR     | VARCHAR2(128) | NO             | The name of the user who granted the privilege.    |
| GRANTEE     | VARCHAR2(128) | NO             | The name of the user to whom the privilege was granted. |
| OWNER       | VARCHAR2(128) | NO             | The owner of the object.        |
| TABLE_NAME  | VARCHAR2(256) | NO             | The name of the object.         |
| COLUMN_NAME | VARCHAR2(128) | NO             | The name of the column.          |
| PRIVILLEGE  | VARCHAR2(40)  | NO             | The column privilege.          |
| GRANTABLE   | VARCHAR2(3)   | NO             | Indicates whether the grant option is available.      |
