# USER_VIEWS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

This view displays the views owned by the current user.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view is available starting with OceanBase Database Oracle mode. </p>
  </main>

## Columns

|    **Column**     |     **Type**     | **Nullable** |                           **Description**                           |

## Sample query

Query the information about the view `VIEW1` owned by the current user.

```shell
obclient [SYS]> SELECT VIEW_NAME, TEXT_LENGTH, TEXT FROM SYS.USER_VIEWS WHERE VIEW_NAME='VIEW1';
```

The query result is as follows:

```shell
+-----------+-------------+-----------------------------+
| VIEW_NAME | TEXT_LENGTH | TEXT                        |
+-----------+-------------+-----------------------------+
| VIEW1     |          27 | SELECT col1, col2 FROM tbl1 |
+-----------+-------------+-----------------------------+
1 row in set
```
