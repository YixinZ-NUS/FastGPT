# USER_TAB_PARTITIONS

## Purpose

View the multi-level partition information, partition storage parameters, and partition statistics of the current user.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## References

* DBA_TAB_PARTITIONS

* ALL_TAB_PARTITIONS
