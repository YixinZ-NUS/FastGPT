# DBA_PART_INDEXES

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view was introduced in OceanBase Database V2.2.77. </p>
</main>

## Purpose

This view displays information about all partitions of partitioned indexes.

## Columns

| Column |  Type  | Nullable? | Description |
|------------|----------|----------|-------------|
| OWNER      | VARCHAR2(128)  | NO         | The owner of the partitioned index. |
| INDEX_NAME | VARCHAR2(128)  | NO         | The name of the partitioned index. |
| TABLE_NAME | VARCHAR2(128)  | NO         | The name of the table associated with the index. |
| PARTITIONING_TYPE | VARCHAR2(9)    | NO         | The partitioning method. Valid values include: <li> HASH   <li> RANGE   <li> LIST        |
| SUBPARTITIONING_TYPE | VARCHAR2(9)    | NO         | The subpartitioning method. Valid values include: <li> HASH   <li> RANGE   <li> LIST    |
| PARTITION_COUNT | NUMBER         | NO         | The number of partitions in the index.           |
| DEF_SUBPARTITION_COUNT | NUMBER         | NO         | The default number of subpartitions for composite partitioned indexes (if specified).        |
| PARTITIONING_KEY_COUNT | NUMBER         | NO         | The number of columns in the partitioning key.                    |
| SUBPARTITIONING_KEY_COUNT | NUMBER         | NO         | The number of columns in the subpartitioning key for composite partitioned indexes.       |
| LOCALITY | VARCHAR2(6)    | NO         | LOCAL or GLOBAL, indicating whether the partitioned index is local or global.          |
| ALIGNMENT | VARCHAR2(12)   | NO         | PREFIXED or NON_PREFIXED, indicating whether the partitioned index is prefixed or non-prefixed.       |
| DEF_TABLESPACE_NAME | VARCHAR2(30)   | NO         | The default tablespace for adding or splitting a table partition for local indexes.           |
| DEF_PCT_FREE | NUMBER         | NO             | Not supported. The default value is 0.          |
| DEF_INI_TRANS | NUMBER         | NO             | Not supported. The default value is 0.              |
| DEF_MAX_TRANS | NUMBER         | NO             | Not supported. The default value is 0.                |
| DEF_INITIAL_EXTENT | VARCHAR2(40)   | NO             | Not supported. The default value is NULL.             |
| DEF_NEXT_EXTENT | VARCHAR2(40)   | NO             | Not supported. The default value is NULL.          |
| DEF_MIN_EXTENTS | VARCHAR2(40)   | NO             | Not supported. The default value is NULL.            |
| DEF_MAX_EXTENTS | VARCHAR2(40)   | NO             | Not supported. The default value is NULL.       |
| DEF_MAX_SIZE | VARCHAR2(40)   | NO             | Not supported. The default value is NULL.       |
| DEF_PCT_INCREASE | VARCHAR2(40)   | NO             | Not supported. The default value is NULL.       |
| DEF_FREELISTS | NUMBER         | NO             | Not supported. The default value is 0.         |
| DEF_FREELIST_GROUPS | NUMBER         | NO             | Not supported. The default value is 0.            |
| DEF_LOGGING | VARCHAR2(7)    | NO             | Not supported. The default value is NULL.          |
| DEF_BUFFER_POOL | VARCHAR2(7)    | NO             | Not supported. The default value is NULL.           |
| DEF_FLASH_CACHE | VARCHAR2(7)    | NO             | Not supported. The default value is NULL.            |
| DEF_CELL_FLASH_CACHE | VARCHAR2(7)    | NO             | Not supported. The default value is NULL.             |
| DEF_PARAMETERS | VARCHAR2(1000) | NO             | Not supported. The default value is NULL.                |
| INTERVAL | VARCHAR2(1000) | NO             | Not supported. The default value is `NO`.           |
| AUTOLIST | VARCHAR2(3)    | NO         | Indicates whether the local index is automatically list-partitioned. Valid values include: <li>YES<li>NO      |
| INTERVAL_SUBPARTITION | VARCHAR2(1000) | NO         | Not supported. The default value is NULL.     |
| AUTOLIST_SUBPARTITION | VARCHAR2(1000) | NO         | Not supported. The default value is NULL.     |
