# DBA_PART_TABLES

## Purpose

This view displays the partition information of partitions that the current user can access.

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This function was introduced in V2.2.77. </p>
</main>

## References

* `ALL_PART_TABLES`

* `USER_PART_TABLES`
