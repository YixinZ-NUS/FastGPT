# USER_ERRORS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

This view displays the error information of the storage objects owned by the current user.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

|    **Field name**    |     **Type**     | **Nullable?** |                                             **Description**                                              |
|----------------|----------------|----------------|-----------------------------------|
| NAME           | VARCHAR2(128)  | NO             | The name of the object.       |
| TYPE           | VARCHAR2(19)   | NO             | The type of the object: <ul><li> VIEW </li>  <li> PROCEDUCE  </li> <li> FUNCTION  </li> <li> PACKAGE </li>  <li> PACKAGE_BODY   </li> <li> TRIGGER  </li> <li> TYPE  </li> <li> TYPE BODY  </li> <li> LIBRARY </li>  <li> JAVA SOURCE  </li> <li> JAVA CLASS  </li> <li> DIMENSION </li> </ul>  |
| SEQUENCE       | NUMBER         | NO             | The error number of the errors that occur during the same compilation (not sorted).                                                                            |
| LINE           | NUMBER         | NO             | The line number where the error occurs.     |
| POSITION       | NUMBER         | NO             | The column number where the error occurs.     |
| TEXT           | VARCHAR2(4000) | NO             | The error message.      |
| ATTRIBUTE      | VARCHAR2(9)    | NO             | Indicates whether the error is an error or a warning.                                                                           |
| MESSAGE_NUMBER | NUMBER         | NO             | The error code.        |
