# DBA_TABLESPACES

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

This view displays all tablespaces in the current tenant.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view is available only in Oracle mode. </p>
  </main>

## Sample query

Query all tablespaces in the current tenant.

```shell
obclient [SYS]> SELECT TABLESPACE_NAME, ENCRYPTED FROM SYS.DBA_TABLESPACES;
```

The query result is as follows:

```shell
+-----------------+-----------+
| TABLESPACE_NAME | ENCRYPTED |
+-----------------+-----------+
| SECTEST_TS1     | YES       |
| TS2             | NO        |
+-----------------+-----------+
2 rows in set
```
