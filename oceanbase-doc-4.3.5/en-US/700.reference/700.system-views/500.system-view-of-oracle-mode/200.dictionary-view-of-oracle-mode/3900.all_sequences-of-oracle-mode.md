# ALL_SEQUENCES

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view was introduced in V2.2.77. </p>
</main>

## Purpose

Lists all sequences accessible to the user.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

|   **Column**   |    **Type**     | **Nullable?** |         **Description**          |
|--------------|---------------|-----------|-------------------------|
| OWNER        | VARCHAR2(128) | NO        | The owner of the object.                |
| OBJECT_NAME  | VARCHAR2(128) | NO        | The name of the SEQUENCE.              |
| MIN_VALUE    | NUMBER(28)    | NO        | The minimum value (inclusive) that the SEQUENCE can take.      |
| MAX_VALUE    | NUMBER(28)    | NO        | The maximum value (inclusive) that the SEQUENCE can take.      |
| INCREMENT_BY | NUMBER(28)    | NO        | The step length for each value, which can be a positive or negative integer, but not 0. |
| CYCLE_FLAG   | CHAR(1)       | NO        | Whether the SEQUENCE can cycle.                |
| ORDER_FLAG   | CHAR(1)       | NO        | Whether the values are globally ordered.              |
| CACHE_SIZE   | NUMBER(28)    | NO        | The number of values cached on each node.              |
| LAST_NUMBER  | NUMBER(38)    | NO        | The value to be assigned to the next node.           |

## Sample query

Query the details of all sequences that the current user can access.

```shell
obclient [SYS]> SELECT * FROM SYS.ALL_SEQUENCES;
```

The query result is as follows:

```shell
+----------------+---------------+-----------+------------------------------+--------------+------------+------------+------------+-------------+
| SEQUENCE_OWNER | SEQUENCE_NAME | MIN_VALUE | MAX_VALUE                    | INCREMENT_BY | CYCLE_FLAG | ORDER_FLAG | CACHE_SIZE | LAST_NUMBER |
+----------------+---------------+-----------+------------------------------+--------------+------------+------------+------------+-------------+
| SYS            | SEQ_T1        |         1 | 9999999999999999999999999999 |            1 | N          | N          |         50 |       10050 |
+----------------+---------------+-----------+------------------------------+--------------+------------+------------+------------+-------------+
1 row in set
```
