# USER_SUBPART_KEY_COLUMNS

## Purpose

The view displays the columns that are part of the subpartitioning key for all partitioned objects owned by the user.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## References

* DBA_SUBPART_KEY_COLUMNS

* ALL_SUBPART_KEY_COLUMNS

## Columns

|      **Column**      |     **Type**     | **Nullable?** |                                                              **Description**                                                              |
|--------------------|----------------|----------------|----------------------------------------------------------------------------------------------------------------------------------|
| NAME               | VARCHAR2(128)  | NO             | The name of the partitioned table or partitioned index.                                   |
| OBJECT_TYPE        | CHAR(5)        | NO             | The type of the partitioned object: <li> TABLE   <li> INDEX    |
| COLUMN_NAME        | VARCHAR2(4000) | NO             | The name of the column.                                          |
| COLUMN_POSITION    | NUMBER         | NO             | The position of the column in the subpartitioning key.                                     |
| COLLATED_COLUMN_ID | NUMBER         | NO             | Currently, this field is not supported and is set to NULL by default.                       |
