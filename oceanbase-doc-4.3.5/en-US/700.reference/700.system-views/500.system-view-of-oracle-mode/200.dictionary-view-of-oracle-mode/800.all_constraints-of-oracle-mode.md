# ALL_CONSTRAINTS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

View the constraint information of all tables accessible to the user.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

|     **Column**      |     **Type**     | **Nullable?** |                      **Description**                      |
|-------------------|----------------|----------------|--------------------------------------------------|
| OWNER             | VARCHAR2(128)  | NO             | The owner of the constraint.                                           |
| CONSTRAINT_NAME   | VARCHAR2(128)  | NO             | The name of the constraint.                                             |
| CONSTRAINT_TYPE   | VARCHAR2(1)    | NO             | The type of the index. Currently, only the NORMAL type is supported.                             |
| TABLE_NAME        | VARCHAR2(128)  | NO             | The name of the table where the constraint is located.                                        |
| SEARCH_CONDITION  | VARCHAR2(4000) | NO             | The text of the search condition of a check constraint.                    |
| R_OWNER           | VARCHAR2(128)  | NO             | The owner of the referenced constraint.                                        |
| R_CONSTRAINT_NAME | VARCHAR2(128)  | NO             | The name of the referenced constraint.                                         |
| DELETE_RULE       | VARCHAR2(9)    | NO             | The cascading delete rule of a foreign key constraint.                                      |
| STATUS            | VARCHAR2(8)    | NO             | Whether the constraint is ENABLE or DISABLE.                          |
| DEFERRABLE        | VARCHAR2(14)   | NO             | Whether the constraint is DEFERRABLE or NOT DEFERRABLE.                |
| DEFERRED          | VARCHAR2(9)    | NO             | Whether the constraint is DEFERRED or IMMEDIATE.                      |
| VALIDATED         | VARCHAR2(13)   | NO             | Whether the constraint is VALIDATED or NOT VALIDATED. |
| GENERATED         | VARCHAR2(14)   | NO             | Whether the name of the constraint is USER NAME or GENERATED NAME.             |
| BAD               | VARCHAR2(3)    | NO             | This column is not supported. The default value is NULL.                         |
| RELY              | VARCHAR2(4)    | NO             | Whether the constraint is RELY.                                  |
| LAST_CHANGE       | DATE           | NO             | The time when the constraint was last enabled or disabled.                   |
| INDEX_OWNER       | VARCHAR2(128)  | NO             | The owner of the index on which the constraint is located.                                        |
| INDEX_NAME        | VARCHAR2(128)  | NO             | The name of the index on which the constraint is located.                                         |
| INVALID           | VARCHAR2(7)    | NO             | Whether the constraint is INVALIDATE.                            |
| VIEW_RELATED      | VARCHAR2(14)   | NO             | Whether the constraint is created on a view.                                      |
