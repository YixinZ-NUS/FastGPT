# STMT_AUDIT_OPTION_MAP

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

Records the type, name, and priority of a statement-level audit option.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

| **Column** |    **Type**     | **Nullable?** |               **Description**                |
|----------|---------------|----------------|-------------------------------------|
| OPTION#  | NUMBER(38)    | NO             | The type ID of the statement-level audit option. |
| NAME     | VARCHAR2(128) | NO             | The name of the statement-level audit option. |
| PROPERTY | NUMBER        | NO             | The priority of the statement-level audit option. This field is not supported at present. The default value is NULL. |
