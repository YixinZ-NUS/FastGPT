# NLS_INSTANCE_PARAMETERS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

This view displays the NLS parameters at the instance (tenant) level.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view is applicable only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

| **Column** |     **Type**      | **Nullable** | **Description** |
|-----------|-----------------|----------------|--------|
| PARAMETER | VARCHAR2(128)   | NO             | The name of the parameter.   |
| VALUE     | VARCHAR2(65536) | YES            | The value of the parameter.   |

## Sample query

Query the NLS parameters of the current tenant.

```shell
obclient> SELECT * FROM SYS.NLS_INSTANCE_PARAMETERS;
```

The query result is as follows:

```shell
+-------------------------+----------------------------------+
| PARAMETER               | VALUE                            |
+-------------------------+----------------------------------+
| NLS_DATE_FORMAT         | YYYY-MM-DD HH24:MI:SS            |
| NLS_TIMESTAMP_FORMAT    | YYYY-MM-DD HH24:MI:SS.FF         |
| NLS_TIMESTAMP_TZ_FORMAT | YYYY-MM-DD HH24:MI:SS.FF TZR TZD |
| NLS_TERRITORY           | AMERICA                          |

| NLS_COMP                | BINARY                           |

13 rows in set
```
