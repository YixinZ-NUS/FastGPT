# DBA_RSRC_PLAN_DIRECTIVES

<main id="notice" type='explain'>
<h4>Note</h4>
<ul><li>This view was introduced in OceanBase Database V3.1.2. </li>
<li>It is recommended to use the <a href="15020.dba_ob_rsrc_directives-of-oracle-mode.md">DBA_OB_RSRC_DIRECTIVES</a> view to view the resource configuration parameters of a tenant in the Resource Manager. </li></ul>
</main>

## Purpose

The `DBA_RSRC_PLAN_DIRECTIVES` view records the details of resource plan directives in the database.
