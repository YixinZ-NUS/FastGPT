# USER_OBJECTS

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view was introduced in V2.2.77. </p>
</main>

## Purpose

The `USER_OBJECTS` view displays all database objects owned by the user. It is similar to the ALL_OBJECTS view, except that the OWNER column is not included because the owner is the user itself.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

|   Column  |    Type  | Nullable? |     Description |
|-------------------|---------------|------------|-------------|
| OBJECT_NAME    | VARCHAR2(128) | NO             | Name of the object.                                         |
| SUBOBJECT_NAME | VARCHAR2(128) | NO             | Name of the subobject (for example, partition).                          |
| OBJECT_ID      | NUMBER        | NO             | Dictionary ID of the object.                                   |
| DATA_OBJECT_ID | NUMBER        | NO             | Dictionary ID of the segment that contains the object.                       |
| OBJECT_TYPE    | VARCHAR2(23)  | NO             | Type of the object. Valid values include table, index, partition, package, stored procedure, synonym, and sequence.                |
| CREATED        | DATE          | NO             | Creation time of the object.                                       |
| LAST_DDL_TIME  | DATE          | NO             | Last DDL time of the object.                                |
| TIMESTAMP      | VARCHAR2(19)  | NO             | Timestamp of the object in character format.                             |
| STATUS         | VARCHAR2(7)   | NO             | Status of the object. Valid values include <li>VALID: valid<li> INVALID: invalid     |
| TEMPORARY      | VARCHAR2(1)   | NO             | Indicates whether the object is a temporary object.                                 |
| GENERATED      | VARCHAR2(1)   | NO             | Indicates whether the name of the object is generated by the system.                             |
| SECONDARY      | VARCHAR2(1)   | NO             | Reserved.                                      |
| NAMESPACE      | NUMBER        | NO             | Namespace of the object.                                      |

| SHARING           | VARCHAR2(18)  | NO         | Default value: `NULL`                    |
| EDITIONABLE       | VARCHAR2(1)   | NO         | Default value: `NULL`                    |
| ORACLE_MAINTAINED | VARCHAR2(1)   | NO         | Default value: `NULL`                            |
| APPLICATION       | VARCHAR2(1)   | NO         | Default value: `NULL`                        |
| DEFAULT_COLLATION | VARCHAR2(1)   | NO         | Default value: `NULL`                            |
| DUPLICATED        | VARCHAR2(1)   | NO         | Default value: `NULL`         |
| SHARDED           | VARCHAR2(1)   | NO         | Default value: `NULL`                             |
| IMPORTED_OBJECT   | VARCHAR2(1)   | NO         | Default value: `NULL`                           |
| CREATED_APPID     | NUMBER    | NO         | Default value: `NULL`    |
| CREATED_VSNID     | NUMBER    | NO         | Default value: `NULL`          |
| MODIFIED_APPID    | NUMBER    | NO         | Default value: `NULL`      |
| MODIFIED_VSNID    | NUMBER    | NO         | Default value: `NULL`        |

## Sample query

View the first two records of all database objects owned by the current user.

```shell
obclient [SYS]> SELECT * FROM SYS.USER_OBJECTS WHERE ROWNUM <= 2;
```
