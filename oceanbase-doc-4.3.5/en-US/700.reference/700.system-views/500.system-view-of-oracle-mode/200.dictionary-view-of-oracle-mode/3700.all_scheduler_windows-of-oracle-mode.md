# ALL_SCHEDULER_WINDOWS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V4.0.0. </p>
</main>

## Purpose

This view displays information about all scheduler windows that the current user can access.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

| Column | Type | Nullable? | Description |
|-------|------|----------|-------------|
| OWNER | VARCHAR2(128) | NO | The owner of the scheduler window. |
| WINDOW_NAME | VARCHAR2(128) | NO | The name of the scheduler window. |
| RESOURCE_PLAN | VARCHAR2(128) | NO | The name of the resource plan associated with the window. This field is currently not used. |
| SCHEDULE_OWNER | VARCHAR2(4000) | NO | The owner of the window schedule. This field is currently not used. |
| SCHEDULE_NAME | VARCHAR2(4000) | NO | The name of the window schedule. This field is currently not used. |
| SCHEDULE_TYPE | VARCHAR2(8) | NO | The type of the window schedule. Valid values include:<ul><li>ONCE: the repeat interval is NULL</li><li>NAMED: named schedule</li><li>CALENDAR: calendar expression used as a schedule</li></ul>This field is currently not used. |
| START_DATE | TIMESTAMP(6) WITH TIME ZONE | YES | The start time of the window. |
| REPEAT_INTERVAL | VARCHAR2(4000) | YES | The repeat interval of the window. |
| END_DATE | TIMESTAMP(6) WITH TIME ZONE | YES | The end time of the window. |
| DURATION | INTERVAL DAY (3) TO SECOND (0) | YES | The duration of the window. |
| WINDOW_PRIORITY | VARCHAR2(4) | NO | The priority of the job relative to other windows. Valid values include:<ul><li>HIGH</li><li>LOW</li></ul>This field is currently not used. |
| NEXT_RUN_DATE | TIMESTAMP(6) WITH TIME ZONE | NO | The start time of the next window. |
| LAST_START_DATE | TIMESTAMP(6) WITH TIME ZONE | YES | The start time of the last window. |
| ENABLED | VARCHAR2(5) | YES | Indicates whether the window is enabled (TRUE) or disabled (FALSE). |
| ACTIVE | VARCHAR2(5) | NO | Indicates whether the window is open (TRUE) or closed (FALSE). This field is currently not used. |
| MANUAL_OPEN_TIME | TIMESTAMP(6) WITH TIME ZONE | NO | The time when the window was manually opened. This field is NULL if the window was not manually opened. This field is currently not used. |
| MANUAL_DURATION | INTERVAL DAY (3) TO SECOND (0) | NO | The duration of the manually opened window. This field is NULL if the window was not manually opened. This field is currently not used. |
| COMMENTS | VARCHAR2(4000) | YES | The description of the window. |
