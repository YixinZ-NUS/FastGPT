# DBA_PLSQL_TYPES

<main id="notice" >
    <h4>Note</h4>
    <p>This view is available starting with V4.3.5 for V4.3.x versions and V4.2.5 for V4.2.x versions. </p>
  </main>

## Purpose

`DBA_PLSQL_TYPES` displays information about package types of the current user tenant. A package type can be displayed in this view only after the package is successfully compiled.

<main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

| **Column** | **Type** | **Nullable** | **Description** |
|---------------------------|---|---|--------------------------------------------|
| OWNER | VARCHAR2(128) | YES | Type |
| TYPE_NAME      | VARCHAR2(136) | NO   | The name of a PL/SQL type. |
| PACKAGE_NAME   | VARCHAR2(128) | YES  | The name of the package to which the PL/SQL type belongs. |
| Column | Type | Nullable | Description | | --- | --- | --- | --- | | TYPE_OID | RAW(16) | NO | The object identifier of the PL/SQL type. |
| Column | Type | Nullable | Description | | --- | --- | --- | --- | | TYPECODE | VARCHAR2(58)  | NO   | The code of the PL/SQL type. |
| ATTRIBUTES     | NUMBER(38)    | NO   | The number of attributes included in the PL/SQL type. |
| CONTAINS_PLSQL | VARCHAR2(3)   | NO   | Indicates whether the PL/SQL type contains PL/SQL code. |

## Sample query

Query the overall information of package types in the current tenant.

```shell
obclient> SELECT * FROM DBA_PLSQL_TYPES;
```
