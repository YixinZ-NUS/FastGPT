# DBA_RSRC_PLANS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is introduced in V3.1.2. </p>
</main>

## Purpose

The `DBA_RSRC_PLANS` view records all system-defined resource management plans.

## Columns

|           Column            |       Type       | Nullable? |                         Description                         |
|---------------------------|----------------|------------|----------------------------------------------------|
| PLAN_ID                   | NUMBER         | NO         | The ID of the resource plan.                                          |
| PLAN                      | VARCHAR2(128)  | NO         | The name of the resource plan.                                             |
| NUM_PLAN_DIRECTIVES       | NUMBER         | NO         | Reserved for compatibility. The value is always `NULL`. |
| CPU_METHOD                | VARCHAR2(128)  | NO         | Reserved for compatibility. The value is always `NULL`.                               |
| MGMT_METHOD               | VARCHAR2(128)  | NO         | Reserved for compatibility. The value is always `NULL`.                               |
| ACTIVE_SESS_POOL_MTH      | VARCHAR2(128)  | NO         | Reserved for compatibility. The value is always `NULL`.                               |
| PARALLEL_DEGREE_LIMIT_MTH | VARCHAR2(128)  | NO         | Reserved for compatibility. The value is always `NULL`.                               |
| QUEUING_MTH               | VARCHAR2(128)  | NO         | Reserved for compatibility. The value is always `NULL`.                               |
| SUB_PLAN                  | VARCHAR2(3)    | NO         | Reserved for compatibility. The value is always `NULL`.                               |
| COMMENTS                  | VARCHAR2(2000) | YES        | The remarks for the resource plan.                                         |
| STATUS                    | VARCHAR2(128)  | NO         | Reserved for compatibility. The value is always `NULL`.                               |
| MANDATORY                 | VARCHAR2(3)    | NO         | Reserved for compatibility. The value is always `NULL`.                               |

## Sample query

Query all resource management plans defined in the system.

```shell
obclient [SYS]> SELECT * FROM SYS.DBA_RSRC_PLANS;
```

The query result is as follows:

```shell
+---------+---------+---------------------+------------+-------------+----------------------+---------------------------+-------------+----------+----------+--------+-----------+
| PLAN_ID | PLAN    | NUM_PLAN_DIRECTIVES | CPU_METHOD | MGMT_METHOD | ACTIVE_SESS_POOL_MTH | PARALLEL_DEGREE_LIMIT_MTH | QUEUING_MTH | SUB_PLAN | COMMENTS | STATUS | MANDATORY |
+---------+---------+---------------------+------------+-------------+----------------------+---------------------------+-------------+----------+----------+--------+-----------+
|    NULL | daytime |                NULL | NULL       | NULL        | NULL                 | NULL                      | NULL        | NULL     | TPFirst  | NULL   | NULL      |
+---------+---------+---------------------+------------+-------------+----------------------+---------------------------+-------------+----------+----------+--------+-----------+
1 row in set
```
