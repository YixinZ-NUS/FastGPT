# USER_TAB_SUBPARTITIONS

## Purpose

Describes the subpartition names, table names, partitions to which the subpartitions belong, and storage attributes of all subpartitions owned by the current user.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>
