# ALL_USERS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

The `ALL_USERS` view displays all database users visible to the current user.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view is available only in OceanBase Database Enterprise Edition. </p>
  </main>

## Columns

| Column | Type | Nullable? | Description |
|----------|--------------|----------------|---------|
| USERNAME | VARCHAR2(30) | NO             | The name of the user. |
| USERID   | NUMBER(38)   | NO             | The ID of the user. |
| CREATED  | DATE         | NO             | The date when the user was created. |
| COMMON                      | VARCHAR2(3)    | NO             | Indicates whether the user is a common user.<main id="notice" type='explain'><h4>Note</h4><p>This column is available in OceanBase Database V4.2.1 BP2 and later. </p></main>|
| ORACLE_MAINTAINED           | VARCHAR2(1)    | NO             | Indicates whether the user was created and maintained by using a script provided by OceanBase Database in Oracle mode.<main id="notice" type='explain'><h4>Note</h4><p>This column is available in OceanBase Database V4.2.1 BP2 and later. </p></main>|
| INHERITED                   | VARCHAR2(3)    | NO             | Indicates whether the user is inherited from other containers.<main id="notice" type='explain'><h4>Note</h4><p>This column is available in OceanBase Database V4.2.1 BP2 and later. </p></main> |
| DEFAULT_COLLATION           | VARCHAR2(100)  | NO             | The default character set.<main id="notice" type='explain'><h4>Note</h4><p>This column is available in OceanBase Database V4.2.1 BP2 and later. </p></main> |
| IMPLICIT                    | VARCHAR2(3)    | NO             | Indicates whether the user was created by an implicit application.<main id="notice" type='explain'><h4>Note</h4><p>This column is available in OceanBase Database V4.2.1 BP2 and later. </p></main> |
| ALL_SHARD                   | VARCHAR2(3)    | NO             | Indicates whether the user was created with sharding enabled.<main id="notice" type='explain'><h4>Note</h4><p>This column is available in OceanBase Database V4.2.1 BP2 and later. </p></main> |

## Sample query

Query all database users visible to the current user.

```shell
obclient [SYS]> SELECT * FROM SYS.ALL_USERS\G
```
