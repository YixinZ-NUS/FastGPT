# ALL_OBJECTS

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This view was introduced in OceanBase Database V2.2.77. </p>
</main>

## Purpose

The `ALL_OBJECTS` view displays all database objects that the user has access to.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

| Column |  Type  | Nullable? | Description |
|-------------------|---------------|------------|-------------|
| OWNER          | VARCHAR2(128) | NO             | The owner of the object.                                     |
| OBJECT_NAME    | VARCHAR2(128) | NO             | The name of the object.                                         |
| SUBOBJECT_NAME | VARCHAR2(128) | NO             | The name of a subobject (for example, a partition).                          |
| OBJECT_ID      | NUMBER    | NO             | The dictionary ID of the object.                                     |
| DATA_OBJECT_ID | NUMBER        | NO             | The dictionary ID of the segment that contains the object.                       |
| OBJECT_TYPE    | VARCHAR2(23)  | NO             | The type of the object, which can be a table, index, partition, package, stored procedure, synonym, or sequence.                |
| CREATED        | DATE          | NO             | The time when the object was created.                                       |
| LAST_DDL_TIME  | DATE          | NO             | The time when the object was last modified by DDL.                                |
| TIMESTAMP      | VARCHAR2(19) | NO             | The timestamp of the object in character format.                             |
| STATUS         | VARCHAR2(7)   | NO             | The status of the object. <li>VALID: The object is valid.<li> INVALID: The object is invalid.    |
| TEMPORARY      | VARCHAR2(1)   | NO             | Indicates whether the object is a temporary object.                                 |
| GENERATED      | VARCHAR2(1)   | NO             | Indicates whether the name of the object is generated by the system.                             |
| SECONDARY      | VARCHAR2(1)   | NO             | Not used.                                         |
| NAMESPACE      | NUMBER        | NO             | The namespace of the object.                                      |
| EDITION_NAME   | VARCHAR2(128) | NO             | Reserved.                                         |
| SHARING           | VARCHAR2(18)  | NO         | The default value is `NULL`.                             |
| EDITIONABLE       | VARCHAR2(1)   | NO         | The default value is `NULL`.                              |
| ORACLE_MAINTAINED | VARCHAR2(1)   | NO         | The default value is `NULL`.                              |
| APPLICATION       | VARCHAR2(1)   | NO         | The default value is `NULL`.                             |
| DEFAULT_COLLATION | VARCHAR2(1)   | NO         | The default value is `NULL`.                          |
| DUPLICATED        | VARCHAR2(1)   | NO         | The default value is `NULL`.          |
| SHARDED           | VARCHAR2(1)   | NO         | The default value is `NULL`.                             |
| IMPORTED_OBJECT   | VARCHAR2(1)   | NO         | The default value is `NULL`.                              |
| CREATED_APPID     | NUMBER    | NO         | The default value is `NULL`.        |
| CREATED_VSNID     | NUMBER    | NO         | The default value is `NULL`.            |
| MODIFIED_APPID    | NUMBER    | NO         | The default value is `NULL`.       |
| MODIFIED_VSNID    | NUMBER   | NO         | The default value is `NULL`.       |
