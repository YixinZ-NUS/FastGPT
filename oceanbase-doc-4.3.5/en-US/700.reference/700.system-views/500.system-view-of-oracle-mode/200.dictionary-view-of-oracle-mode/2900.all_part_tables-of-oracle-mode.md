# ALL_PART_TABLES

## Purpose

This view displays the partition information of the partitions that the current user can access in partitioned tables.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view applies only to OceanBase Database in Oracle mode. </p>
  </main>
