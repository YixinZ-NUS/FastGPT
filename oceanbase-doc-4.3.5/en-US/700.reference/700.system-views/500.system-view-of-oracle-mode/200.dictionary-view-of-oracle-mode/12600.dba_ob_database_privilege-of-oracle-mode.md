# DBA_OB_DATABASE_PRIVILEGE

<main id="notice" type='explain'>
<h4>Instructions</h4>
<p>This view has been introduced since version V4.0.0.</p>
</main>

## Purpose Description

Display the permissions owned by the current tenant to control the current user.

## Columns

| Column | Type | Nullable? | Description |
|-------------------|-----------------------------------|------|-----|
| USER_ID             | NUMBER(38)                       | NO       | The user ID.                   |
| USERNAME            | VARCHAR2(128)                    | NO       | The username.                  |
| DATABASE_NAME       | VARCHAR2(128)                    | NO       | The current database name associated with the user. |
| GMT_CREATE          | TIMESTAMP(6) WITH LOCAL TIME ZONE | NO       | The creation time.             |
| GMT_MODIFIED        | TIMESTAMP(6) WITH LOCAL TIME ZONE | NO       | The modification time.         |
| PRIV_ALTER          | VARCHAR2(3)                      | NO       | Whether the privilege to modify tables exists.|
| PRIV_CREATE         | VARCHAR2(3)                      | NO       | Whether the privilege to create tables exists.|
| PRIV_DELETE         | VARCHAR2(3)                      | NO       | Whether the privilege to delete table records exists.|
| PRIV_DROP           | VARCHAR2(3)                      | NO       | Whether the privilege to drop tables exists.  |
| PRIV_GRANT_OPTION   | VARCHAR2(3)                      | NO       | Whether the privilege to grant other privileges exists.|
| PRIV_INSERT         | VARCHAR2(3)                      | NO       | Whether the privilege to insert records exists.|
| PRIV_UPDATE         | VARCHAR2(3)                      | NO       | Whether the privilege to update records exists.|
| PRIV_SELECT         | VARCHAR2(3)                      | NO       | Whether the privilege to query records exists.|
| PRIV_INDEX          | VARCHAR2(3)                      | NO       | Whether the privilege to add indexes to tables exists.|
| PRIV_CREATE_VIEW    | VARCHAR2(3)                      | NO       | Whether the privilege to create views exists. |
| PRIV_SHOW_VIEW      | VARCHAR2(3)                      | NO       | Whether the privilege to view existing views exists.|

## Sample query

A user tenant views the privileges within the current tenant that control the current user's access.

```shell
obclient [SYS]> SELECT * FROM SYS.DBA_OB_DATABASE_PRIVILEGE;
```
