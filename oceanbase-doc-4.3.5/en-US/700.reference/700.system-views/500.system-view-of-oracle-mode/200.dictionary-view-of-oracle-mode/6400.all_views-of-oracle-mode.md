# ALL_VIEWS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.30. </p>
</main>

## Purpose

This view displays the views accessible to the current user.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view is applicable only to OceanBase Database in Oracle mode. </p>
  </main>

## Columns

|    **Column**     |     **Type**     | **Nullable** |                    **Description**                    |
|-----------------|----------------|----------------|----------------------------------------------|
| OWNER           | VARCHAR2(128)  | NO             | The user to which the view belongs.                                |
| VIEW_NAME       | VARCHAR2(128)  | NO             | The name of the view.                                         |
| TEXT_LENGTH     | NUMBER         | NO             | The length of the view definition.                                      |
| TEXT            | VARCHAR2(4000) | NO             | The definition of the view.                                         |
| OID_TEXT_LENGTH | NUMBER         | NO             | This column is not supported. The default value is NULL. |
| OID_TEXT        | VARCHAR2(4000) | NO             | This column is not supported. The default value is NULL.     |
| VIEW_TYPE       | VARCHAR2(30)   | NO             | This column is not supported. The default value is NULL.     |
| SUPERVIEW_NAME  | VARCHAR2(30)   | NO             | This column is not supported. The default value is NULL.              |
| EDITIONING_VIEW | VARCHAR2(1)    | NO             | This column is not supported. The default value is NULL.                                    |
| READ_ONLY       | VARCHAR2(1)    | NO             | This column is not supported. The default value is NULL.             |
| CONTAINER_DATA  | VARCHAR2(1)    | NO             | This column is not supported. The default value is NULL. |
| BEQUEATH        | VARCHAR2(12)   | NO             | This column is not supported. The default value is NULL. |
| ORIGIN_CON_ID   | NUMBER         | NO             | This column is not supported. The default value is NULL. |
| DEFAULT_COLLATION      | VARCHAR2(100)     | NO   | This column is not supported. The default value is NULL. |
| CONTAINERS_DEFAULT     | VARCHAR2(3)       | NO   | This column is not supported. The default value is NULL. |
| CONTAINER_MAP          | VARCHAR2(2)       | NO   | This column is not supported. The default value is NULL. |
| EXTENDED_DATA_LINK     | VARCHAR2(3)       | NO   | This column is not supported. The default value is NULL. |
| EXTENDED_DATA_LINK_MAP | VARCHAR2(3)       | NO   | This column is not supported. The default value is NULL. |
| HAS_SENSITIVE_COLUMN   | VARCHAR2(3)       | NO   | This column is not supported. The default value is NULL. |
| ADMIT_NULL             | VARCHAR2(3)       | NO   | This column is not supported. The default value is NULL. |
| PDB_LOCAL_ONLY         | VARCHAR2(3)       | NO   | This column is not supported. The default value is NULL. |

## Sample query

Query the information of the view `VIEW2` from all views accessible to the current user.

```shell
obclient [SYS]> SELECT OWNER, VIEW_NAME, TEXT_LENGTH, TEXT FROM SYS.DBA_VIEWS WHERE VIEW_NAME='VIEW2';
```

The query result is as follows:

```shell
+-------+-----------+-------------+--------------------------------------------+
| OWNER | VIEW_NAME | TEXT_LENGTH | TEXT                                       |
+-------+-----------+-------------+--------------------------------------------+
| SYS   | VIEW2     |          42 | SELECT "A","B" FROM tbl2 WHERE tbl2.b > 1  |
+-------+-----------+-------------+--------------------------------------------+
1 row in set
```
