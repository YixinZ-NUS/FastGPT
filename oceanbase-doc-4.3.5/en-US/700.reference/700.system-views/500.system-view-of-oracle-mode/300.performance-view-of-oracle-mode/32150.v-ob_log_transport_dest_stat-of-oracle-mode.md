# V$OB_LOG_TRANSPORT_DEST_STAT

<main id="notice" type='explain'>
  <h4>Note</h4>
  <ul><li>For OceanBase Database V4.3.x, this view is introduced in V4.3.5. </li> <li>For OceanBase Database V4.2.x, this view is introduced in V4.2.5. </li></ul>
</main>

## Purpose

The `V$OB_LOG_TRANSPORT_DEST_STAT` view displays information about downstream log synchronization links (OBCDCs or network standby databases) of this OBServer of this tenant.

## Columns

| **Column** | **Type** | **Nullable?** | **Description** |
| -------- | -------- | ------------------ | -------- |
| TENANT_ID                   | NUMBER(38)                        | NO   | Tenant ID     |
| SVR_IP                      | VARCHAR2(46)                      | NO   | IP address of the server     |
| SVR_PORT                    | NUMBER(38)                        | NO   | RPC_PORT of the server     |
| LS_ID                       | NUMBER(38)                        | NO   | ID of the requested log stream     |
| CLIENT_IP                   | VARCHAR2(46)                      | NO   | IP address of the client machine     |
| CLIENT_PID                  | NUMBER(38)                        | NO   | Process ID of the client     |
| CLIENT_TENANT_ID            | NUMBER(38)                        | NO   | Tenant ID of the client     |
| CLIENT_TYPE                 | VARCHAR2(7)                       | NO   | Type of the client:<ul><li>STANDBY: network standby database </li><li>CDC: OBCDC </li><li>UNKNOWN: unknown client type </li></ul>    |
| START_SERVE_TIME            | TIMESTAMP(6) WITH LOCAL TIME ZONE | NO   | Time when the record was created     |
| LAST_SERVE_TIME             | TIMESTAMP(6) WITH LOCAL TIME ZONE | NO   | Time when the record was last modified     |
| LAST_READ_SOURCE            | VARCHAR2(7)          | NO   | Source of the log read during the last request:<ul><li>ONLINE: online logs </li><li>ARCHIVE: archived logs </li><li>UNKNOWN: unknown source </li></ul>     |
| LAST_REQUEST_TYPE           | VARCHAR2(24)         | NO   | RPC type of the last request:<ul><li>0: SEQUENTIAL_READ_SERIAL, which indicates that the serial log transfer protocol is used to read logs sequentially. </li><li>1: SEQUENTIAL_READ_PARALLEL, which indicates that the parallel log transfer protocol is used to read logs sequentially. </li><li>2: SCATTERED_READ, which indicates that the last request was to read scattered logs. </li><li>Other values: indicates an unknown RPC type. </li></ul>     |
| LAST_REQUEST_LOG_LSN        | NUMBER(38)                        | NO   | LSN of the last requested log     |
| LAST_REQUEST_LOG_SCN        | NUMBER(38)                        | NO   | SCN of the last requested log     |
| LAST_FAILED_REQUEST         | CLOB                              | NO   | Information about the last failed RPC, including the time of sending, TRACE_ID, and error code.    |
| AVG_REQUEST_PROCESS_TIME    | NUMBER(38)                        | NO   | Average RPC processing time     |
| AVG_REQUEST_QUEUE_TIME      | NUMBER(38)                        | NO   | Average RPC queuing time     |
| AVG_REQUEST_READ_LOG_TIME   | NUMBER(38)                        | NO   | Average RPC log reading time     |
| AVG_REQUEST_READ_LOG_SIZE   | NUMBER(38)                        | NO   | Average RPC log reading size     |
| AVG_LOG_TRANSPORT_BANDWIDTH | VARCHAR2(44)                      | NO   | Average log transfer bandwidth     |
