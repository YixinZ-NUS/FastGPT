# GV$DBLINK

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is not supported. To view the created DBLink, use <a href="../200.dictionary-view-of-oracle-mode/7900.dba_db_links.md">DBA_DB_LINKS</a>, <a href="../200.dictionary-view-of-oracle-mode/900.all_db_links.md">ALL_DB_LINKS</a>, or <a href="../200.dictionary-view-of-oracle-mode/26000.user_db_links.md">USER_DB_LINKS</a>. </p>
</main>

## Purpose

This view displays all database links ( `IN_TRANSACTION=YES` ) opened after a query session is started on all nodes. These database links must be committed or rolled back before they are closed.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view is available only in Oracle mode of OceanBase Database. </p>
  </main>
