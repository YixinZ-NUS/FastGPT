# V$TIMEZONE_NAMES

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.77. </p>
</main>

## **Description**

This view displays the names of valid time zones.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view is available starting with OceanBase Database in Oracle mode. </p>
  </main>

## Columns

| **Column** |    **Type**     | **Nullable** |     **Description**     |
|----------|---------------|----------------|----------------|
| TZNAME   | varchar(64)   | NO             | Time zone region           |
| TZABBREV | varchar(8)    | NO             | Relative time abbreviation (for example, PDT) |
| CON_ID   | decimal(38,0) | NO             | Tenant ID          |
