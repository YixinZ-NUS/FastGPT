# V$DML_STATS

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is introduced since OceanBase Database V4.0.0. </p>
</main>

## Purpose

This view displays the DML information of the current tenant's user table on the current OBServer node.

## Columns

| Column | Type | Nullable? | Description |
| --- | --- | --- | --- |
| SVR_IP | VARCHAR2(46) | NO | The IP address of the server. |
| SVR_PORT | NUMBER | NO | The port number of the server. |
| INST_ID | NUMBER | NO | The ID of the tenant. |
| OBJN | NUMBER | NO | The ID of the object. |
| INS | NUMBER | NO | The approximate number of insert operations performed since the statistics were last collected. |
| UPD | NUMBER | NO | The approximate number of update operations performed since the statistics were last collected. |
| DEL | NUMBER | NO | The approximate number of delete operations performed since the statistics were last collected. |
| DROPSEG | NUMBER | NO | Not supported. The value of this field is NULL. |
| CURROWS | NUMBER | NO | Not supported. The value of this field is NULL. |
| PAROBJN | NUMBER | NO | Not supported. The value of this field is NULL. |
| LASTUSED | NUMBER | NO | Not supported. The value of this field is NULL. |
| FLAGS | NUMBER | NO | Not supported. The value of this field is NULL. |
| CON_ID | NUMBER | NO | Not supported. The value of this field is NULL. |

## Sample query

View the DML information of a user table in the current OBServer node of the current tenant.

```shell
obclient [SYS]> SELECT * FROM SYS.V$DML_STATS;
```
