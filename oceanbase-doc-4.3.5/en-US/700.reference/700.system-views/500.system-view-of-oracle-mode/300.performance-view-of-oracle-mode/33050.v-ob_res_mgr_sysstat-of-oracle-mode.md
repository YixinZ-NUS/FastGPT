# V$OB_RES_MGR_SYSSTAT

<main id="notice" type='explain'>
 <h4>Note</h4>
 <ul><li>The view was introduced in OceanBase Database V4.3.5 for V4.3.x. </li> <li>The view was introduced in OceanBase Database V4.2.5 for V4.2.x. </li></ul>
</main>

## Purpose

The `V$OB_RES_MGR_SYSSTAT` view displays the statistics of resource groups in the current tenant on the current OBServer node.

## Columns

| **Column** | **Type**  | **Nullable?** | **Description**                               |
|------------|-------------|---------------------|----------------------------------------|
| CON_ID     | NUMBER       | NO   | Tenant ID |
| GROUP_ID   | NUMBER       | NO   | Resource group ID |
| SVR_IP     | VARCHAR2(46) | NO   | IP address of the node |
| SVR_PORT   | NUMBER(38)   | NO   | Port of the node |
| STATISTIC# | NUMBER       | NO   | Statistics ID |
| NAME       | VARCHAR2(64) | NO   | Statistics name |
| CLASS      | NUMBER       | NO   | Statistics category |
| VALUE      | NUMBER       | NO   | Statistics value |
| VALUE_TYPE | VARCHAR2(16) | NO   | Value type: cumulative or assignment |
| STAT_ID    | NUMBER       | NO   | Statistics ID |

## Sample query

The following example queries the statistics of resource groups on the current OBServer node in the current tenant.

```shell
obclient [SYS]> SELECT * FROM SYS.V$OB_RES_MGR_SYSSTAT  WHERE ROWNUM < 10;
```
