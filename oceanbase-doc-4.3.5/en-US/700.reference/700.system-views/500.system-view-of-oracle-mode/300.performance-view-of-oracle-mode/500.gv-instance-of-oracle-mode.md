# GV$INSTANCE

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This view is available starting with V2.2.77. </p>
</main>

## Purpose

This view displays the status of the current database instance.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This view is available only in Oracle mode of OceanBase Database. </p>
  </main>

## Columns

|   **Column**   |    **Type**  | **Nullable** |        **Description**                     |
|------------------|--------------|---------------|-----------------------------------------|
| INST_ID          | NUMBER       | NO            | The ID of the database instance. |
| INSTANCE_NUMBER  | NUMBER       | NO            | This column is not supported and is NULL by default. |
| INSTANCE_NAME    | VARCHAR2(16) | NO            | This column is not supported and is NULL by default. |
| HOST_NAME        | VARCHAR2(64) | NO            | The IP address of the server where the instance resides. |
| VERSION          | VARCHAR2(17) | NO            | The version of the database instance. |
| STARTUP_TIME     | DATE         | NO            | This column is not supported and is NULL by default. |
| STATUS           | VARCHAR2(12) | NO            | This column is not supported and is NULL by default. |
| PARALLEL         | VARCHAR2(3)  | NO            | This column is not supported and is NULL by default. |
| THREAD#          | NUMBER       | NO            | This column is not supported and is NULL by default. |
| ARCHIVER         | VARCHAR2(7)  | NO            | This column is not supported and is NULL by default. |
| LOG_SWITCH_WAIT  | VARCHAR2(15) | NO            | This column is not supported and is NULL by default. |
| LOGINS           | VARCHAR2(10) | NO            | This column is not supported and is NULL by default. |
| SHUTDOWN_PENDING | VARCHAR2(3)  | NO            | This column is not supported and is NULL by default. |
| DATABASE_STATUS  | VARCHAR2(17) | NO            | The status of the database. |
| INSTANCE_ROLE    | VARCHAR2(18) | NO            | Reserved. The value is NULL by default. |
| ACTIVE_STATE     | VARCHAR2(9)  | NO            | Reserved. The value is NULL by default. |
| BLOCKED          | VARCHAR2(2)  | NO            | Reserved. The value is NULL by default. |
| CON_ID           | NUMBER       | NO            | Reserved. The value is NULL by default. |
| INSTANCE_MODE    | VARCHAR2(11) | NO            | Reserved. The value is NULL by default. |
| EDITION          | VARCHAR2(7)  | NO            | Reserved. The value is NULL by default. |
| FAMILY           | VARCHAR2(80) | NO            | Reserved. The value is NULL by default. |
| DATABASE_TYPE    | VARCHAR2(15) | NO            | Reserved. The value is NULL by default. |
