# GV$OB_SQLSTAT

<main id="notice" type='explain'>
<h4>Note</h4>
<ul><li>In V4.3.x, this view was introduced in V4.3.5. </li> <li>In V4.2.x, this view was introduced in V4.2.2. </li></ul>
</main>

## Purpose

The `GV$OB_SQLSTAT` view displays the basic performance statistics of SQL statements in the cluster. Each row represents a SQL statement (that is, each unique combination of `SQL_ID` and `Plan_Hash` values). The columns with `_DELTA` in their names indicate the incremental statistics from the last WR snapshot to the current time.
