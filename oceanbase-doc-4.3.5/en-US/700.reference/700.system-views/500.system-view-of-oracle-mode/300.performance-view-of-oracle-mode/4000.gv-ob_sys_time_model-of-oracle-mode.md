# GV$OB_SYS_TIME_MODEL

<main id="notice" type='explain'>
<h4>Note</h4>
<ul><li>For V4.3.x, this view was introduced in V4.3.5. </li> <li>For V4.2.x, this view was introduced in V4.2.2. </li></ul>
</main>

## Purpose

The `GV$OB_SYS_TIME_MODEL` view displays the Time Model statistics at the tenant level in a cluster.

## Columns

| **Column** | **Type**  | **Nullable?** | **Description**                               |
|------------|-------------|---------------------|----------------------------------------|
| TENANT_ID  | NUMBER       | NO   | Tenant ID     |
| SVR_IP     | VARCHAR2(46) | NO   | Node IP       |
| SVR_PORT   | NUMBER(38)   | NO   | Node port        |
| STAT_ID    | NUMBER       | NO   | Stat ID         |
| STAT_NAME  | VARCHAR2(64) | NO   | Stat name       |
| VALUE      | NUMBER       | NO   | Stat value        |

The following table describes the statistics events in the `STAT_NAME` column and their corresponding IDs.

| Statistic event name | Description | Statistic event ID | Statistic event type |
| ------------|------|------------|----------------|
| DB time                                |  Total time of database activity, which is the sum of CPU time and non-idle wait time.  |  200001 |  3072 |
| DB CPU                                 |  CPU time of database activity.                    |  200002 |  3072 |
| background elapsed time                |  Total execution time of background processes.                           |  200005 |  3072 |
| background cpu time                    |  CPU time consumed by background processes.                        |  200006 |  3072 |
| non idle wait time                     |  Total time of non-idle wait. This is the total time when sessions are in a waiting state but not in an idle wait state.  |  200010 |  3072 |
| idle wait time                         |  Total time of idle wait. This is the total time when sessions are in an idle wait state.    |  200011 |  3072 |
| background database time               |  Total time of database operations executed by background processes.                   |  200012 |  3072 |
| background database non-idle wait time |  Total time of database operations executed by background processes in a non-idle wait state.    |  200013 |  3072 |
| background database idle wait time     |  Total time of database operations executed by background processes in an idle wait state.      |  200014 |  3072 |
| concurrency wait total time            | Total wait time due to concurrency issues, such as wait time due to resource lock. | 220001 |  3072 |
| user io wait total time                | Total wait time of user processes for I/O operations, such as reading data or writing data to disk. | 220002 |  3072 |
| application wait total time            | Total wait time generated by user application code, such as wait time due to row-level locking or explicit locking commands. | 2200013 |  3072 |

obclient [SYS]> SELECT * FROM SYS.GV$OB_SYS_TIME_MODEL;
