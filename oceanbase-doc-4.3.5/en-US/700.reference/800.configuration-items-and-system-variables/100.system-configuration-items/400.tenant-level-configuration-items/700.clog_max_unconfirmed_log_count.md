# clog_max_unconfirmed_log_count

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>In OceanBase Database V4.x, the corresponding logic is changed, and the corresponding feature is no longer available. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`clog_max_unconfirmed_log_count` specifies the maximum number of unconfirmed logs allowed in the transaction module.

## Attributes

| **Attribute** | **Description** |
|------------------|----------------|
| Type | Integer |
| Default value | 1500 |
| Value range | \[100, 50000\] |
| Effective upon OBServer node restart | No |
