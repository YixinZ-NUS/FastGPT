# ignore_replica_checksum_error

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This parameter may cause replica checksum verification to be ignored, potentially leading to significant errors. It has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`ignore_replica_checksum_error` specifies whether to ignore the checksum errors of replicas.

## Attributes

| **Attribute** | **Description** |
|------------------|-----------------------|
| Type | Boolean |
| Default value | False |
| Value range | <ul><li>True </li><li>False </li></ul> |
| Effective upon OBServer node restart | No. The setting takes effect immediately.     |
