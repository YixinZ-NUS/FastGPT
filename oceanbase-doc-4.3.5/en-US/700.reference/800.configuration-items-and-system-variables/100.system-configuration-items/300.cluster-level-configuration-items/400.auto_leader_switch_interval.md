# auto_leader_switch_interval

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the system architecture optimization of OceanBase Database V4.x, no refresh interval is required for leader election, and this parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>
