# merger_warm_up_duration_time

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the storage architecture upgrade of OceanBase Database V4.x, the feature corresponding to this parameter is no longer supported, and this parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`merger_warm_up_duration_time` specifies the preload time of new baseline data in a major compaction.

## Attributes

| **Attribute** | **Description** |
|------------------|-------------|
| Type | Time |
| Default value | 0s |
| Value range | \[0s, 60m\] |
| Effective upon OBServer node restart | No |
