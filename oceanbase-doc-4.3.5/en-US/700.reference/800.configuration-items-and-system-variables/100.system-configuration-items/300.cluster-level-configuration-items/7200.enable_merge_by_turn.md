# enable_merge_by_turn

<main id="notice" type='explain'>
<h4>Note</h4>
<p>OceanBase Database V4.x does not support rotating compactions. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`enable_merge_by_turn` specifies whether to enable the rotating compaction strategy.

## Attributes

| **Attribute** | **Description** |
|------------------|------------------------------------------------------------------|
| Type | Boolean |
| Default value | False |
| Value range | </li><li>True</li><li>False |
| Effective upon OBServer node restart | No |
