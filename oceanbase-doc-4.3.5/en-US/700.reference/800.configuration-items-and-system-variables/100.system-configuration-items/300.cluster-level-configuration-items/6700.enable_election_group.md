# enable_election_group

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the system architecture optimization of OceanBase Database V4.x, election groups are no longer supported, and this parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`enable_election_group` specifies whether to enable the election group strategy.

## Attributes

| Attribute | Description |
|------------------|--------------------------------------------------|
| Type | Boolean |
| Default value | True |
| Value range | </li><li>True</li><li>False |
| Effective upon OBServer node restart | No |
