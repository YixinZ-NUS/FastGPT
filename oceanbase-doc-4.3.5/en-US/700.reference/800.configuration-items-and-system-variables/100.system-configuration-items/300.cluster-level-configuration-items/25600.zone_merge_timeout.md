# zone_merge_timeout

<main id="notice" type='explain'>
<h4>Note</h4>
<p>OceanBase Database V4.x does not support the control of a major compaction by zone. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`zone_merge_timeout` specifies the timeout period for the major compaction of a zone.
