# balancer_emergency_percentage

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the storage architecture upgrade of OceanBase Database V4.x, this parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`balancer_emergency_percentage` specifies the load threshold of each unit. When the load of a unit exceeds the specified threshold, you can enable replica migration to an external system for load balancing even during a major compaction.

## Attributes

| **Attribute** | **Description** |
|------------------|------------|
| Type | Integer |
| Default value | 80 |
| Value range | \[1, 100\] |
| Effective upon OBServer node restart | No |
