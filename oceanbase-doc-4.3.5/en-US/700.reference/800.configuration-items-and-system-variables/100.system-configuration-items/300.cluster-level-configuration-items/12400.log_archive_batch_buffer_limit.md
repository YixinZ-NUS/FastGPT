# log_archive_batch_buffer_limit

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the architecture upgrade of OceanBase Database V4.x, this parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

When physical backup is enabled, `log_archive_batch_buffer_limit` specifies the maximum memory available for log archiving on a single server.

|------------------|-----------|
| Type | Capacity |
