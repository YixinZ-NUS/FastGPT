# replica_safe_remove_time

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the optimization of the replica management feature in OceanBase Database V4.x, you do not need to specify the retention period of deleted replicas. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`replica_safe_remove_time` specifies the retention period of a deleted replica before it can be cleared.
