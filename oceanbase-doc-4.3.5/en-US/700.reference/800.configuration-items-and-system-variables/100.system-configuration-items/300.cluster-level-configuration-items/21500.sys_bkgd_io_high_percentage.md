# sys_bkgd_io_high_percentage

<main id="notice" type='explain'>
<h4>Note</h4>
<p>OceanBase Database V4.x supports I/O isolation between tenants. This parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`sys_bkgd_io_high_percentage` specifies the highest percentage of I/O resources that can be occupied by background tasks.

## Attributes

| **Attribute** | **Description** |
|------------------|------------|
| Type | Integer |
| Default value | 90 |
| Value range | \[1, 100\] |
| Effective upon OBServer node restart | No |

## References
