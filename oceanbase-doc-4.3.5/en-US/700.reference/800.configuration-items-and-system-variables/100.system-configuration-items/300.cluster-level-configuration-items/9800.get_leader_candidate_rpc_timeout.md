# get_leader_candidate_rpc_timeout

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the architecture upgrade of OceanBase Database V4.x, this parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`get_leader_candidate_rpc_timeout` specifies the timeout period of an internal request for obtaining the leader candidate in the automatic leader switchover strategy.

## Attributes

| **Attribute** | **Description** |
|------------------|--------------|
| Type | Time |
| Default value | 9s |
| Value range | \[2s, 180s\] |
| Effective upon OBServer node restart | No |
