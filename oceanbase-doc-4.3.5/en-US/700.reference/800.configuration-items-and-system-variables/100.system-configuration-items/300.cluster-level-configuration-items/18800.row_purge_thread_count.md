# row_purge_thread_count

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>The feature corresponding to this parameter does not take effect. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`row_purge_thread_count` specifies the maximum number of worker threads for a major compaction of rows in the memory.

## Attributes

| **Attribute** | **Description** |
|------------------|-----------|
| Type | Int |
| Default value | 4 |
| Value range | \[1, 64\] |
| Effective upon OBServer node restart | No |
