# zone_merge_concurrency

<main id="notice" type='explain'>
<h4>Note</h4>
<p>OceanBase Database V4.x does not support the control of a major compaction by zone. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`zone_merge_concurrency` specifies the number of zones that can be concurrently compacted in a major compaction. If this parameter is set to `0`, the system determines the best level of concurrency based on the actual deployment status.
