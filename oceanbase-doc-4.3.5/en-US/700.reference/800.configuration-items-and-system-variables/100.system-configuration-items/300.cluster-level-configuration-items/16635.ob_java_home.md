# ob_java_home

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This parameter is available starting with V4.3.5 BP1.</p>
</main>

## Description

`ob_java_home` specifies the Java home directory on the current OBServer node.

## Privilege requirements

* Query the parameter

    The `sys` tenant and all user tenants can use the `SHOW PARAMETERS` statement or the `GV$OB_PARAMETERS` view to query this parameter.

* Modify the parameter

    Only the `sys` tenant can modify this parameter. User tenants cannot modify this parameter.

## Attributes

|Attribute|Description|
|---|---|
|Type|String|
|Default value|NULL|
|Modifiable|Yes. You can use the `ALTER SYSTEM SET` statement to modify the parameter.|
|Effective upon OBServer node restart|No. The setting takes effect immediately.|

## Examples

Set the Java home directory on the current OBServer node to `/home/admin/jdk8u422_05`.

```shell
ALTER SYSTEM SET ob_java_home="/home/admin/jdk8u422_05";
```
