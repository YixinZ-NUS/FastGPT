# all_server_list

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This parameter is available starting with V1.4. </p>
</main>

## Description

`all_server_list` specifies the addresses of all servers in the cluster.

## Privilege requirements

* Query the parameter

  The `sys` tenant and all user tenants can use the `SHOW PARAMETERS` statement to query this parameter.

* Modify the parameter

  Only the `sys` tenant can modify this parameter. User tenants cannot modify this parameter.

## Attributes

|      **Attribute**      | **Description** |
|--------------------|----------|
| Type            | String    |
| Default value              | "" indicates an empty string.     |
| Modifiable          | Yes. You can use the `ALTER SYSTEM SET` statement to modify the parameter.    |
| Effective upon OBServer node restart | No. The setting takes effect immediately.      |

## Examples

Set the server addresses in the cluster to `xx.xx.xx.01:2882;xx.xx.xx.02:2882;xx.xx.xx.03:2882`.

```shell
obclient> ALTER SYSTEM SET all_server_list='xx.xx.xx.01:2882;xx.xx.xx.02:2882;xx.xx.xx.03:2882';
```
