# enable_rootservice_standalone

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the storage architecture optimization of OceanBase Database V4.x, the <code>sys</code> tenant stores only the private data of the cluster but not the data of user tenants. This parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0. </p>
</main>

`enable_rootservice_standalone` specifies whether to allow the `sys` tenant and RootService to exclusively occupy an OBServer node.

| **Attribute** | **Description** |
|------------------|------------------------------------------------------------|
| Type | Boolean |
| Default value | False |
| Value range | </li><li>True</li><li>False |
| Effective upon OBServer node restart | No |
