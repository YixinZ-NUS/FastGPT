# tablet_ls_cache_priority

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This parameter is available starting with V4.0.0. </p>
</main>

## Description

`tablet_ls_cache_priority` specifies the priority of the tablet mapping cache.

## Privilege requirements

* Query the parameter

  The `sys` tenant and all user tenants can query this parameter by using the `SHOW PARAMETERS` statement or the `GV$OB_PARAMETERS` view.

* Modify the parameter

  Only the `sys` tenant can modify this parameter. User tenants cannot modify this parameter.

## Attributes

| **Attribute** | **Description** |
| -------- | -------- |

## Examples

Set the priority of the tablet mapping cache to 1000.

```shell
obclient> ALTER SYSTEM SET tablet_ls_cache_priority = 1000;
```
