# minor_warm_up_duration_time

<main id="notice" type='explain'>
<h4>Note</h4>
<p>We recommend that you do not use this parameter in OceanBase Database V2.2.3x and later. It has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`minor_warm_up_duration_time` specifies the preload time of the new MemTable generated after a minor compaction.

## Attributes

| **Attribute** | **Description** |
|------------------|-------------|
| Type | Time |
| Default value | 30s |
| Value range | \[0s, 60m\] |
| Effective upon OBServer node restart | No |

## Considerations

If this parameter is set to `0s`, preloading is not performed.
