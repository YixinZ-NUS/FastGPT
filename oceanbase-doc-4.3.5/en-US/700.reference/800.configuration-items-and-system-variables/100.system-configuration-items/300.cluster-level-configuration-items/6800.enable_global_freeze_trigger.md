# enable_global_freeze_trigger

<main id="notice" type='explain'>
<h4>Note</h4>
<p>OceanBase Database V4.x supports tenant-level freezes. This parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0.  </p>
</main>

## Description

`enable_global_freeze_trigger` specifies whether to enable automatic triggering of a global freeze.

## Attributes

| **Attribute** | **Description** |
|------------------|-----------------------------------------------|
| Type | Boolean |
| Default value | False |
| Value range | </li><li>True</li><li>False |
| Effective upon OBServer node restart | No |

<main id="notice" type='notice'>
  <h4>Notice</h4>
  <p>We recommend that you use the default value of this parameter.  </p>
</main>
