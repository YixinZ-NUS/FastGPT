# merger_completion_percentage

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This parameter had been rarely used since it was introduced. Therefore, it has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`merger_completion_percentage` specifies the percentage of compacted replicas at which the major compaction task is considered completed.

## Attributes

| **Attribute** | **Description** |
|------------------|------------|
| Type | Integer |
| Default value | 100 |
| Value range | \[5, 100\] |
| Effective upon OBServer node restart | No |
