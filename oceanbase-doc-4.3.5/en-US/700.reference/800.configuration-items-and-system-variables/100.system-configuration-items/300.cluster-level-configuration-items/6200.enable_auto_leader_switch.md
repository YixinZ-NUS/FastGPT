# enable_auto_leader_switch

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the system architecture optimization of OceanBase Database V4.x, automatic leader switchover is controlled by a single log stream without load balancing, and this parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`enable_auto_leader_switch` specifies whether to enable automatic leader switchover.

## Attributes

| **Attribute** | **Description** |
|------------------|---------------------------------------------------------|
| Type | Boolean |
| Default value | True |
| Value range | </li><li>True</li><li>False |
| Effective upon OBServer node restart | No |
