# rpc_port

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This parameter is available starting with V2.2.50. </p>
</main>

## Description

`rpc_port` specifies the port number of the remote access protocol.

## Privilege requirements

* Query the parameter

  The `sys` tenant and all user tenants can use the `SHOW PARAMETERS` statement or the `GV$OB_PARAMETERS` view to query this parameter.

* Modify the parameter

  **This parameter can be set only when the cluster is deployed. After the cluster is deployed, the parameter cannot be modified.**

## Attributes

|      **Attribute**      |     **Description**      |
|------------------|-----------------|
| Type             | Int              |
| Default value               | 2882  <main id="notice" type='explain'><h4>Note</h4><p>For V3.2.4, the default value is changed from 2500 to 2882. </p></main>          |
| Value range             | [1025, 65535] |
| Modifiable                | No. You cannot use the `ALTER SYSTEM SET` statement to modify the parameter.  |
| Effective upon OBServer node restart | No. The parameter is read-only.               |
