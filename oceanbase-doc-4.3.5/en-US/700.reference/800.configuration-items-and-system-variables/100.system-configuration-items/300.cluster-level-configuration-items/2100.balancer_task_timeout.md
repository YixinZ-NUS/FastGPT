# balancer_task_timeout

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This parameter is available starting with V1.4. </p>
</main>

## Description

`balancer_task_timeout` specifies the timeout period of background tasks such as load balancing.

## Privilege requirements

* Query the parameter

  The `sys` tenant and all user tenants can query the value of this parameter by using the `SHOW PARAMETERS` statement.

* Modify the parameter

  Only the `sys` tenant can modify this parameter. User tenants cannot modify this parameter.

## Attributes

|      **Attribute**      |    **Description**    |
|------------------|--------------|
| Type             | Time         |
| Default value               | 20m, which indicates 20 minutes |
| Value range             | [1s, +âˆž)    |
| Modifiable           | Yes. You can use the `ALTER SYSTEM SET` statement to modify the parameter. |

## Examples

Set the timeout period of background tasks such as load balancing to 20m.

```shell
obclient> ALTER SYSTEM SET balancer_task_timeout='20m';
```
