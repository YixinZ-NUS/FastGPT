# zone_merge_order

<main id="notice" type='explain'>
<h4>Note</h4>
<p>OceanBase Database V4.x does not support the control of a major compaction by zone or rotating compactions. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`zone_merge_order` specifies the order of zones in a rotating compaction. If you do not specify a value, the system determines the parameter value.

## Attributes

| **Attribute** | **Description** |
|------------------|---------------------|
| Type | String |
| Default value | Empty string |
| Effective upon OBServer node restart | No |
