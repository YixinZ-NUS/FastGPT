# server_check_interval

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This parameter is available starting with V1.4. </p>
</main>

## Description

`server_check_interval` specifies the time interval for checking the consistency of the Server table.

## Privilege requirements

* Query the parameter

  The `sys` tenant and all user tenants can query the value of this parameter by using the `SHOW PARAMETERS` statement or the `GV$OB_PARAMETERS` view.

* Modify the parameter

  Only the `sys` tenant can modify this parameter. User tenants cannot modify this parameter.

## Examples

Set the time interval for checking the consistency of the Server table to 30s.

```shell
obclient> ALTER SYSTEM SET server_check_interval = '30s';
```
