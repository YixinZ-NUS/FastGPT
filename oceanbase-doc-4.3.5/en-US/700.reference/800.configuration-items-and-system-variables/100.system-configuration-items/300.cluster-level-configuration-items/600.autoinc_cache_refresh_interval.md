# autoinc_cache_refresh_interval

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This parameter is available starting with V1.4. </p>
</main>

## Description

`autoinc_cache_refresh_interval` specifies the interval for refreshing the cache of an auto-increment column.

## Privilege requirements

* Query the parameter

  The `sys` tenant and all user tenants can use the `SHOW PARAMETERS` statement to query this parameter.

* Modify the parameter

  Only the `sys` tenant can modify this parameter. User tenants cannot modify this parameter.

## Attributes

|      **Attribute**    |    **Description**   |
|------------------|--------------|
| Type            | Time         |
| Default value              | 3600s        |
| Value range            | \[100ms, +âˆž) |
| Modifiable          | Yes. You can use the `ALTER SYSTEM SET` statement to modify the parameter.    |
| Effective upon OBServer node restart | No. The setting takes effect immediately.      |

## Examples

Set the refresh interval of the cache of an auto-increment column to 3600s.

```shell
obclient> ALTER SYSTEM SET autoinc_cache_refresh_interval ='3600s';
```
