# index_info_block_cache_priority

<main id="notice" type='explain'>
<h4>Note</h4>
<ul><li>Due to the architecture upgrade of OceanBase Database V4.x, this parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0. </li>
<li>In OceanBase Database V4.x, you can use the <code>index_block_cache_priority</code> parameter to specify the priority of the microblock cache for metadata indexes. For more information about this parameter, see <a href="10600.index_block_cache_priority.md">index_block_cache_priority</a>. </li></ul>
</main>

## Description

`index_info_block_cache_priority` specifies the priority of the block index cache in the cache system.
