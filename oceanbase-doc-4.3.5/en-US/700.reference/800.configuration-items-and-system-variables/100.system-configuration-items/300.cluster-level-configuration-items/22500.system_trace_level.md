# system_trace_level

<main id="notice" type='explain'>
<h4>Note</h4>
<p>In OceanBase Database V4.x, the corresponding logic is changed, and the corresponding feature is no longer available. This parameter has been deprecated since OceanBase Database V4.2.0. </p>
</main>

## Description

`system_trace_level` specifies the level of trace logs.

## Attributes

| **Attribute** | **Description** |
|-------------|--------------------|
| Type | Integer |
| Default value | 1 |
| Value range |<ul><li>`0`: none</li><li>`1`: standard</li><li>`2`: debug</li></ul> |
| Modifiable | Yes. You can use the `ALTER SYSTEM SET` statement to modify the parameter. |
| Effective upon OBServer node restart | No. The setting takes effect immediately.     |

## Examples

Set the level of trace logs to debug.

```shell
obclient> ALTER SYSTEM SET system_trace_level=2;
```
