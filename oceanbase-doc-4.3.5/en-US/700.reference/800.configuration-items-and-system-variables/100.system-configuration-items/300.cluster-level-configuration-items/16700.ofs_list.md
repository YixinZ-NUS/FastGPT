# ofs_list

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>In OceanBase Database V4.x, the corresponding feature is no longer available. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`ofs_list` specifies the list of URLs for connecting to the OceanBase File System (OFS) of all the zones so that RootService can access files across zones in OFS deployment mode. OFS is a distributed storage system independently designed for OceanBase Database.

## Attributes

| Attribute | Description |
|------------------|--------------------------|
| Type | String |
| Default value | The URL list specified during deployment |
| Value range | `[zone]@[ofs_url];` |
| Effective upon OBServer node restart | No |
