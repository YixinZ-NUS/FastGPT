# virtual_table_location_cache_expire_time

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This parameter is available starting with V1.4. </p>
</main>

## Description

`virtual_table_location_cache_expire_time` specifies the cache expiration time for the location information of a virtual table.

## Privilege requirements

* Query the parameter

  The `sys` tenant and all user tenants can query this parameter by using the `SHOW PARAMETERS` statement or the `GV$OB_PARAMETERS` view.

* Modify the parameter

  Only the `sys` tenant can modify this parameter. User tenants cannot modify this parameter.

## Examples

Set the cache expiration time for the location information of a virtual table to 8s.

```shell
obclient> ALTER SYSTEM SET virtual_table_location_cache_expire_time ='8s';
```
