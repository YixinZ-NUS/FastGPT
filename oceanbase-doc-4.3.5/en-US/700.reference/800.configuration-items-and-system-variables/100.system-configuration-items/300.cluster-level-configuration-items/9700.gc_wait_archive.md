# gc_wait_archive

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the storage architecture upgrade of OceanBase Database V4.x, this parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`gc_wait_archive` specifies whether to start garbage collection for a partition until all the logs in the partition have been archived.

## Attributes

| **Attribute** | **Description** |
|------------------|--------------------------------------------------------------------------------------------------------|
| Type | Boolean |
| Default value | True |
| Value range | </li><li> False   </li><li> True    |
| Effective upon OBServer node restart | No |
