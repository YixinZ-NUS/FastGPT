# sys_bkgd_io_low_percentage

<main id="notice" type='explain'>
<h4>Note</h4>
<p>OceanBase Database V4.x supports I/O isolation between tenants. This parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`sys_bkgd_io_low_percentage` specifies the lowest percentage of I/O resources that can be occupied by background tasks.

## Attributes

| **Attribute** | **Description** |
|------------------|------------|
| Type | Integer |
| Default value | 0 |
| Value range | \[0, 100\] |
| Effective upon OBServer node restart | No |

## Considerations

If the value of this parameter is set to `0`, the system will automatically adjust the percentage based on the environment.
