# backup_net_limit

<main id="notice" type='explain'>
<h4>Note</h4>
<P>Due to the architecture optimization of the physical backup and restore feature in OceanBase Database V4.x, cluster-level backup is no longer supported, and this parameter has been deprecated since OceanBase Database V4.0.0. </P>
</main>

## Description

`backup_net_limit` specifies the total bandwidth for cluster backup.
