# wait_leader_batch_count

<main id="notice" type='explain'>
<h4>Note</h4>
<p>OceanBase Database V4.x adopts the standalone log stream architecture, where leader switchover is controlled by a single log stream. This parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`wait_leader_batch_count` specifies the maximum number of partitions to which RootService can send a command for leader switchover.

## Attributes

| **Attribute** | **Description** |
|------------------|---------------|
| Type | Integer |
| Default value | 1024 |
| Value range | \[128, 5000\] |
| Effective upon OBServer node restart | No |
