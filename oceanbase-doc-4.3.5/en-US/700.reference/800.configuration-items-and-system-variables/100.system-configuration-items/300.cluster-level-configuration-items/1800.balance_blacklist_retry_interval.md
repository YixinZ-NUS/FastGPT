# balance_blacklist_retry_interval

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the storage architecture upgrade of OceanBase Database V4.x, this parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`balance_blacklist_retry_interval` specifies the retry interval for a background task such as replica migration that is added to the blocklist.

## Attributes

| **Attribute** | **Description** |
|------------------|--------------|
| Type | Time |
| Default value | `30m`, indicating 30 minutes. |
| Value range | \[0s, 180m\] |
| Effective upon OBServer node restart | No |
