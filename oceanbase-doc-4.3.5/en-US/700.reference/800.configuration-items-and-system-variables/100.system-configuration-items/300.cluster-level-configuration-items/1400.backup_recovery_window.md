# backup_recovery_window

<main id="notice" type='explain'>
<h4>Note</h4>
<ul><li>Due to the architecture optimization of the physical backup and restore feature in OceanBase Database V4.x, this parameter has been deprecated since OceanBase Database V4.0.0. </li>
<li>In OceanBase Database V4.x, you can use the <code>RECOVERY_WINDOW</code> parameter in the <a href="../../../500.sql-reference/100.sql-syntax/100.system-tenants/200.alter-system/1600.delete-backup.md">ALTER SYSTEM ADD DELETE BACKUP POLICY</a> statement to specify the time window for restoring backup data in the <code>sys</code> tenant or a user tenant. </li></ul>
</main>

## Description

`backup_recovery_window` specifies the time window in which backups can be restored.
