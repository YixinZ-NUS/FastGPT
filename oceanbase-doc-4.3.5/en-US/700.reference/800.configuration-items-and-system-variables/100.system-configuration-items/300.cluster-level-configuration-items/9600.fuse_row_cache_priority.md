# fuse_row_cache_priority

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This parameter is available starting with V2.2.50. </p>
</main>

## Description

`fuse_row_cache_priority` specifies the priority of the row cache in the cache system.

## Privilege requirements

* Query the parameter

  `sys` tenant and all user tenants can use the `SHOW PARAMETERS` statement or the `GV$OB_PARAMETERS` view to query this parameter.

* Modify the parameter

  Only the `sys` tenant can modify this parameter. User tenants cannot modify this parameter.

## Examples

Set the priority of the row cache in the cache system to 1.

```shell
obclient> ALTER SYSTEM SET fuse_row_cache_priority = 1;
```
