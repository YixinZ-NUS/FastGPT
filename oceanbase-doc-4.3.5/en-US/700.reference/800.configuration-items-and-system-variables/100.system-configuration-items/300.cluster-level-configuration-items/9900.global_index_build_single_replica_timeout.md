# global_index_build_single_replica_timeout

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>Due to logic optimization of OceanBase Database V4.x, you do not need to specify the timeout period for creating a replica, and the corresponding feature is no longer available. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`global_index_build_single_replica_timeout` specifies the timeout period for creating a replica during global index creation.
