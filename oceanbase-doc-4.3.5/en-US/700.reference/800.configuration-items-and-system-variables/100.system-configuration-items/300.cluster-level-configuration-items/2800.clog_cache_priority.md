# clog_cache_priority

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the architecture upgrade of OceanBase Database V4.x, transaction logs are no longer cached, and this parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`clog_cache_priority` specifies the priority of the transaction log cache.
