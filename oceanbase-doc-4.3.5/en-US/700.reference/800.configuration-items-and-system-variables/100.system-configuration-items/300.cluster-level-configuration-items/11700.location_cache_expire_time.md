# location_cache_expire_time

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the architecture upgrade of OceanBase Database V4.x, this parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`location_cache_expire_time` specifies the retention period of partition location information in the partition location cache. We recommend that you do not modify this parameter.
