# data_dir

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This parameter is available starting with V1.4.</p>
</main>

## Description

`data_dir` specifies the directory for storing SSTable files and other data.

## Privilege requirements

The `sys` tenant and all user tenants can use the `SHOW PARAMETERS` statement or the `GV$OB_PARAMETERS` view to query this parameter.

## Attributes

| **Attribute** | **Description** |
|--------|--------|
| Type | String |
| Default value | store |
| Modifiable | No. You cannot use the `ALTER SYSTEM SET` statement to modify the parameter. |
| Effective upon OBServer node restart | No. It is read-only. |
