# user_iort_up_percentage

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>The I/O scheduling algorithm is modified, and the corresponding feature is no longer available. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`user_iort_up_percentage` specifies the I/O latency threshold for the user disk. When the I/O latency of the user disk reaches this threshold, the traffic of background I/O operations is throttled.
