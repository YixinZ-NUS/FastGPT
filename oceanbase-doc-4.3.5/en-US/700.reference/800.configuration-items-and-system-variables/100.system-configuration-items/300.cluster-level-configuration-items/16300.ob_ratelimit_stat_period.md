# ob_ratelimit_stat_period

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>For V3.2.2, this parameter is available starting with V3.2.2.</p>
</main>

## Description

`ob_ratelimit_stat_period` specifies the time interval for an OBServer node to calculate and update the maximum bandwidth.

## Privilege requirements

* Query the parameter

  The `sys` tenant and all user tenants can use the `SHOW PARAMETERS` statement or the `GV$OB_PARAMETERS` view to query this parameter.

* Modify the parameter

  Only the `sys` tenant can modify this parameter. User tenants cannot modify this parameter.

|      **Attribute**      |    **Description**    |

## Examples

Set the time interval for an OBServer node to calculate and update the maximum bandwidth to 1s.

```shell
obclient> ALTER SYSTEM SET ob_ratelimit_stat_period = '1s';
```
