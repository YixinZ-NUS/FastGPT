# devname

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>devname is available starting with V1.4. </p>
</main>

## Description

`devname` specifies the name of the network interface card (NIC) to which the service process is bound.

## Privilege requirements

The `sys` tenant and all user tenants can query the `devname` parameter by using the `SHOW PARAMETERS` statement or the `GV$OB_PARAMETERS` view.

## Attributes

|      **Attribute**      | **Description** |
|------------------|------------|
| Parameter type             | String    |
| Default value               | bond0  |
| Modifiable           | No. The parameter is read-only.<main id="notice" type='explain'><h4>Note</h4><p>devname is available starting with V4.2.0.  </p></main>|
