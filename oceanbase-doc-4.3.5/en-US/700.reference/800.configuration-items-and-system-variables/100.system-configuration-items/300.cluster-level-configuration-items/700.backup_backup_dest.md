# backup_backup_dest

<main id="notice" type='explain'>
<h4>Note</h4>
<p>For V4.x, the backup backup (secondary backup) feature is no longer supported. This parameter is removed starting from V4.0.0. </p>
</main>

## Description

`backup_backup_dest` specifies the destination for backing up backup data.

## Attributes

|        Attribute        |  Description   |
|------------------|-------|
| Type             | String |
| Default value              | Empty string  |
| Effective upon OBServer node restart | No     |
