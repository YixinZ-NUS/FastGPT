# tenant_groups

<main id="notice" type='explain'>
<h4>Note</h4>
<p>OceanBase Database V4.x does not support tenant groups. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`tenant_groups` specifies the tenant groups used in the load balancing strategy.

## Attributes

| **Attribute** | **Description** |
|------------------|--------|
| Type | String |
| Default value | Empty string |
| Effective upon OBServer node restart | No |
