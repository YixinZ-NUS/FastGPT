# enable_smooth_leader_switch

<main id="notice" type='explain'>
<h4>Note</h4>
<p>OceanBase Database V4.x does not support leader switchover for partitions. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`enable_smooth_leader_switch` specifies whether to enable smooth leader switchover.

## Attributes

| **Attribute** | **Description** |
|------------------|------------------------------------------------------------|
| Type | Boolean |
| Default value | True |
| Value range | </li><li>True</li><li>False |
| Effective upon OBServer node restart | No |
