# switchover_process_thread_count

<main id="notice" type='explain'>
<h4>Note</h4>
<p>In OceanBase Database V4.x, the switchover logic adapts to log stream changes, and no independent worker thread is required. This parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`switchover_process_thread_count` specifies the size of the thread pool for primary/standby database switchover.

## Attributes

| **Attribute** | **Description** |
|------------------|-------------|
| Type | Integer |
| Default value | 4 |
| Value range | \[1, 1000\] |
| Effective upon OBServer node restart | No |
