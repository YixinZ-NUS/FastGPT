# max_kept_major_version_number

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This parameter retains old baseline versions, which occupies additional storage space. In addition, this parameter had been rarely used since it was introduced. Therefore, it has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`max_kept_major_version_number` specifies the number of frozen data versions to be retained.

## Attributes

| **Attribute** | **Description** |
|------------------|-----------|
| Type | Integer |
| Default value | 2 |
| Value range | \[1, 16\] |
| Effective upon OBServer node restart | No |
