# ob_esi_session_timeout

<main id="notice" type='explain'>
<h4>Note</h4>
<p>Due to the architecture optimization of the physical backup and restore feature in OceanBase Database V4.x, the feature corresponding to this parameter is no longer supported and this parameter has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`ob_esi_session_timeout` specifies the timeout period of the active session resources for the obesi process.

## Attributes

| **Attribute** | **Description** |
| --- | --- |
| Type | Time |
| Default value | 1m |
| Value range | [10s, 30m] |
| Effective upon OBServer node restart | No |
