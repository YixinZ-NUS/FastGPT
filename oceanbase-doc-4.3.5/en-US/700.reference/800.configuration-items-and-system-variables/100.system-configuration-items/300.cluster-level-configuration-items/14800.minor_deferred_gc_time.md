# minor_deferred_gc_time

<main id="notice" type='explain'>
<h4>Note</h4>
<p>We recommend that you do not use this parameter in OceanBase Database V2.2.3x and later. It has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`minor_deferred_gc_time` specifies the interval between the time when garbage collection starts for SSTables and the time when the major compaction ends.

## Attributes

| **Attribute** | **Description** |
|------------------|-------------|
| Type | Time |
| Default value | 0s |
| Value range | \[0s, 24h] |
| Effective upon OBServer node restart | No |
