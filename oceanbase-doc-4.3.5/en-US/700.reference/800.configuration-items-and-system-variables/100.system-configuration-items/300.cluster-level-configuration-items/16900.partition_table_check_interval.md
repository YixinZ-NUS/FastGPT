# partition_table_check_interval

<main id="notice" type='explain'>
<h4>Note</h4>
<p>OceanBase Database V4.x does not support partition replicas. This parameter is no longer applicable and has been deprecated since OceanBase Database V4.0.0. </p>
</main>

## Description

`partition_table_check_interval` specifies the interval at which the OBServer node deletes non-existent replicas from a partitioned table.
