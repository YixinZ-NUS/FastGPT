# ob_enable_hash_group_by

<main id="notice" type='explain'>
 <h4>Note</h4>
 <p>Due to the system architecture optimization in V4.x, this parameter was removed starting from V4.0.0. </p>
</main>

## Description

The `ob_enable_hash_group_by` variable specifies whether to enable the hash group by path.

## Attributes

|  **Attribute**  |       **Description**       |
|------------|----------------------|
| Type     | Bool                 |
| Default value       | 1                    |
| Valid values     | <ul><li>0: disabled </li><li>1: enabled </li></ul>|
| Scope     | <ul><li>Global </li><li>Session </li></ul>   |
