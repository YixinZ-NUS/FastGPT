# performance_schema

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This variable is used only for compatibility with MySQL 8.0, but its functionality is not implemented. You can query and set this variable, but the setting does not take effect even though the system does not return an error. </p>
</main>

## Description

`performance_schema` declares to the client whether performance queries are supported.

## Attributes

| Attribute | Description |
|--------------|-------------------------------------------------------|
| Type | Boolean |
| Default value | OFF |
| Value range | <ul><li>`ON`: supported</li><li>`OFF`: not supported</li></ul> |
| Effective scope | Global |
| Modifiable | Yes. You can use the `SET` statement to modify the variable. |
