# ob_enable_blk_nestedloop_join

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>Due to the changes in the corresponding logic in V4.x, the corresponding feature is no longer used. This variable is removed in V4.0.0. </p>
</main>

## Description

The `ob_enable_blk_nestedloop_join` variable specifies whether to allow block nested loop join.

## Attributes

|    **Attribute**    |    **Description**       |
|----------------|-------------------|
| Type        | Bool                                      |
| Default value          | 0                                         |
| Value range        | <ul><li>0: disabled</li><li>1: enabled</li></ul> |
| Applicability range        | <ul><li>Global</li><li>SESSION</li></ul>   |
