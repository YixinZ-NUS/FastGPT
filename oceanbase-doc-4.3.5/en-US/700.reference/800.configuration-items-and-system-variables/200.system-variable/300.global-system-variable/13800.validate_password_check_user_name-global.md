# validate_password_check_user_name

The `validate_password_check_user_name` variable specifies whether the user password can be the same as the username.

| **Attribute**  |                                                              **Description**                                                               |
|---------|-----------------------------------------------------------------------------------------------------------------------------------|
| Type    | enum                                           |
| Default value     | on                                            |
| Valid values    | <li> on: specifies that the user password cannot be the same as the username.   <li> off: specifies that the user password can be the same as the username.    |
| Effective scope    | Global                                         |
| Modifiable  | Yes. You can execute the `SET` statement to modify the value.|

  <main id="notice" type='notice'>
    <h4>Notice</h4>
    <p>This variable must be used in conjunction with the <code>validate_password_policy</code> variable. This variable takes effect only when the value of the <code>validate_password_policy</code> variable is <code>MEDIUM</code>. For more information about the <code>validate_password_policy</code> variable, see <a href="14200.validate_password_policy-global.md">validate_password_policy</a>. </p>
