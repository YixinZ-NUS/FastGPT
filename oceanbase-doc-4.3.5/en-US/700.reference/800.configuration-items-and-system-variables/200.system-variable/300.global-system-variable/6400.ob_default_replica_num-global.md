# ob_default_replica_num

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>Due to the system architecture optimization in V3.x, the related feature is no longer used. This parameter is deprecated starting from V3.2.1. </p>
</main>

## Description

`ob_default_replica_num` specifies the default number of replicas for tables in a cluster.

## Attributes

| **Attribute**  |    **Description**     |
|-----------|-----------------|
| Type   | Int     |
| Default value     | 1   |
| Valid values   | [1, 7] |
| Effective scope   | <ul><li>Global </li><li>Session </li></ul>    |
