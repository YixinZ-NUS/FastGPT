# sql_safe_updates

The sql_safe_updates variable specifies whether to enable the SQL safe mode in MySQL. The default value is 0.

  <main id="notice" type='explain'>
    <h4>Note</h4>
    <p>This variable is supported only for compatibility. OceanBase Database does not implement this variable. </p>
  </main>

| **Attribute**  |                                                          **Description**                                                          |
|---------|--------------------------------------------------------------------------------------------------------------------------|
| Type    | bool                                  |
| Default value     | 0                                     |
| Value range    | <li> 0: indicates FALSE, which means disabled.   <li> 1: indicates TRUE, which means enabled.    |
| Effective scope    | <li> GLOBAL   <li> SESSION                  |
