# ob_temp_tablespace_size_percentage

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This variable is no longer available due to system architecture adjustments. The percentage of disk data written is controlled by the <a href="../../../../700.reference/800.configuration-items-and-system-variables/100.system-configuration-items/300.cluster-level-configuration-items/4100.data_disk_usage_limit_percentage.md">data_disk_usage_limit_percentage</a> parameter. This variable is removed starting from V2.1. </p>
</main>

## Description

`ob_temp_tablespace_size_percentage` specifies the percentage limit of the temporary tablespace size in a tenant's disk.

## Attributes

| **Attribute**  |    **Description**  |
|-----------|---------------|
| Type    | Int                     |
| Default value      | 0                       |
| Value range    | [0, 100]              |
| Scope    |  <ul><li>GLOBAL</li><li>Session</li></ul>  |
| Modifiable | Yes. You can execute the `SET` statement to modify the value. |
