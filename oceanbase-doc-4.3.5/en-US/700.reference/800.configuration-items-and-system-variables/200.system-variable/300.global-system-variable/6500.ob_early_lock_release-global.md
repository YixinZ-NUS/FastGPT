# ob_early_lock_release

<main id="notice" type='explain'>
 <h4>Note</h4>
 <p>This variable is deprecated starting with V2.2.30. The behavior of the Early Lock Release (ELR) feature is now controlled by the tenant-level parameter <a href="../../../../700.reference/800.configuration-items-and-system-variables/100.system-configuration-items/400.tenant-level-configuration-items/2500.enable_early_lock_release.md">enable_early_lock_release</a>. </p>
</main>

## Description

`ob_early_lock_release` specifies whether to enable the Early Lock Release (ELR) feature for a transaction.

## Attributes

| **Attribute**  |       **Description**       |
|-----------|----------------------|
| Type    | Bool                |
| Default value      | 0                   |
| Value range    | <ul><li>0: Disabled.  </li><li>1: Enabled. </li></ul>   |
| Scope    | Global              |
