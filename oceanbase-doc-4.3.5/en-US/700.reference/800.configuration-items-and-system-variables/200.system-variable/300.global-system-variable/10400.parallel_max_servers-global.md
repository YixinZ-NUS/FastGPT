# parallel_max_servers

<main id="notice" type='explain'>
<h4>Note</h4>
<p>This variable is no longer used in the current version. Use the <a href="10600.parallel_servers_target-global.md">parallel_servers_target</a> variable to control the resources used for parallel execution. </p>
</main>

The parallel_max_servers variable specifies the maximum number of threads in the parallel execution (Parallel eXecution, PX) thread pool on each server.

| **Attribute**  |   **Description**    |
|---------|-------------|
| Type    | int         |
| Default value     | 0           |
| Value range    | \[0, 1800\] |
| Valid scope    | GLOBAL      |
| Modifiable  | Yes. You can execute the `SET` statement to modify this variable. |
