# nls_language

<main id="notice" type='explain'>
  <h4>Note</h4>
  <p>This variable was introduced in V2.2.77. </p>
</main>

## Description

`nls_language` specifies the default language of prompt messages, date, month, and the default values of `nls_sort` and `nls_date_language`.

<main id="notice">
  <h4>Applicability</h4>
  <p>This variable applies only to the Oracle-compatible mode of OceanBase Database. </p>
</main>

## Privilege requirements

* Query variables

  Both the `sys` tenant and all user tenants can execute the `SHOW VARIABLES` statement or query the `SYS.TENANT_VIRTUAL_SESSION_VARIABLE` view (in Oracle-compatible mode) to query the values of session system variables.

* Modify variables

  The value of this variable cannot be modified.

## Attribute description

| **Attribute** | **Description** |
|---------|---------|
| Type | Varchar |
| Default value | AMERICA |
| Effective scope | Session |
| Modifiability | No, this variable is read-only. |
